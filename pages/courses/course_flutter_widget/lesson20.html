第20课 隐式动画组件

课程系列：Flutter 常用组件系统学习
适用人群：有 Dart 语言基础，了解 Flutter 基本概念的开发者
课程目标：掌握 Flutter 隐式动画体系，能够使用内置隐式动画组件和 TweenAnimationBuilder 为应用添加流畅的过渡动画效果

本课是动画系列的第一课。Flutter 的动画体系可以分为隐式动画和显式动画两大阵营。隐式动画是上手最简单、使用最频繁的动画方式——你只需要改变属性值，Flutter 就会自动帮你完成从旧值到新值的平滑过渡。本课将系统讲解所有常用的隐式动画组件，帮你在不接触 AnimationController 的情况下，轻松实现大部分动画需求。

1. 隐式动画原理

1.1 什么是隐式动画

隐式动画（Implicit Animation）是 Flutter 中最简单的动画模式。它的核心思想是：

你只负责声明"目标状态"，Flutter 负责计算"过渡过程"。

当你在 setState 中改变一个属性值（比如颜色从红色变为蓝色），隐式动画组件会自动在指定的时间（duration）内，按照指定的曲线（curve）从旧值平滑过渡到新值。你无需手动创建 AnimationController、管理 Ticker、处理 addListener——这一切都被封装好了。

1.2 隐式动画 vs 显式动画

对比维度 隐式动画 显式动画（第21课）
核心思想 声明目标值，自动过渡 手动控制动画的每一帧
代码复杂度 低，只需设置属性和 duration 高，需要 AnimationController + Tween + Listener
控制粒度 粗粒度（开始→结束） 细粒度（暂停、倒放、重复、联动多个动画）
典型组件 AnimatedContainer、AnimatedOpacity 等 AnimationController + AnimatedBuilder 等
适用场景 属性变化触发的一次性过渡 循环动画、交错动画、手势驱动动画
状态管理 仅需 StatefulWidget + setState 需要 TickerProviderStateMixin

选择原则：优先使用隐式动画。只有当你需要更精细的控制（如循环播放、暂停/恢复、多个动画编排）时，才考虑显式动画。

1.3 ImplicitlyAnimatedWidget 基类

Flutter 中所有内置的隐式动画组件都继承自 ImplicitlyAnimatedWidget。这个基类提供了以下核心能力：

  • 自动管理 AnimationController：内部创建并维护一个 AnimationController，开发者无需操心
  • 自动检测属性变化：当 didUpdateWidget 检测到属性值发生变化时，自动启动动画
  • duration 和 curve：所有隐式动画组件都接受 duration（动画时长）和 curve（动画曲线）参数
  • onEnd 回调：动画结束时的回调（部分子类支持）

继承关系如下：

Widget
  └── StatefulWidget
        └── ImplicitlyAnimatedWidget
              ├── AnimatedContainer
              ├── AnimatedOpacity
              ├── AnimatedPadding
              ├── AnimatedPositioned
              ├── AnimatedDefaultTextStyle
              ├── AnimatedPhysicalModel
              └── ...

1.4 隐式动画的工作流程

  1. 首次构建：组件以初始属性值进行渲染
  2. 属性变化：通过 setState 更改目标属性值
  3. 检测变化：didUpdateWidget 发现新旧属性不同
  4. 启动动画：内部 AnimationController 从 0 到 1 播放
  5. 插值计算：每一帧根据 curve 和 Tween 计算中间值
  6. 重新渲染：以中间值绘制 UI，直到到达目标值
import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: const ImplicitAnimationDemo(),
    );
  }
}

/// 最简隐式动画示例：点击按钮改变容器颜色，自动产生过渡动画
class ImplicitAnimationDemo extends StatefulWidget {
  const ImplicitAnimationDemo({super.key});

  @override
  State<ImplicitAnimationDemo> createState() => _ImplicitAnimationDemoState();

2. AnimatedContainer

AnimatedContainer 是最常用、最强大的隐式动画组件，它几乎能对 Container 的所有视觉属性做动画。

2.1 可动画化的属性

属性 类型 说明
width double? 宽度
height double? 高度
color Color? 背景颜色（不可与 decoration 中的 color 同时使用）
padding EdgeInsetsGeometry? 内边距
margin EdgeInsetsGeometry? 外边距
decoration Decoration? 装饰（背景色、渐变、边框、圆角、阴影）
alignment AlignmentGeometry? 子组件对齐方式
transform Matrix4? 矩阵变换（旋转、缩放、平移）
constraints BoxConstraints? 额外约束

核心参数：

参数 类型 必填 说明
duration Duration 是 动画时长
curve Curve 否 动画曲线，默认 Curves.linear
onEnd VoidCallback? 否 动画结束时的回调

2.2 基础用法：尺寸与颜色动画

import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: const AnimatedContainerBasicDemo(),
    );
  }
}

class AnimatedContainerBasicDemo extends StatefulWidget {
  const AnimatedContainerBasicDemo({super.key});

  @override
  State<AnimatedContainerBasicDemo> createState() =>
      _AnimatedContainerBasicDemoState();
}

class _AnimatedContainerBasicDemoState
    extends State<AnimatedContainerBasicDemo> {
  bool _toggled = false;

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('AnimatedContainer 基础')),
      body: Center(
        child: AnimatedContainer(
          duration: const Duration(milliseconds: 600),
          curve: Curves.easeInOut,
          width: _toggled ? 250 : 120,
          height: _toggled ? 250 : 120,
          decoration: BoxDecoration(
            color: _toggled ? Colors.deepPurple : Colors.teal,
            borderRadius: BorderRadius.circular(_toggled ? 32 : 8),
          ),
          // 动画结束时打印日志
          onEnd: () => debugPrint('AnimatedContainer 动画完成'),
          child: const Center(
            child: Text(
              '点我变化',
              style: TextStyle(color: Colors.white, fontSize: 16),
            ),
          ),
        ),
      ),
      floatingActionButton: FloatingActionButton(
        onPressed: () => setState(() => _toggled = !_toggled),
        child: const Icon(Icons.refresh),
      ),
    );
  }
}

2.3 进阶用法：多属性同时动画

AnimatedContainer 的强大之处在于可以同时对多个属性做动画，所有属性的变化会在同一个 duration 内同步完成。

import 'package:flutter/material.dart';
import 'dart:math';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: const AnimatedContainerAdvancedDemo(),
    );
  }
}

class AnimatedContainerAdvancedDemo extends StatefulWidget {
  const AnimatedContainerAdvancedDemo({super.key});

  @override
  State<AnimatedContainerAdvancedDemo> createState() =>
      _AnimatedContainerAdvancedDemoState();
}

class _AnimatedContainerAdvancedDemoState
    extends State<AnimatedContainerAdvancedDemo> {
  bool _toggled = false;

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('AnimatedContainer 多属性动画')),
      body: Center(
        child: AnimatedContainer(
          duration: const Duration(milliseconds: 800),
          curve: Curves.easeInOutCubic,
          width: _toggled ? 280 : 150,
          height: _toggled ? 280 : 150,
          // padding 动画
          padding: EdgeInsets.all(_toggled ? 24 : 8),
          // margin 动画
          margin: EdgeInsets.all(_toggled ? 10 : 40),
          // alignment 动画
          alignment: _toggled ? Alignment.bottomRight : Alignment.topLeft,
          // decoration 动画（颜色、圆角、阴影同时变化）
          decoration: BoxDecoration(
            color: _toggled ? Colors.indigo : Colors.orange,
            borderRadius: BorderRadius.circular(_toggled ? 40 : 12),
            boxShadow: [
              BoxShadow(
                color: (_toggled ? Colors.indigo : Colors.orange)
                    .withOpacity(0.4),
                blurRadius: _toggled ? 20 : 4,
                offset: Offset(0, _toggled ? 10 : 2),
              ),
            ],
          ),
          // transform 动画：旋转
          transform: _toggled
              ? (Matrix4.identity()..rotateZ(pi / 12))
              : Matrix4.identity(),
          transformAlignment: Alignment.center,
          child: const Icon(
            Icons.star,
            color: Colors.white,
            size: 48,
          ),
        ),
      ),
      floatingActionButton: FloatingActionButton(
        onPressed: () => setState(() => _toggled = !_toggled),
        child: Icon(_toggled ? Icons.undo : Icons.play_arrow),
      ),
    );
  }
}

2.4 实战：动画卡片展开效果

import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      theme: ThemeData(useMaterial3: true, colorSchemeSeed: Colors.blue),
      home: const ExpandableCardDemo(),
    );
  }
}

class ExpandableCardDemo extends StatefulWidget {
  const ExpandableCardDemo({super.key});

  @override
  State<ExpandableCardDemo> createState() => _ExpandableCardDemoState();
}

class _ExpandableCardDemoState extends State<ExpandableCardDemo> {
  bool _expanded = false;

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('动画卡片展开')),
      body: Padding(
        padding: const EdgeInsets.all(16),
        child: GestureDetector(
          onTap: () => setState(() => _expanded = !_expanded),
          child: AnimatedContainer(
            duration: const Duration(milliseconds: 500),
            curve: Curves.easeInOut,
            width: double.infinity,
            height: _expanded ? 260 : 100,
            padding: const EdgeInsets.all(16),
            decoration: BoxDecoration(
              color: _expanded
                  ? Theme.of(context).colorScheme.primaryContainer
                  : Theme.of(context).colorScheme.surfaceContainerHighest,
              borderRadius: BorderRadius.circular(_expanded ? 24 : 12),
              boxShadow: [
                BoxShadow(
                  color: Colors.black.withOpacity(_expanded ? 0.15 : 0.05),
                  blurRadius: _expanded ? 16 : 4,
                  offset: Offset(0, _expanded ? 8 : 2),
                ),
              ],
            ),
            child: Column(
              crossAxisAlignment: CrossAxisAlignment.start,
              children: [
                Row(
                  children: [
                    const Icon(Icons.info_outline),
                    const SizedBox(width: 8),
                    const Text(
                      '点击展开 / 收起',
                      style: TextStyle(
                        fontSize: 18,
                        fontWeight: FontWeight.w600,
                      ),
                    ),
                    const Spacer(),
                    // 图标也可以用 AnimatedRotation 做旋转动画
                    AnimatedRotation(
                      turns: _expanded ? 0.5 : 0,
                      duration: const Duration(milliseconds: 500),
                      child: const Icon(Icons.expand_more),
                    ),
                  ],
                ),
                // 展开时显示的详细内容
                if (_expanded) ...[
                  const SizedBox(height: 16),
                  const Text(
                    'AnimatedContainer 能同时对宽度、高度、颜色、圆角、阴影等属性做平滑过渡动画。'
                    '只需在 setState 中修改属性值，剩下的工作全部交给 Flutter。',
                    style: TextStyle(fontSize: 14, height: 1.6),
                  ),
                ],
              ],
            ),
          ),
        ),
      ),
    );
  }
}

注意事项：

  1. AnimatedContainer 的 color 参数和 decoration 中的 color 不能同时使用，否则会抛出异常。需要颜色动画时，建议统一放在 BoxDecoration 里。
  2. transform 动画只对矩阵中可以线性插值的部分有效。如果旧值和新值的矩阵结构差异过大，可能出现非预期的过渡效果。
  3. AnimatedContainer 不会对 child 做动画。如果需要对子组件做切换动画，请使用 AnimatedSwitcher。

3. AnimatedOpacity

AnimatedOpacity 用于对组件的透明度做平滑过渡，是实现淡入/淡出效果最简单的方式。

3.1 核心属性

属性 类型 说明
opacity double 目标透明度，取值 0.0（完全透明）到 1.0（完全不透明）
duration Duration 动画时长
curve Curve 动画曲线
onEnd VoidCallback? 动画结束回调
child Widget 子组件

3.2 代码示例：淡入淡出效果

import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: const AnimatedOpacityDemo(),
    );
  }
}

class AnimatedOpacityDemo extends StatefulWidget {
  const AnimatedOpacityDemo({super.key});

  @override
  State<AnimatedOpacityDemo> createState() => _AnimatedOpacityDemoState();
}

class _AnimatedOpacityDemoState extends State<AnimatedOpacityDemo> {
  double _opacity = 1.0;

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('AnimatedOpacity')),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: [
            AnimatedOpacity(
              opacity: _opacity,
              duration: const Duration(milliseconds: 500),
              curve: Curves.easeInOut,
              // 动画结束时执行回调
              onEnd: () => debugPrint('透明度动画完成，当前值: $_opacity'),
              child: Container(
                width: 200,
                height: 200,
                decoration: BoxDecoration(
                  color: Colors.blue,
                  borderRadius: BorderRadius.circular(16),
                ),
                child: const Center(
                  child: Text(
                    '淡入淡出',
                    style: TextStyle(color: Colors.white, fontSize: 20),
                  ),
                ),
              ),
            ),
            const SizedBox(height: 32),
            Row(
              mainAxisAlignment: MainAxisAlignment.center,
              children: [
                ElevatedButton(
                  onPressed: () => setState(() => _opacity = 0.0),
                  child: const Text('淡出'),
                ),
                const SizedBox(width: 16),
                ElevatedButton(
                  onPressed: () => setState(() => _opacity = 1.0),
                  child: const Text('淡入'),
                ),
                const SizedBox(width: 16),
                ElevatedButton(
                  onPressed: () => setState(() => _opacity = 0.3),
                  child: const Text('半透明'),
                ),
              ],
            ),
          ],
        ),
      ),
    );
  }
}

最佳实践：

  • 如果只是想隐藏组件，AnimatedOpacity 配合 opacity: 0.0 虽然视觉上不可见，但组件仍然占据布局空间并且可以接收手势。如果需要完全移除组件，应使用 AnimatedCrossFade 或 AnimatedSwitcher 配合 SizedBox.shrink()。
  • Opacity 组件（非动画版本）在 opacity 不为 0 或 1 时，会触发离屏缓冲（offscreen buffer），可能影响性能。AnimatedOpacity 同理——尽量避免在大量组件上同时使用半透明动画。

5. AnimatedPositioned

AnimatedPositioned 用于在 Stack 中对子组件的位置做平滑过渡动画。它是 Positioned 的隐式动画版本。

5.1 核心属性

属性 类型 说明
left double? 距左边距离
top double? 距顶部距离
right double? 距右边距离
bottom double? 距底部距离
width double? 宽度
height double? 高度
duration Duration 动画时长
curve Curve 动画曲线
onEnd VoidCallback? 动画结束回调

5.2 代码示例：在 Stack 中移动元素

import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: const AnimatedPositionedDemo(),
    );
  }
}

class AnimatedPositionedDemo extends StatefulWidget {
  const AnimatedPositionedDemo({super.key});

  @override
  State<AnimatedPositionedDemo> createState() =>
      _AnimatedPositionedDemoState();
}

class _AnimatedPositionedDemoState extends State<AnimatedPositionedDemo> {
  bool _moved = false;

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('AnimatedPositioned')),
      body: Stack(
        children: [
          // 背景网格辅助可视化
          Positioned.fill(
            child: CustomPaint(painter: _GridPainter()),
          ),
          AnimatedPositioned(
            // 在两个位置之间切换
            left: _moved ? 200 : 20,
            top: _moved ? 400 : 40,
            width: _moved ? 120 : 80,
            height: _moved ? 120 : 80,
            duration: const Duration(milliseconds: 600),
            curve: Curves.easeInOutBack,
            child: GestureDetector(
              onTap: () => setState(() => _moved = !_moved),
              child: Container(
                decoration: BoxDecoration(
                  color: Colors.deepOrange,
                  borderRadius: BorderRadius.circular(_moved ? 60 : 12),
                  boxShadow: const [
                    BoxShadow(
                      color: Colors.black26,
                      blurRadius: 8,
                      offset: Offset(2, 4),
                    ),
                  ],
                ),
                child: const Center(
                  child: Text(
                    '点我移动',
                    style: TextStyle(
                      color: Colors.white,
                      fontWeight: FontWeight.bold,
                    ),
                  ),
                ),
              ),
            ),
          ),
        ],
      ),
    );
  }
}

/// 绘制背景网格，便于观察位置变化
class _GridPainter extends CustomPainter {
  @override
  void paint(Canvas canvas, Size size) {
    final paint = Paint()
      ..color = Colors.grey.shade200
      ..strokeWidth = 0.5;
    for (double x = 0; x < size.width; x += 40) {
      canvas.drawLine(Offset(x, 0), Offset(x, size.height), paint);
    }
    for (double y = 0; y < size.height; y += 40) {
      canvas.drawLine(Offset(0, y), Offset(size.width, y), paint);
    }
  }

  @override
  bool shouldRepaint(covariant CustomPainter oldDelegate) => false;
}

注意事项：

  • AnimatedPositioned 必须放在 Stack 中使用，否则会报错。
  • 和 Positioned 一样，left/right/width 三个属性中最多只能指定两个（竖直方向同理）。如果三个都指定了会产生约束冲突。
  • 如果某个定位属性从 null 变为非 null（或反过来），不会产生动画——只有从一个数值变为另一个数值时才会做动画。

6. AnimatedCrossFade

AnimatedCrossFade 在两个子组件之间做交叉淡入淡出切换——一个淡出的同时另一个淡入，尺寸也会平滑过渡。

6.1 核心属性

属性 类型 说明
firstChild Widget 第一个子组件
secondChild Widget 第二个子组件
crossFadeState CrossFadeState 当前显示哪个子组件（.showFirst 或 .showSecond）
duration Duration 动画时长
firstCurve Curve 第一个子组件的动画曲线，默认 Curves.linear
secondCurve Curve 第二个子组件的动画曲线，默认 Curves.linear
sizeCurve Curve 尺寸变化的动画曲线，默认 Curves.linear
layoutBuilder AnimatedCrossFadeBuilder? 自定义两个子组件的叠放方式

6.2 代码示例

import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: const AnimatedCrossFadeDemo(),
    );
  }
}

class AnimatedCrossFadeDemo extends StatefulWidget {
  const AnimatedCrossFadeDemo({super.key});

  @override
  State<AnimatedCrossFadeDemo> createState() => _AnimatedCrossFadeDemoState();
}

class _AnimatedCrossFadeDemoState extends State<AnimatedCrossFadeDemo> {
  bool _showFirst = true;

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('AnimatedCrossFade')),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: [
            AnimatedCrossFade(
              // 第一个子组件：紧凑状态
              firstChild: Container(
                width: 200,
                height: 80,
                decoration: BoxDecoration(
                  color: Colors.blue.shade100,
                  borderRadius: BorderRadius.circular(12),
                ),
                child: const Center(
                  child: Text('收起状态', style: TextStyle(fontSize: 18)),
                ),
              ),
              // 第二个子组件：展开状态
              secondChild: Container(
                width: 300,
                height: 200,
                decoration: BoxDecoration(
                  color: Colors.green.shade100,
                  borderRadius: BorderRadius.circular(12),
                ),
                child: const Padding(
                  padding: EdgeInsets.all(16),
                  child: Column(
                    mainAxisAlignment: MainAxisAlignment.center,
                    children: [
                      Icon(Icons.check_circle, size: 48, color: Colors.green),
                      SizedBox(height: 12),
                      Text('展开状态', style: TextStyle(fontSize: 18)),
                      SizedBox(height: 8),
                      Text('两个子组件交叉淡入淡出，\n尺寸也会平滑过渡'),
                    ],
                  ),
                ),
              ),
              // 通过此属性控制显示哪个子组件
              crossFadeState: _showFirst
                  ? CrossFadeState.showFirst
                  : CrossFadeState.showSecond,
              duration: const Duration(milliseconds: 400),
              firstCurve: Curves.easeIn,
              secondCurve: Curves.easeOut,
              sizeCurve: Curves.easeInOut,
            ),
            const SizedBox(height: 32),
            ElevatedButton.icon(
              onPressed: () => setState(() => _showFirst = !_showFirst),
              icon: Icon(_showFirst ? Icons.unfold_more : Icons.unfold_less),
              label: Text(_showFirst ? '展开' : '收起'),
            ),
          ],
        ),
      ),
    );
  }
}

最佳实践：

  • AnimatedCrossFade 适合在两个固定子组件之间切换。如果需要在多个不同子组件之间切换，使用 AnimatedSwitcher 更合适。
  • 当两个子组件尺寸差异很大时，sizeCurve 可以让尺寸变化更自然。建议使用 Curves.easeInOut 而不是默认的 linear。

7. AnimatedSwitcher

AnimatedSwitcher 是最灵活的子组件切换动画组件。当 child 发生变化时（通过 key 区分），旧组件执行退出动画，新组件执行进入动画。

7.1 核心属性

属性 类型 说明
child Widget? 当前子组件（通过 key 区分新旧）
duration Duration 动画时长
reverseDuration Duration? 反向动画时长（退出动画），默认等于 duration
transitionBuilder AnimatedSwitcherTransitionBuilder 过渡效果构建器，默认为 FadeTransition
layoutBuilder AnimatedSwitcherLayoutBuilder 新旧子组件的叠放布局方式

7.2 关键机制：用 key 触发切换

AnimatedSwitcher 通过比较 child 的 key（或 runtimeType）来判断是否需要做切换动画。这是最容易犯错的地方——如果新旧 child 的 key 和 runtimeType 都相同，不会触发动画。

// ❌ 错误：Text 类型没变，文本变了但 key 相同（默认 null），不会触发动画
AnimatedSwitcher(
  duration: const Duration(milliseconds: 300),
  child: Text('$_count'),
)

// ✅ 正确：使用 ValueKey 让 Flutter 识别这是一个新的子组件
AnimatedSwitcher(
  duration: const Duration(milliseconds: 300),
  child: Text('$_count', key: ValueKey<int>(_count)),
)

10. TweenAnimationBuilder

TweenAnimationBuilder 是隐式动画体系中最灵活的组件。前面的组件都是针对特定属性做动画，而 TweenAnimationBuilder 可以对任意类型的值做动画——只要该类型支持插值计算。

10.1 核心属性

属性 类型 说明
tween Tween<T> 定义动画的起始值和结束值
duration Duration 动画时长
curve Curve 动画曲线
builder ValueWidgetBuilder<T> 根据当前动画值构建 UI，签名为 Widget Function(BuildContext, T value, Widget? child)
child Widget? 不随动画变化的子组件（性能优化，避免不必要的重建）
onEnd VoidCallback? 动画结束回调

10.2 工作机制

  1. 首次构建时，TweenAnimationBuilder 立即从 tween.begin 动画到 tween.end
  2. 后续重建时，如果 tween.end 发生变化，会从当前值自动过渡到新的 tween.end
  3. builder 在每一帧被调用，接收当前插值，用于构建 UI

import 'dart:math';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {

  }
}

class CircularProgressAnimationDemo extends StatefulWidget {

  @override
  State<CircularProgressAnimationDemo> createState() =>
      _CircularProgressAnimationDemoState();
}

class _CircularProgressAnimationDemoState
    extends State<CircularProgressAnimationDemo> {
  double _progress = 0.0;

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('TweenAnimationBuilder 自定义绘制')),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: [

              curve: Curves.easeInOut,
              builder: (context, value, child) {
                return SizedBox(
                  width: 200,
                  height: 200,
                  child: CustomPaint(
                    painter: _CircularProgressPainter(progress: value),
                    child: Center(
                      child: Text(
                        '${(value * 100).toInt()}%',
                        style: const TextStyle(
                          fontSize: 36,
                          fontWeight: FontWeight.bold,
                        ),
                      ),
                    ),
                  ),
                );
              },
            ),
            const SizedBox(height: 32),
            Wrap(
              spacing: 12,
              children: [0.0, 0.25, 0.5, 0.75, 1.0].map((v) {
                return ElevatedButton(
                  onPressed: () => setState(() => _progress = v),
                  child: Text('${(v * 100).toInt()}%'),
                );
              }).toList(),
            ),
          ],
        ),
      ),
    );
  }
}

/// 自定义画笔：绘制圆形进度条
class _CircularProgressPainter extends CustomPainter {
  final double progress;

  _CircularProgressPainter({required this.progress});

  @override
  void paint(Canvas canvas, Size size) {
    final center = Offset(size.width / 2, size.height / 2);
    final radius = size.width / 2 - 12;

    // 背景圆环
    final bgPaint = Paint()
      ..color = Colors.grey.shade200
      ..style = PaintingStyle.stroke
      ..strokeWidth = 12
      ..strokeCap = StrokeCap.round;
    canvas.drawCircle(center, radius, bgPaint);

    // 进度圆弧
    final progressPaint = Paint()
      ..color = Colors.blue
      ..style = PaintingStyle.stroke
      ..strokeWidth = 12
      ..strokeCap = StrokeCap.round;
    canvas.drawArc(
      Rect.fromCircle(center: center, radius: radius),
      -pi / 2, // 从顶部开始
      2 * pi * progress,
      false,
      progressPaint,
    );
  }

  @override
  bool shouldRepaint(_CircularProgressPainter oldDelegate) =>
      oldDelegate.progress != progress;
}

10.6 builder 中的 child 参数

TweenAnimationBuilder 的 child 参数是一个重要的性能优化手段：

TweenAnimationBuilder<double>(
  tween: Tween(begin: 0.0, end: 1.0),
  duration: const Duration(seconds: 1),
  // child 放在这里：只创建一次，不会随动画每帧重建
  child: const MyExpensiveWidget(),
  builder: (context, value, child) {
    return Opacity(
      opacity: value,
      // 直接使用透传过来的 child
      child: child,
    );
  },
)

builder 在动画的每一帧都会被调用（比如 60fps 下每秒调用 60 次）。如果 builder 内部有复杂的子组件树，每帧都重建会造成性能浪费。将不变的部分放到外层 child 参数中，builder 通过第三个参数接收即可。

最佳实践：

  1. 始终优先设置 tween.end：TweenAnimationBuilder 首次构建时会立即从 begin 播放到 end。如果你不想首次就有动画，可以将 begin 和 end 设为相同值，之后再通过 setState 改变 end。
  2. 善用 child 参数：将不随动画变化的组件放在 child 中，避免不必要的重建。
  3. Tween 类型丰富：除了 Tween<double> 和 ColorTween，Flutter 还提供了 SizeTween、RectTween、AlignmentTween、DecorationTween、EdgeInsetsTween 等。
  4. onEnd 回调：可以在动画结束后执行额外操作，例如触发下一段动画、更新状态等。

11. Curves 常用动画曲线

动画曲线（Curve）决定了动画值随时间的变化规律。所有隐式动画组件都支持 curve 参数。

11.1 常用曲线一览

曲线 效果描述 适用场景
Curves.linear 匀速运动 进度条、计时器
Curves.easeIn 缓入（慢→快） 元素消失、移出屏幕
Curves.easeOut 缓出（快→慢） 元素出现、移入屏幕
Curves.easeInOut 缓入缓出（慢→快→慢） 通用过渡，最自然的效果
Curves.ease 类似 easeInOut 但略有不同 CSS 默认动画曲线
Curves.bounceIn 弹跳进入 趣味性、游戏化 UI
Curves.bounceOut 弹跳结束 元素"落地弹跳"效果
Curves.bounceInOut 两端弹跳 强调弹性效果
Curves.elasticIn 弹性进入（有回弹） 弹性拉伸效果
Curves.elasticOut 弹性结束（有回弹） 弹性释放效果
Curves.elasticInOut 两端弹性 强弹性效果
Curves.decelerate 持续减速 自然减速
Curves.fastOutSlowIn Material Design 标准曲线 Material 风格过渡
Curves.easeInOutCubic 三次方缓入缓出 平滑的通用过渡
Curves.easeInOutBack 缓入缓出带过冲 活泼的过渡效果

11.3 自定义曲线

除了内置曲线，你还可以使用 Cubic 创建自定义的三次贝塞尔曲线（类似 CSS 的 cubic-bezier），或者继承 Curve 类完全自定义：

// 使用三次贝塞尔曲线（与 CSS cubic-bezier 参数一致）
const myCustomCurve = Cubic(0.68, -0.55, 0.27, 1.55);

// 使用 Interval 截取曲线的一段（常用于交错动画）
const firstHalf = Interval(0.0, 0.5, curve: Curves.easeIn);
const secondHalf = Interval(0.5, 1.0, curve: Curves.easeOut);

// 翻转曲线
final reversedCurve = Curves.easeIn.flipped; // 等价于 easeOut

最佳实践：

  • 通用过渡选择 Curves.easeInOut 或 Curves.fastOutSlowIn（Material Design 推荐）。
  • 元素出现用 easeOut（快速出现，缓慢就位），消失用 easeIn（缓慢起步，快速消失）。
  • bounceOut 和 elasticOut 很有趣，但不要在严肃的商务应用中滥用。
  • 动画时长一般建议在 200ms ~ 500ms 之间，太短用户感知不到，太长影响效率感。

本课小结

本课系统讲解了 Flutter 隐式动画体系的核心知识：

  1. 隐式动画原理：只需声明目标属性值，Flutter 自动处理从旧值到新值的平滑过渡。所有隐式动画组件继承自 ImplicitlyAnimatedWidget，内部自动管理 AnimationController。

  2. AnimatedContainer：最强大的隐式动画组件，可以同时对尺寸、颜色、圆角、阴影、内外边距、对齐、变换等多种属性做动画。

  3. AnimatedOpacity：透明度渐变动画，适合做淡入淡出效果，注意透明度为 0 时组件仍然占据空间。

  4. AnimatedPadding：间距动画，常配合滑块等交互使用。

  5. AnimatedPositioned：在 Stack 中做位置动画，注意必须放在 Stack 内，且定位属性不能从 null 变为数值。

  6. AnimatedCrossFade：在两个子组件之间做交叉淡入淡出，支持独立的进入/退出/尺寸曲线。

  7. AnimatedSwitcher：最灵活的切换动画组件，通过 key 触发切换，支持自定义 transitionBuilder。

  8. AnimatedDefaultTextStyle 和 AnimatedPhysicalModel：分别用于文本样式和物理模型（阴影、形状）的动画。

  9. TweenAnimationBuilder：自定义隐式动画的利器，可以对任意可插值类型做动画，通过 builder 回调驱动 UI 更新，child 参数用于性能优化。

  10. Curves：动画曲线决定了动画的节奏感，easeInOut 和 fastOutSlowIn 是最常用的选择。

一句话总结：隐式动画 = 改属性 + 设时长，Flutter 帮你搞定其余一切。
