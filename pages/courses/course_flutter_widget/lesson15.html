第15课 选择类组件

课程系列：Flutter 常用组件系统学习
适用人群：有 Dart 语言基础，了解 Flutter 基本概念的开发者
课程目标：系统掌握 Flutter 中各类选择组件的使用方式、状态管理方法和适用场景

选择类组件是用户与应用交互的核心手段之一。无论是勾选协议、切换设置、选择日期还是从下拉列表中选值，都离不开这些组件。本课将逐一讲解 Flutter 中最常用的选择组件，帮助你在不同场景下做出正确的组件选型。

1. Checkbox 与 CheckboxListTile

1.1 使用场景

  • Checkbox 适用于需要用户做出"是/否"二选一确认的场景，例如"同意用户协议"、"记住密码"。
  • CheckboxListTile 是 Checkbox 与 ListTile 的组合，适用于设置列表中需要在文字旁显示勾选框的场景，点击整行即可切换状态。

1.2 核心属性

属性 类型 说明
value bool? 当前勾选状态。true=选中，false=未选中，null=中间态（需启用 tristate）
onChanged ValueChanged<bool?>? 状态变化回调，设为 null 则禁用
tristate bool 是否支持三态（默认 false）
activeColor Color? 选中时的填充颜色
checkColor Color? 选中时对勾的颜色

1.3 基础用法

import 'package:flutter/material.dart';

void main() => runApp(const MaterialApp(home: CheckboxDemoPage()));

class CheckboxDemoPage extends StatefulWidget {
  const CheckboxDemoPage({super.key});

  @override
  State<CheckboxDemoPage> createState() => _CheckboxDemoPageState();
}

class _CheckboxDemoPageState extends State<CheckboxDemoPage> {
  bool _agreeTerms = false;

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('Checkbox 演示')),
      body: Padding(
        padding: const EdgeInsets.all(16),
        child: Column(
          crossAxisAlignment: CrossAxisAlignment.start,
          children: [
            // 基础 Checkbox：通常需要搭配 Text 一起使用
            Row(
              children: [
                Checkbox(
                  value: _agreeTerms,
                  activeColor: Colors.teal,
                  checkColor: Colors.white,
                  onChanged: (bool? newValue) {
                    setState(() {
                      _agreeTerms = newValue ?? false;
                    });
                  },
                ),
                GestureDetector(
                  onTap: () => setState(() => _agreeTerms = !_agreeTerms),
                  child: const Text('我已阅读并同意用户协议'),
                ),
              ],
            ),
            const SizedBox(height: 16),
            ElevatedButton(
              // 未勾选时禁用按钮
              onPressed: _agreeTerms ? () {} : null,
              child: const Text('注册'),
            ),
          ],
        ),
      ),
    );
  }
}

1.4 三态 Checkbox（tristate）

三态复选框在"全选/部分选/全不选"场景中非常有用，例如管理子项的批量勾选。

import 'package:flutter/material.dart';

void main() => runApp(const MaterialApp(home: TristateCheckboxPage()));

class TristateCheckboxPage extends StatefulWidget {
  const TristateCheckboxPage({super.key});

  @override
  State<TristateCheckboxPage> createState() => _TristateCheckboxPageState();
}

class _TristateCheckboxPageState extends State<TristateCheckboxPage> {
  // 子项选中状态
  final List<bool> _items = [true, false, false];
  final List<String> _labels = ['苹果', '香蕉', '橙子'];

  // 计算父级复选框的状态
  bool? get _parentValue {
    final allSelected = _items.every((v) => v);
    final noneSelected = _items.every((v) => !v);
    if (allSelected) return true;
    if (noneSelected) return false;
    return null; // 部分选中 → 中间态
  }

  void _onParentChanged(bool? value) {
    setState(() {
      // 点击父级时：如果当前是部分选中或未选中，则全选；否则全不选
      final newValue = value ?? false;
      for (int i = 0; i < _items.length; i++) {
        _items[i] = newValue;
      }
    });
  }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('三态 Checkbox 演示')),
      body: Padding(
        padding: const EdgeInsets.all(16),
        child: Column(
          children: [
            // 父级三态复选框
            CheckboxListTile(
              title: const Text('全选水果',
                  style: TextStyle(fontWeight: FontWeight.bold)),
              value: _parentValue,
              tristate: true, // 启用三态
              onChanged: _onParentChanged,
            ),
            const Divider(),
            // 子项复选框
            ...List.generate(_items.length, (index) {
              return CheckboxListTile(
                title: Text(_labels[index]),
                value: _items[index],
                activeColor: Colors.green,
                onChanged: (bool? value) {
                  setState(() {
                    _items[index] = value ?? false;
                  });
                },
              );
            }),
          ],
        ),
      ),
    );
  }
}

注意事项：

  1. Checkbox 本身不维护状态，必须通过 setState 在 onChanged 回调中更新 value。
  2. CheckboxListTile 整行可点击，用户体验优于单独的 Checkbox + Text 组合。
  3. 三态模式下 value 为 null 时显示横线（dash），表示"部分选中"。

2. Radio 与 RadioListTile

2.1 使用场景

Radio 用于互斥的单选场景，即一组选项中只能选择一个。典型场景包括：性别选择、支付方式选择、主题模式选择等。

2.2 核心属性

属性 类型 说明
value T 当前 Radio 代表的值
groupValue T? 当前选中的值（整组共享同一个 groupValue）
onChanged ValueChanged<T?>? 选中时的回调

Radio 是一个泛型类 Radio<T>，T 可以是 int、String、enum 等任意类型。推荐使用枚举来保证类型安全。

2.3 使用枚举的 Radio 示例

import 'package:flutter/material.dart';

// 定义枚举，让选项具有类型安全
enum PaymentMethod { alipay, wechat, bankCard }

void main() => runApp(const MaterialApp(home: RadioDemoPage()));

class RadioDemoPage extends StatefulWidget {
  const RadioDemoPage({super.key});

  @override
  State<RadioDemoPage> createState() => _RadioDemoPageState();
}

class _RadioDemoPageState extends State<RadioDemoPage> {
  // groupValue 代表当前选中的支付方式
  PaymentMethod? _selectedPayment;

  String _paymentLabel(PaymentMethod method) {
    return switch (method) {
      PaymentMethod.alipay => '支付宝',
      PaymentMethod.wechat => '微信支付',
      PaymentMethod.bankCard => '银行卡',
    };
  }

  IconData _paymentIcon(PaymentMethod method) {
    return switch (method) {
      PaymentMethod.alipay => Icons.account_balance_wallet,
      PaymentMethod.wechat => Icons.chat,
      PaymentMethod.bankCard => Icons.credit_card,
    };
  }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('Radio 演示')),
      body: Padding(
        padding: const EdgeInsets.all(16),
        child: Column(
          crossAxisAlignment: CrossAxisAlignment.start,
          children: [
            Text('选择支付方式', style: Theme.of(context).textTheme.titleMedium),
            const SizedBox(height: 12),

            // 使用 RadioListTile 构建单选列表
            ...PaymentMethod.values.map((method) {
              return RadioListTile<PaymentMethod>(
                title: Text(_paymentLabel(method)),
                secondary: Icon(_paymentIcon(method)),
                value: method, // 此 Radio 代表的值
                groupValue: _selectedPayment, // 当前选中值
                onChanged: (PaymentMethod? value) {
                  setState(() {
                    _selectedPayment = value;
                  });
                },
              );
            }),

            const Divider(height: 32),
            Text(
              _selectedPayment != null
                  ? '已选择：${_paymentLabel(_selectedPayment!)}'
                  : '请选择支付方式',
              style: const TextStyle(fontSize: 16),
            ),
          ],
        ),
      ),
    );
  }
}

2.4 使用裸 Radio 的场景

当你需要自定义布局而不想使用 ListTile 样式时，可以直接使用 Radio 组件：

import 'package:flutter/material.dart';

void main() => runApp(const MaterialApp(home: RawRadioDemoPage()));

class RawRadioDemoPage extends StatefulWidget {
  const RawRadioDemoPage({super.key});

  @override
  State<RawRadioDemoPage> createState() => _RawRadioDemoPageState();
}

class _RawRadioDemoPageState extends State<RawRadioDemoPage> {
  int _selectedSize = 0;
  final List<String> _sizes = ['S', 'M', 'L', 'XL', 'XXL'];

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('裸 Radio 自定义布局')),
      body: Padding(
        padding: const EdgeInsets.all(16),
        child: Column(
          crossAxisAlignment: CrossAxisAlignment.start,
          children: [
            const Text('选择尺码：', style: TextStyle(fontSize: 16)),
            const SizedBox(height: 8),
            // 横向排列的 Radio 组
            Row(
              children: List.generate(_sizes.length, (index) {
                return Row(
                  mainAxisSize: MainAxisSize.min,
                  children: [
                    Radio<int>(
                      value: index,
                      groupValue: _selectedSize,
                      onChanged: (int? value) {
                        setState(() => _selectedSize = value!);
                      },
                    ),
                    Text(_sizes[index]),
                    const SizedBox(width: 8),
                  ],
                );
              }),
            ),
          ],
        ),
      ),
    );
  }
}

最佳实践：

  1. 优先使用枚举作为 Radio 的泛型类型，避免使用魔法数字或字符串。
  2. 当选项超过 5 个时，考虑使用 DropdownButton 或 DropdownMenu 代替 Radio，避免界面过于拥挤。
  3. RadioListTile 整行可点击，比 Radio + Text 的组合体验更好。

3. Switch 与 SwitchListTile

3.1 使用场景

Switch 用于开关型的二选一切换，典型场景：开启/关闭通知、启用深色模式、Wi-Fi 开关等。它与 Checkbox 的语义区别在于：Checkbox 表示"选中/不选中"，Switch 表示"开启/关闭"。

3.2 核心属性

属性 类型 说明
value bool 当前开关状态
onChanged ValueChanged<bool>? 状态变化回调
activeColor Color? 开启时滑块（thumb）的颜色
activeTrackColor Color? 开启时轨道（track）的颜色
inactiveThumbColor Color? 关闭时滑块的颜色
inactiveTrackColor Color? 关闭时轨道的颜色

3.3 完整示例

import 'package:flutter/material.dart';

void main() => runApp(const MaterialApp(home: SwitchDemoPage()));

class SwitchDemoPage extends StatefulWidget {
  const SwitchDemoPage({super.key});

  @override
  State<SwitchDemoPage> createState() => _SwitchDemoPageState();
}

class _SwitchDemoPageState extends State<SwitchDemoPage> {
  bool _darkMode = false;
  bool _notifications = true;
  bool _autoUpdate = false;
  bool _locationService = true;

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('Switch 演示')),
      body: ListView(
        children: [
          // 基础 Switch + 自定义颜色
          Padding(
            padding: const EdgeInsets.all(16),
            child: Row(
              mainAxisAlignment: MainAxisAlignment.spaceBetween,
              children: [
                const Text('深色模式（自定义颜色）', style: TextStyle(fontSize: 16)),
                Switch(
                  value: _darkMode,
                  activeColor: Colors.amber,
                  activeTrackColor: Colors.amber.shade100,
                  inactiveThumbColor: Colors.grey,
                  inactiveTrackColor: Colors.grey.shade300,
                  onChanged: (bool value) {
                    setState(() => _darkMode = value);
                  },
                ),
              ],
            ),
          ),
          const Divider(),

          // SwitchListTile：设置列表中最常用的形式
          SwitchListTile(
            title: const Text('推送通知'),
            subtitle: const Text('接收新消息和活动提醒'),
            secondary: const Icon(Icons.notifications),
            value: _notifications,
            onChanged: (bool value) {
              setState(() => _notifications = value);
            },
          ),
          SwitchListTile(
            title: const Text('自动更新'),
            subtitle: const Text('在 Wi-Fi 环境下自动更新应用'),
            secondary: const Icon(Icons.system_update),
            value: _autoUpdate,
            onChanged: (bool value) {
              setState(() => _autoUpdate = value);
            },
          ),
          SwitchListTile(
            title: const Text('定位服务'),
            subtitle: const Text('允许应用获取您的位置信息'),
            secondary: const Icon(Icons.location_on),
            value: _locationService,
            onChanged: (bool value) {
              setState(() => _locationService = value);
            },
          ),

          // 禁用状态
          SwitchListTile(
            title: const Text('飞行模式（禁用示例）'),
            secondary: const Icon(Icons.airplanemode_active),
            value: false,
            onChanged: null, // 设为 null 即禁用
          ),
        ],
      ),
    );
  }
}

注意事项：

  1. Material 3 中的 Switch 外观发生了较大变化（更圆润、更大），可通过 SwitchThemeData 统一定制样式。
  2. Switch 和 Checkbox 语义不同：用户期望 Switch 代表"即时生效的操作"（如开关 Wi-Fi），而 Checkbox 代表"标记/勾选"（如同意协议后再提交）。
  3. SwitchListTile 的 secondary 属性用于放置前导图标（等价于 ListTile 的 leading）。

4. Slider 与 RangeSlider

4.1 使用场景

  • Slider 用于在一个连续或离散的范围内选取单个值，如音量调节、亮度调节、字体大小选择。
  • RangeSlider 用于选取一个区间（两个值），如价格范围筛选、日期范围筛选。

4.2 核心属性

属性 说明
value / values Slider 用 double value；RangeSlider 用 RangeValues values
onChanged 值变化回调
min / max 范围的最小值和最大值（默认 0.0 ~ 1.0）
divisions 将范围等分为多少段（不设则为连续滑动）
label 滑块上方的气泡标签文字
activeColor 已选区间的颜色
inactiveColor 未选区间的颜色

4.3 Slider 示例

import 'package:flutter/material.dart';

void main() => runApp(const MaterialApp(home: SliderDemoPage()));

class SliderDemoPage extends StatefulWidget {
  const SliderDemoPage({super.key});

  @override
  State<SliderDemoPage> createState() => _SliderDemoPageState();
}

class _SliderDemoPageState extends State<SliderDemoPage> {
  double _fontSize = 16.0;
  double _volume = 50;

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('Slider 演示')),
      body: Padding(
        padding: const EdgeInsets.all(16),
        child: Column(
          crossAxisAlignment: CrossAxisAlignment.start,
          children: [
            // 连续 Slider：字体大小调节
            Text('字体大小：${_fontSize.toStringAsFixed(1)}',
                style: Theme.of(context).textTheme.titleMedium),
            Slider(
              value: _fontSize,
              min: 10,
              max: 40,
              activeColor: Colors.deepPurple,
              onChanged: (double value) {
                setState(() => _fontSize = value);
              },
            ),
            // 实时预览效果
            Text(
              '这是预览文字',
              style: TextStyle(fontSize: _fontSize),
            ),

            const Divider(height: 40),

            // 离散 Slider：音量调节（带 divisions 和 label）
            Text('音量：${_volume.round()}',
                style: Theme.of(context).textTheme.titleMedium),
            Slider(
              value: _volume,
              min: 0,
              max: 100,
              divisions: 20, // 分为 20 档，每档步进 5
              label: '${_volume.round()}', // 拖动时显示气泡标签
              activeColor: Colors.orange,
              onChanged: (double value) {
                setState(() => _volume = value);
              },
            ),
          ],
        ),
      ),
    );
  }
}

4.4 RangeSlider 示例

import 'package:flutter/material.dart';

void main() => runApp(const MaterialApp(home: RangeSliderDemoPage()));

class RangeSliderDemoPage extends StatefulWidget {

  State<RangeSliderDemoPage> createState() => _RangeSliderDemoPageState();

class _RangeSliderDemoPageState extends State<RangeSliderDemoPage> {

  Widget build(BuildContext context) {

            ),
            RangeSlider(
              values: _priceRange,
              min: 0,
              max: 1000,
              divisions: 100, // 每档 10 元
              labels: RangeLabels(
                '¥${_priceRange.start.round()}',
                '¥${_priceRange.end.round()}',
              ),
              activeColor: Colors.redAccent,
              onChanged: (RangeValues values) {
                setState(() => _priceRange = values);
              },
            ),
            const SizedBox(height: 16),
            // 模拟筛选结果
            Text('符合条件的商品（¥${_priceRange.start.round()} ~ ¥${_priceRange.end.round()}）：'),
          ],
        ),
      ),
    );
  }
}

最佳实践：

  1. 使用 divisions 让滑块吸附到特定刻度，避免用户选到无意义的小数值。
  2. 搭配 label 显示气泡提示，让用户清楚当前选择的值。
  3. 可通过 SliderThemeData 全局定制 Slider 的外观（轨道高度、滑块形状等）。
  4. onChangeStart 和 onChangeEnd 回调可用于在开始/结束拖拽时执行操作（如发起网络请求筛选）。

5. DropdownButton 与 DropdownMenu（Material 3）

5.1 使用场景

当备选项较多（通常超过 5 个）且不适合全部展示在界面上时，使用下拉选择。例如：选择国家/地区、选择分类、选择排序方式。

5.2 DropdownButton 基础用法

DropdownButton 是 Flutter 早期就存在的经典下拉组件。

import 'package:flutter/material.dart';

void main() => runApp(const MaterialApp(home: DropdownButtonDemoPage()));

class DropdownButtonDemoPage extends StatefulWidget {
  const DropdownButtonDemoPage({super.key});

  @override
  State<DropdownButtonDemoPage> createState() => _DropdownButtonDemoPageState();
}

class _DropdownButtonDemoPageState extends State<DropdownButtonDemoPage> {
  String? _selectedCity;

  final List<String> _cities = [
    '北京', '上海', '广州', '深圳', '杭州', '成都', '武汉', '南京',
  ];

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('DropdownButton 演示')),
      body: Padding(
        padding: const EdgeInsets.all(16),
        child: Column(
          crossAxisAlignment: CrossAxisAlignment.start,
          children: [
            Text('选择城市', style: Theme.of(context).textTheme.titleMedium),
            const SizedBox(height: 12),

            // DropdownButton 需要手动添加装饰
            Container(
              padding: const EdgeInsets.symmetric(horizontal: 12),
              decoration: BoxDecoration(
                border: Border.all(color: Colors.grey),
                borderRadius: BorderRadius.circular(8),
              ),
              child: DropdownButton<String>(
                value: _selectedCity,
                hint: const Text('请选择城市'),
                isExpanded: true, // 让下拉按钮撑满容器宽度
                underline: const SizedBox(), // 去掉默认下划线
                items: _cities.map((city) {
                  return DropdownMenuItem<String>(
                    value: city,
                    child: Text(city),
                  );
                }).toList(),
                onChanged: (String? value) {
                  setState(() => _selectedCity = value);
                },
              ),
            ),

            const SizedBox(height: 24),

            // DropdownButtonFormField：适合表单场景，自带下划线装饰和验证
            DropdownButtonFormField<String>(
              value: _selectedCity,
              decoration: const InputDecoration(
                labelText: '城市（表单版）',
                border: OutlineInputBorder(),
                prefixIcon: Icon(Icons.location_city),
              ),
              items: _cities.map((city) {
                return DropdownMenuItem<String>(
                  value: city,
                  child: Text(city),
                );
              }).toList(),
              onChanged: (String? value) {
                setState(() => _selectedCity = value);
              },
              validator: (value) {
                if (value == null) return '请选择一个城市';
                return null;
              },
            ),
          ],
        ),
      ),
    );
  }
}

5.3 DropdownMenu（Material 3 推荐）

DropdownMenu 是 Flutter 3.7+ 引入的 Material 3 下拉菜单组件，相比 DropdownButton 有以下优势：

  • 内置搜索/过滤功能（enableFilter / enableSearch）
  • 内置输入框装饰，无需手动包装
  • 支持 menuStyle 定制下拉菜单外观
  • 更符合 Material 3 设计规范
import 'package:flutter/material.dart';

void main() => runApp(MaterialApp(
      theme: ThemeData(useMaterial3: true, colorSchemeSeed: Colors.blue),
      home: const DropdownMenuDemoPage(),
    ));

class DropdownMenuDemoPage extends StatefulWidget {
  const DropdownMenuDemoPage({super.key});

  State<DropdownMenuDemoPage> createState() => _DropdownMenuDemoPageState();
}

class _DropdownMenuDemoPageState extends State<DropdownMenuDemoPage> {
  String? _selectedLanguage;

  // DropdownMenuEntry 是 DropdownMenu 的选项类型
  static final List<DropdownMenuEntry<String>> _languageEntries = [
    const DropdownMenuEntry(value: 'dart', label: 'Dart', leadingIcon: Icon(Icons.code)),
    const DropdownMenuEntry(value: 'kotlin', label: 'Kotlin', leadingIcon: Icon(Icons.android)),
    const DropdownMenuEntry(value: 'swift', label: 'Swift', leadingIcon: Icon(Icons.apple)),
    const DropdownMenuEntry(value: 'python', label: 'Python', leadingIcon: Icon(Icons.terminal)),
    const DropdownMenuEntry(value: 'rust', label: 'Rust', leadingIcon: Icon(Icons.settings)),
    const DropdownMenuEntry(value: 'go', label: 'Go', leadingIcon: Icon(Icons.cloud)),
    const DropdownMenuEntry(value: 'typescript', label: 'TypeScript', leadingIcon: Icon(Icons.web)),

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('DropdownMenu 演示')),
      body: Padding(
        padding: const EdgeInsets.all(16),
        child: Column(
          crossAxisAlignment: CrossAxisAlignment.start,
          children: [
            Text('Material 3 DropdownMenu',
                style: Theme.of(context).textTheme.titleMedium),
            const SizedBox(height: 12),

            // 基础用法
            DropdownMenu<String>(
              label: const Text('选择编程语言'),
              leadingIcon: const Icon(Icons.language),
              initialSelection: _selectedLanguage,
              dropdownMenuEntries: _languageEntries,
              onSelected: (String? value) {
                setState(() => _selectedLanguage = value);
              },
            ),

            const SizedBox(height: 24),

            // 带搜索过滤功能
            DropdownMenu<String>(
              label: const Text('搜索编程语言'),
              leadingIcon: const Icon(Icons.search),
              enableFilter: true, // 启用过滤：输入文字后仅显示匹配项
              enableSearch: true, // 启用搜索：输入文字后高亮匹配项
              dropdownMenuEntries: _languageEntries,
              // 自定义菜单样式
              menuStyle: MenuStyle(
                maximumSize: const WidgetStatePropertyAll(Size(300, 300)),
                shape: WidgetStatePropertyAll(
                  RoundedRectangleBorder(
                    borderRadius: BorderRadius.circular(12),
                  ),
                ),
              ),
              onSelected: (String? value) {
                setState(() => _selectedLanguage = value);
              },
            ),

            const SizedBox(height: 24),
            if (_selectedLanguage != null)
              Text('已选择：$_selectedLanguage',
                  style: const TextStyle(fontSize: 16)),
          ],
        ),
      ),
    );
  }
}

5.4 DropdownButton vs DropdownMenu 对比

特性 DropdownButton DropdownMenu（推荐）
Material 版本 Material 2 Material 3
内置输入框装饰 ❌ 需手动包装 ✅ 自带
搜索/过滤 ❌ 不支持 ✅ enableFilter / enableSearch
菜单样式定制 有限 ✅ menuStyle
表单集成 DropdownButtonFormField 需手动处理
推荐程度 旧项目兼容 新项目首选

最佳实践：

  1. 新项目中推荐使用 DropdownMenu，它是 Material 3 设计语言的一部分，功能更强大。
  2. 如果需要在 Form 中使用并利用 validator 校验，DropdownButtonFormField 仍然是便捷之选。
  3. 当选项特别多时（如国家列表），务必开启 enableFilter，让用户可以通过输入快速定位。

6. PopupMenuButton

6.1 使用场景

PopupMenuButton 用于弹出一个上下文菜单，常见于 AppBar 的"更多操作"按钮（三个点）、列表项的操作菜单等。

6.2 核心属性

属性 类型 说明
itemBuilder PopupMenuItemBuilder<T> 构建菜单项列表（必需）
onSelected PopupMenuItemSelected<T>? 选中某项后的回调
onCanceled PopupMenuCanceled? 用户未选择任何项就关闭菜单的回调
offset Offset 菜单相对于按钮的偏移量
icon Widget? 自定义触发图标
initialValue T? 菜单打开时高亮显示的项

6.3 完整示例

import 'package:flutter/material.dart';

enum SortOption { nameAsc, nameDesc, dateNew, dateOld }

void main() => runApp(const MaterialApp(home: PopupMenuDemoPage()));

class PopupMenuDemoPage extends StatefulWidget {
  const PopupMenuDemoPage({super.key});

  @override
  State<PopupMenuDemoPage> createState() => _PopupMenuDemoPageState();
}

class _PopupMenuDemoPageState extends State<PopupMenuDemoPage> {
  SortOption _currentSort = SortOption.nameAsc;

  String _sortLabel(SortOption option) {
    return switch (option) {
      SortOption.nameAsc => '名称升序',
      SortOption.nameDesc => '名称降序',
      SortOption.dateNew => '最新优先',
      SortOption.dateOld => '最旧优先',
    };
  }

  IconData _sortIcon(SortOption option) {
    return switch (option) {
      SortOption.nameAsc => Icons.sort_by_alpha,
      SortOption.nameDesc => Icons.sort_by_alpha,
      SortOption.dateNew => Icons.arrow_downward,
      SortOption.dateOld => Icons.arrow_upward,
    };
  }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: const Text('PopupMenuButton 演示'),
        actions: [
          // AppBar 中的更多操作按钮
          PopupMenuButton<String>(
            icon: const Icon(Icons.more_vert),
            offset: const Offset(0, 50), // 菜单向下偏移，避免遮挡按钮
            onSelected: (String value) {
              ScaffoldMessenger.of(context).showSnackBar(
                SnackBar(content: Text('选择了：$value')),
              );
            },
            itemBuilder: (BuildContext context) {
              return [
                const PopupMenuItem(value: '分享', child: ListTile(
                  leading: Icon(Icons.share),
                  title: Text('分享'),
                  contentPadding: EdgeInsets.zero,
                )),
                const PopupMenuItem(value: '收藏', child: ListTile(
                  leading: Icon(Icons.favorite_border),
                  title: Text('收藏'),
                  contentPadding: EdgeInsets.zero,
                )),
                const PopupMenuDivider(), // 分割线
                const PopupMenuItem(value: '举报', child: ListTile(
                  leading: Icon(Icons.flag, color: Colors.red),
                  title: Text('举报', style: TextStyle(color: Colors.red)),
                  contentPadding: EdgeInsets.zero,
                )),
              ];
            },
          ),
        ],
      ),
      body: Padding(
        padding: const EdgeInsets.all(16),
        child: Column(
          crossAxisAlignment: CrossAxisAlignment.start,
          children: [
            // 排序选项菜单
            Row(
              children: [
                Text('排序方式：${_sortLabel(_currentSort)}',
                    style: const TextStyle(fontSize: 16)),
                const Spacer(),
                PopupMenuButton<SortOption>(
                  initialValue: _currentSort,
                  onSelected: (SortOption value) {
                    setState(() => _currentSort = value);
                  },
                  itemBuilder: (BuildContext context) {
                    return SortOption.values.map((option) {
                      return PopupMenuItem<SortOption>(
                        value: option,
                        child: Row(
                          children: [
                            Icon(_sortIcon(option), size: 20),
                            const SizedBox(width: 12),
                            Text(_sortLabel(option)),
                            if (option == _currentSort) ...[
                              const Spacer(),
                              const Icon(Icons.check, size: 18, color: Colors.blue),

                    }).toList();

          ],
        ),
      ),

  }
}

注意事项：

  1. PopupMenuButton 没有 child 参数时默认显示三个点图标（Icons.more_vert）。提供 child 后，该 child 将作为触发区域。
  2. PopupMenuDivider 可用于在菜单项之间添加分割线。
  3. 使用 CheckedPopupMenuItem 可以创建带勾选标记的菜单项。

7. showDatePicker 与 showTimePicker

7.1 使用场景

当用户需要选择日期或时间时使用，如选择出生日期、预约时间、航班日期等。Flutter 提供了 Material 风格的日期和时间选择器。

7.2 showDatePicker 核心参数

参数 类型 说明
context BuildContext 构建上下文
initialDate DateTime 初始选中日期
firstDate DateTime 可选范围的起始日期
lastDate DateTime 可选范围的结束日期
initialDatePickerMode DatePickerMode 初始模式（日历/年份）
builder TransitionBuilder? 用于自定义主题或包裹对话框

7.3 showTimePicker 核心参数

参数 类型 说明
context BuildContext 构建上下文
initialTime TimeOfDay 初始选中时间
initialEntryMode TimePickerEntryMode 输入模式（拨盘/输入框）
builder TransitionBuilder? 用于自定义主题或切换 24 小时制

7.4 完整示例

import 'package:flutter/material.dart';

void main() => runApp(MaterialApp(
      theme: ThemeData(useMaterial3: true, colorSchemeSeed: Colors.indigo),
      home: const DateTimePickerDemoPage(),
    ));

class DateTimePickerDemoPage extends StatefulWidget {
  const DateTimePickerDemoPage({super.key});

  @override
  State<DateTimePickerDemoPage> createState() => _DateTimePickerDemoPageState();
}

class _DateTimePickerDemoPageState extends State<DateTimePickerDemoPage> {
  DateTime? _selectedDate;
  TimeOfDay? _selectedTime;
  DateTimeRange? _selectedDateRange;

  // 选择日期
  Future<void> _pickDate() async {
    final DateTime? result = await showDatePicker(
      context: context,
      initialDate: _selectedDate ?? DateTime.now(),
      firstDate: DateTime(2020),
      lastDate: DateTime(2030),
      helpText: '选择出生日期', // 顶部提示文字
      cancelText: '取消',
      confirmText: '确定',
      // 通过 builder 自定义主题
      builder: (BuildContext context, Widget? child) {
        return Theme(
          data: Theme.of(context).copyWith(
            colorScheme: ColorScheme.light(
              primary: Colors.indigo, // 选中日期的背景色
              onPrimary: Colors.white, // 选中日期的文字色
              surface: Colors.white,
            ),
          ),
          child: child!,
        );
      },
    );

    if (result != null) {
      setState(() => _selectedDate = result);
    }
  }

  // 选择时间
  Future<void> _pickTime() async {
    final TimeOfDay? result = await showTimePicker(
      context: context,
      initialTime: _selectedTime ?? TimeOfDay.now(),
      helpText: '选择预约时间',
      // 切换为 24 小时制
      builder: (BuildContext context, Widget? child) {
        return MediaQuery(
          data: MediaQuery.of(context).copyWith(
            alwaysUse24HourFormat: true,
          ),
          child: child!,
        );
      },
    );

    if (result != null) {
      setState(() => _selectedTime = result);
    }
  }

  // 选择日期范围
  Future<void> _pickDateRange() async {
    final DateTimeRange? result = await showDateRangePicker(
      context: context,
      firstDate: DateTime(2020),
      lastDate: DateTime(2030),
      initialDateRange: _selectedDateRange,
      helpText: '选择入住 — 退房日期',
      saveText: '确定',
    );

    if (result != null) {
      setState(() => _selectedDateRange = result);
    }
  }

  String _formatDate(DateTime date) {
    return '${date.year}年${date.month}月${date.day}日';
  }

  String _formatTime(TimeOfDay time) {
    final hour = time.hour.toString().padLeft(2, '0');
    final minute = time.minute.toString().padLeft(2, '0');
    return '$hour:$minute';
  }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('日期与时间选择器')),
      body: Padding(
        padding: const EdgeInsets.all(16),
        child: Column(
          crossAxisAlignment: CrossAxisAlignment.stretch,
          children: [
            // 选择日期
            Card(
              child: ListTile(
                leading: const Icon(Icons.calendar_today),
                title: const Text('选择日期'),
                subtitle: Text(
                  _selectedDate != null
                      ? _formatDate(_selectedDate!)
                      : '未选择',
                ),
                trailing: const Icon(Icons.chevron_right),
                onTap: _pickDate,
              ),
            ),

            // 选择时间
            Card(
              child: ListTile(
                leading: const Icon(Icons.access_time),
                title: const Text('选择时间'),
                subtitle: Text(
                  _selectedTime != null
                      ? _formatTime(_selectedTime!)
                      : '未选择',
                ),
                trailing: const Icon(Icons.chevron_right),
                onTap: _pickTime,
              ),
            ),

            // 选择日期范围
            Card(
              child: ListTile(
                leading: const Icon(Icons.date_range),
                title: const Text('选择日期范围'),
                subtitle: Text(
                  _selectedDateRange != null
                      ? '${_formatDate(_selectedDateRange!.start)} — ${_formatDate(_selectedDateRange!.end)}'
                      : '未选择',
                ),
                trailing: const Icon(Icons.chevron_right),
                onTap: _pickDateRange,
              ),
            ),

            const SizedBox(height: 24),

            // 综合展示
            if (_selectedDate != null && _selectedTime != null)
              Card(
                color: Theme.of(context).colorScheme.primaryContainer,
                child: Padding(
                  padding: const EdgeInsets.all(16),
                  child: Text(
                    '预约时间：${_formatDate(_selectedDate!)} ${_formatTime(_selectedTime!)}',
                    style: Theme.of(context).textTheme.titleMedium,
                    textAlign: TextAlign.center,
                  ),
                ),
              ),
          ],
        ),
      ),
    );
  }
}

最佳实践：

  1. showDatePicker 和 showTimePicker 都返回 Future，使用 await 获取结果，结果为 null 表示用户取消了选择。
  2. 通过 builder 参数包裹 Theme，可以在不改变全局主题的情况下定制选择器的配色。
  3. 对于 24 小时制需求，在 builder 中通过 MediaQuery 设置 alwaysUse24HourFormat: true。
  4. showDateRangePicker 可用于选择日期区间（如酒店入住/退房）。

8. CupertinoDatePicker 与 CupertinoPicker

8.1 使用场景

当应用需要呈现 iOS 原生风格的选择器时使用。在跨平台应用中，可根据当前平台决定使用 Material 风格还是 Cupertino 风格。

8.2 CupertinoDatePicker 核心属性

属性 类型 说明
mode CupertinoDatePickerMode .date（仅日期）、.time（仅时间）、.dateAndTime（日期+时间）、.monthYear（年月）
onDateTimeChanged ValueChanged<DateTime> 值变化回调（必需）
initialDateTime DateTime? 初始日期时间
minimumDate / maximumDate DateTime? 可选范围
use24hFormat bool 是否使用 24 小时制

8.3 CupertinoPicker 核心属性

CupertinoPicker 是通用的 iOS 风格滚轮选择器，不限于日期。

属性 类型 说明
itemExtent double 每个选项的高度（必需）
onSelectedItemChanged ValueChanged<int> 选中索引变化回调
children List<Widget> 选项列表
scrollController FixedExtentScrollController? 控制滚动位置

8.4 CupertinoDatePicker 示例

iOS 风格的日期选择器通常以 bottom sheet 的形式从底部弹出。

8.5 CupertinoPicker 通用滚轮选择器

CupertinoPicker 不局限于日期时间，可用于任何自定义选项的滚轮选择。

import 'package:flutter/cupertino.dart';
import 'package:flutter/material.dart';

void main() => runApp(const MaterialApp(home: CupertinoWheelPickerPage()));

class CupertinoWheelPickerPage extends StatefulWidget {
  const CupertinoWheelPickerPage({super.key});

  @override
  State<CupertinoWheelPickerPage> createState() =>
      _CupertinoWheelPickerPageState();
}

class _CupertinoWheelPickerPageState extends State<CupertinoWheelPickerPage> {
  int _selectedProvinceIndex = 0;

  final List<String> _provinces = [
    '北京市', '天津市', '上海市', '重庆市',
    '河北省', '山西省', '辽宁省', '吉林省',
    '黑龙江省', '江苏省', '浙江省', '安徽省',
    '福建省', '江西省', '山东省', '河南省',
    '湖北省', '湖南省', '广东省', '四川省',
  ];

  void _showProvincePicker() {
    int tempIndex = _selectedProvinceIndex;

    showCupertinoModalPopup(
      context: context,
      builder: (BuildContext context) {
        return Container(
          height: 300,
          color: CupertinoColors.systemBackground.resolveFrom(context),
          child: Column(
            children: [
              SizedBox(
                height: 44,
                child: Row(
                  mainAxisAlignment: MainAxisAlignment.spaceBetween,
                  children: [
                    CupertinoButton(
                      child: const Text('取消'),
                      onPressed: () => Navigator.of(context).pop(),
                    ),
                    CupertinoButton(
                      child: const Text('确定'),
                      onPressed: () {
                        setState(() => _selectedProvinceIndex = tempIndex);
                        Navigator.of(context).pop();
                      },
                    ),
                  ],
                ),
              ),
              const Divider(height: 0),
              Expanded(
                child: CupertinoPicker(
                  itemExtent: 36, // 每项高度
                  scrollController: FixedExtentScrollController(
                    initialItem: _selectedProvinceIndex,
                  ),
                  onSelectedItemChanged: (int index) {
                    tempIndex = index;
                  },
                  children: _provinces.map((province) {
                    return Center(
                      child: Text(province, style: const TextStyle(fontSize: 18)),
                    );
                  }).toList(),
                ),
              ),
            ],
          ),
        );
      },
    );
  }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('CupertinoPicker 演示')),
      body: Padding(
        padding: const EdgeInsets.all(16),
        child: Column(
          crossAxisAlignment: CrossAxisAlignment.stretch,
          children: [
            Card(
              child: ListTile(
                leading: const Icon(CupertinoIcons.map),
                title: const Text('选择省份'),
                subtitle: Text(_provinces[_selectedProvinceIndex]),
                trailing: const Icon(CupertinoIcons.chevron_right),
                onTap: _showProvincePicker,
              ),
            ),
          ],
        ),
      ),
    );
  }
}

最佳实践：

  1. CupertinoDatePicker 不像 showDatePicker 那样自带对话框，需要手动使用 showCupertinoModalPopup 包装弹出。
  2. onDateTimeChanged 会在滚轮每次滚动时实时触发，而非最终确认时触发。建议使用临时变量暂存值，在用户点击"确定"后再更新状态。
  3. 跨平台应用可通过 Platform.isIOS 判断平台，分别使用 Cupertino 和 Material 风格的选择器。
  4. CupertinoPicker 的 itemExtent 是必需参数，它决定了滚轮中每个选项的高度。

本课小结

本课系统讲解了 Flutter 中 8 类常用选择组件：

  1. Checkbox / CheckboxListTile：勾选确认场景，支持三态（全选/部分选/全不选）。
  2. Radio / RadioListTile：互斥单选场景，推荐使用枚举作为泛型类型。
  3. Switch / SwitchListTile：开关切换场景，语义上表示"即时生效"的操作。
  4. Slider / RangeSlider：在数值范围内选值/选区间，divisions 控制离散档位。
  5. DropdownButton / DropdownMenu：下拉选择，新项目推荐 Material 3 的 DropdownMenu，支持搜索和过滤。
  6. PopupMenuButton：弹出上下文菜单，适用于操作菜单和选项菜单。
  7. showDatePicker / showTimePicker：Material 风格的日期和时间选择器，通过 builder 自定义主题。
  8. CupertinoDatePicker / CupertinoPicker：iOS 风格的滚轮选择器，需搭配 showCupertinoModalPopup 使用。

所有选择组件的状态管理核心：组件本身不维护状态，必须在 onChanged 回调中通过 setState 更新对应的 value / groupValue / values 变量，形成"数据驱动 UI"的模式。

练习1：设置页面（基础）

创建一个"应用设置"页面，包含以下选择组件：

  • 一个 SwitchListTile 控制"深色模式"
  • 一个 SwitchListTile 控制"消息通知"
  • 一组 RadioListTile 选择"语言"（中文/英文/日文）
  • 一个 Slider 调节"字体大小"（12 ~ 24，步进 2）
  • 一个 DropdownMenu 选择"主题色"

所有选择的结果实时显示在页面底部的摘要卡片中。

练习2：商品筛选器（中级）

创建一个商品筛选侧边栏，包含：

  • 多个 Checkbox 选择品牌（支持三态的"全选"父复选框）
  • 一个 RangeSlider 筛选价格范围
  • 一组 Radio 选择排序方式
  • 一个 DropdownMenu（带 enableFilter）选择商品分类
  • 底部有"重置"和"应用筛选"按钮

练习3：预约表单（中级）

创建一个"预约挂号"表单页面：

  • 使用 DropdownMenu 选择科室
  • 使用 showDatePicker 选择预约日期（只能选未来 30 天内，排除周末）
  • 使用 showTimePicker 选择预约时段（限制为 8:00 ~ 17:00）
  • 使用 Checkbox 勾选"是否需要医保"
  • 提交前验证所有字段

练习4：跨平台选择器（进阶）

封装一个 AdaptiveDatePicker 组件：

  • 在 iOS 上使用 CupertinoDatePicker（底部弹出滚轮样式）
  • 在 Android 上使用 showDatePicker（Material 日历样式）
  • 通过 Platform.isIOS 判断平台
  • 对外暴露统一的 onDateSelected 回调

练习5：综合思考题

  1. Checkbox 的 onChanged 回调类型是 ValueChanged<bool?>（可空 bool），而 Switch 的 onChanged 类型是 ValueChanged<bool>（非空 bool）。为什么 Checkbox 用可空类型？
  2. DropdownButton 的 value 必须是 items 中某个 DropdownMenuItem 的 value，否则会报错。如何优雅地处理"初始未选择"的状态？
  3. 如果在 CupertinoDatePicker 的 onDateTimeChanged 回调中直接调用 setState，会导致什么问题？为什么要用临时变量暂存？
