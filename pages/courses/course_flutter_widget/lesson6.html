第06课 线性布局 - Row 与 Column

本课是布局系列的第一课。掌握 Row 与 Column 是构建 Flutter 界面的基石——几乎所有页面都离不开水平排列与垂直排列。本课将深入讲解主轴与交叉轴的概念，并逐一演示所有对齐方式和弹性布局组件。

1. Row 与 Column 概述

1.1 什么是线性布局

线性布局是指沿单一方向（水平或垂直）依次排列子组件的布局方式：

  • Row — 将子组件沿水平方向（从左到右）排列
  • Column — 将子组件沿垂直方向（从上到下）排列

它们是 Flutter 中最常用的布局组件，几乎贯穿于每一个页面的构建中。

1.2 Row 和 Column 的本质 — Flex 的特化

查看 Flutter 源码你会发现，Row 和 Column 实际上是 Flex 组件的子类：

// Flutter 源码中 Row 的定义（简化版）
class Row extends Flex {
  Row({
    super.key,
    super.mainAxisAlignment,
    super.crossAxisAlignment,
    super.mainAxisSize,
    super.children,
  }) : super(direction: Axis.horizontal);
}

// Flutter 源码中 Column 的定义（简化版）
class Column extends Flex {
  Column({
    super.key,
    super.mainAxisAlignment,
    super.crossAxisAlignment,
    super.mainAxisSize,
    super.children,
  }) : super(direction: Axis.vertical);
}

它们之间唯一的区别就是 direction 参数：

  • Row → Axis.horizontal
  • Column → Axis.vertical

因此，理解了其中一个，另一个也就自然掌握了。如果你需要在运行时动态决定排列方向，可以直接使用 Flex 组件并传入 direction 参数。

1.3 主轴与交叉轴

这是理解整个线性布局体系最核心的概念：

对于 Row（水平排列）：

  • 主轴（Main Axis）：水平方向（→）
  • 交叉轴（Cross Axis）：垂直方向（↓）

对于 Column（垂直排列）：

  • 主轴（Main Axis）：垂直方向（↓）
  • 交叉轴（Cross Axis）：水平方向（→）

可以这样记忆：主轴就是子组件排列的方向，交叉轴就是与主轴垂直的方向。

Row 的坐标系：                    Column 的坐标系：

  交叉轴 ↑                          主轴 ↓
         |                               |
         |                               |
  -------+------→ 主轴           --------+------→ 交叉轴
         |                               |
         |                               |

下面是一个最基础的 Row 和 Column 示例：

import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: const Text('Row 与 Column 基础')),
        body: Column(
          children: [
            // Row：子组件水平排列
            Row(
              children: [
                Container(width: 80, height: 80, color: Colors.red),
                Container(width: 80, height: 80, color: Colors.green),
                Container(width: 80, height: 80, color: Colors.blue),
              ],
            ),
            const SizedBox(height: 20),
            // Column 嵌套在 SizedBox 中限制高度
            SizedBox(
              height: 280,
              child: Column(
                children: [
                  Container(width: 80, height: 80, color: Colors.orange),
                  Container(width: 80, height: 80, color: Colors.purple),
                  Container(width: 80, height: 80, color: Colors.cyan),
                ],
              ),
            ),
          ],
        ),
      ),
    );
  }
}

2. MainAxisAlignment — 主轴对齐方式

MainAxisAlignment 控制子组件在主轴方向上如何分布。它有 6 种取值，每一种都对应不同的分布策略。

下面的示例统一使用 Row 来演示，Column 的效果完全一致——只是方向从水平变为垂直。

2.1 MainAxisAlignment.start（默认值）

子组件从主轴起始位置开始排列，剩余空间在末尾。

| [A] [B] [C]          |

import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: const Text('MainAxisAlignment.start')),
        body: Row(
          mainAxisAlignment: MainAxisAlignment.start,
          children: [
            Container(width: 60, height: 60, color: Colors.red),
            Container(width: 60, height: 60, color: Colors.green),
            Container(width: 60, height: 60, color: Colors.blue),
          ],
        ),
      ),
    );
  }
}

2.2 MainAxisAlignment.end

子组件从主轴末尾位置开始排列，剩余空间在开头。

2.7 六种对齐方式完整对照示例

下面将六种对齐方式放在同一个页面中，方便直观对比：

import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: const Text('MainAxisAlignment 对比')),
        body: SingleChildScrollView(
          child: Column(
            children: MainAxisAlignment.values.map((alignment) {
              return Padding(
                padding: const EdgeInsets.symmetric(vertical: 8),
                child: Column(
                  crossAxisAlignment: CrossAxisAlignment.start,
                  children: [
                    Padding(
                      padding: const EdgeInsets.only(left: 8),
                      // 显示当前对齐方式名称
                      child: Text(
                        alignment.toString().split('.').last,
                        style: const TextStyle(
                          fontSize: 14,
                          fontWeight: FontWeight.bold,
                        ),
                      ),
                    ),
                    const SizedBox(height: 4),
                    Container(
                      color: Colors.grey[200],
                      child: Row(
                        mainAxisAlignment: alignment,
                        children: [
                          Container(width: 50, height: 50, color: Colors.red),
                          Container(width: 50, height: 50, color: Colors.green),
                          Container(width: 50, height: 50, color: Colors.blue),
                        ],
                      ),
                    ),
                  ],
                ),
              );
            }).toList(),
          ),
        ),
      ),
    );
  }
}

注意事项：spaceBetween、spaceAround、spaceEvenly 只有在主轴上存在剩余空间时才有明显效果。如果子组件已经填满了主轴，这三者的表现与 start 一致。

3. CrossAxisAlignment — 交叉轴对齐方式

CrossAxisAlignment 控制子组件在交叉轴方向上如何对齐。它有 5 种取值。

为了让交叉轴对齐效果可见，我们需要让子组件有不同的交叉轴尺寸（例如在 Row 中让子组件有不同的高度）。

3.1 CrossAxisAlignment.center（默认值）

子组件在交叉轴方向上居中对齐。

import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: const Text('CrossAxisAlignment.center')),
        body: Container(
          height: 200,
          color: Colors.grey[200],
          child: Row(
            crossAxisAlignment: CrossAxisAlignment.center,
            children: [
              Container(width: 60, height: 60, color: Colors.red),
              Container(width: 60, height: 100, color: Colors.green),
              Container(width: 60, height: 140, color: Colors.blue),
            ],
          ),
        ),
      ),
    );
  }
}

效果：三个色块在垂直方向上各自居中对齐。

3.2 CrossAxisAlignment.start

子组件在交叉轴方向上靠起始边对齐（Row 中为顶部对齐，Column 中为左侧对齐）。

import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

  const MyApp({super.key});

            children: [
              Container(width: 60, height: 60, color: Colors.red),
              Container(width: 60, height: 100, color: Colors.green),
              Container(width: 60, height: 140, color: Colors.blue),
            ],
          ),
        ),
      ),
    );
  }
}

效果：三个色块都紧贴容器顶部。

3.3 CrossAxisAlignment.end

子组件在交叉轴方向上靠末尾边对齐（Row 中为底部对齐，Column 中为右侧对齐）。

3.4 CrossAxisAlignment.stretch

子组件在交叉轴方向上被拉伸填满整个交叉轴空间。此时子组件自身在交叉轴方向上设置的尺寸会被忽略。

import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: const Text('CrossAxisAlignment.stretch')),
        body: Container(
          height: 200,
          color: Colors.grey[200],
          child: Row(
            crossAxisAlignment: CrossAxisAlignment.stretch,
            children: [
              // 虽然设置了 height: 60，但会被拉伸到 200
              Container(width: 60, height: 60, color: Colors.red),
              Container(width: 60, height: 100, color: Colors.green),
              Container(width: 60, height: 140, color: Colors.blue),
            ],
          ),
        ),
      ),
    );
  }
}

效果：三个色块的高度都被拉伸为 200（容器高度），原本设置的 height 被忽略。

3.5 CrossAxisAlignment.baseline

子组件按文本基线对齐。使用此值时必须同时指定 textBaseline 参数，否则会抛出异常。

这在对齐不同字号的文本时非常有用：

import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: const Text('CrossAxisAlignment.baseline')),
        body: Padding(
          padding: const EdgeInsets.all(16),
          child: Column(
            crossAxisAlignment: CrossAxisAlignment.start,
            children: [
              // 基线对齐：不同字号文本底部对齐
              const Text('baseline 对齐：', style: TextStyle(fontSize: 16)),
              const SizedBox(height: 8),
              Container(
                color: Colors.grey[200],
                child: const Row(
                  crossAxisAlignment: CrossAxisAlignment.baseline,
                  textBaseline: TextBaseline.alphabetic,
                  children: [
                    Text('大标题', style: TextStyle(fontSize: 32)),
                    SizedBox(width: 8),
                    Text('中标题', style: TextStyle(fontSize: 20)),
                    SizedBox(width: 8),
                    Text('正文内容', style: TextStyle(fontSize: 14)),
                  ],
                ),
              ),
              const SizedBox(height: 24),
              // 作为对比：center 对齐
              const Text('center 对齐（对比）：', style: TextStyle(fontSize: 16)),
              const SizedBox(height: 8),
              Container(
                color: Colors.grey[200],
                child: const Row(
                  crossAxisAlignment: CrossAxisAlignment.center,
                  children: [
                    Text('大标题', style: TextStyle(fontSize: 32)),
                    SizedBox(width: 8),
                    Text('中标题', style: TextStyle(fontSize: 20)),
                    SizedBox(width: 8),
                    Text('正文内容', style: TextStyle(fontSize: 14)),
                  ],
                ),
              ),
            ],
          ),
        ),
      ),
    );
  }
}

效果对比：

  • baseline 对齐让三段文字的字母底部处于同一水平线上，视觉上更自然
  • center 对齐让每段文字各自垂直居中，看起来文字"参差不齐"

4. MainAxisSize — 主轴尺寸策略

MainAxisSize 决定 Row/Column 在主轴方向上占据多少空间。它有两个取值：

值 行为
MainAxisSize.max（默认） 尽可能占满主轴方向上所有可用空间
MainAxisSize.min 仅占据子组件所需的最小空间

4.1 示例对比

import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: const Text('MainAxisSize 对比')),
        body: Column(
          crossAxisAlignment: CrossAxisAlignment.start,
          children: [
            const Padding(
              padding: EdgeInsets.all(8),
              child: Text(
                'MainAxisSize.max（默认）：',
                style: TextStyle(fontWeight: FontWeight.bold),
              ),
            ),
            // max：Row 宽度撑满屏幕
            Container(
              color: Colors.amber[100],
              child: Row(
                mainAxisSize: MainAxisSize.max,
                children: [
                  Container(width: 60, height: 60, color: Colors.red),
                  Container(width: 60, height: 60, color: Colors.green),
                  Container(width: 60, height: 60, color: Colors.blue),
                ],
              ),
            ),
            const SizedBox(height: 20),
            const Padding(
              padding: EdgeInsets.all(8),
              child: Text(
                'MainAxisSize.min：',
                style: TextStyle(fontWeight: FontWeight.bold),
              ),
            ),
            // min：Row 宽度仅为三个子组件宽度之和
            Container(
              color: Colors.amber[100],
              child: Row(
                mainAxisSize: MainAxisSize.min,
                children: [
                  Container(width: 60, height: 60, color: Colors.red),
                  Container(width: 60, height: 60, color: Colors.green),
                  Container(width: 60, height: 60, color: Colors.blue),
                ],
              ),
            ),
          ],
        ),
      ),
    );
  }
}

结果：

  • max：黄色背景延伸到屏幕右边缘，说明 Row 占满了水平空间
  • min：黄色背景仅包裹三个色块（宽 180），Row 只占据了子组件所需的空间

注意事项：当 MainAxisSize.min 与 MainAxisAlignment.spaceBetween 等空间分配对齐方式搭配时，由于没有剩余空间可分配，空间分配效果会失效。

5. Expanded — 按比例分配剩余空间

Expanded 用于让子组件填满主轴上的剩余空间。多个 Expanded 之间可以通过 flex 参数按比例分配。

5.1 基本用法

import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: const Text('Expanded 基本用法')),
        body: Column(
          children: [
            // 示例1：单个 Expanded 填满剩余空间
            const Padding(
              padding: EdgeInsets.all(8),
              child: Text('单个 Expanded：'),
            ),
            Row(
              children: [
                Container(width: 80, height: 60, color: Colors.red),
                // 绿色块填满红色和蓝色之间的所有剩余空间
                Expanded(
                  child: Container(height: 60, color: Colors.green),
                ),
                Container(width: 80, height: 60, color: Colors.blue),
              ],
            ),
            const SizedBox(height: 20),
            // 示例2：多个 Expanded 均分空间（默认 flex=1）
            const Padding(
              padding: EdgeInsets.all(8),
              child: Text('多个 Expanded 均分：'),
            ),
            Row(
              children: [
                Expanded(
                  child: Container(height: 60, color: Colors.red),
                ),
                Expanded(
                  child: Container(height: 60, color: Colors.green),

                Expanded(
                  child: Container(height: 60, color: Colors.blue),
                ),
              ],
            ),
          ],
        ),
      ),
    );
  }
}

5.2 flex 参数 — 按比例分配

flex 参数（默认为 1）决定了该 Expanded 占据剩余空间的权重比例：

最佳实践：在列表项布局中，常用"固定宽度 + Expanded + 固定宽度"的模式——左侧放头像、右侧放操作按钮、中间用 Expanded 自适应内容区。

6. Flexible — 灵活占用空间

Flexible 与 Expanded 很相似，但更加灵活。实际上，Expanded 就是 Flexible 的一个特例。

6.1 Flexible 的 fit 参数

Flexible 的核心区别在于 fit 参数，它有两个取值：

值 行为
FlexFit.tight 子组件必须填满分配到的空间（等同于 Expanded 的行为）
FlexFit.loose 子组件可以占据分配到的空间，但也可以更小（子组件保持自身尺寸）

6.2 Expanded 与 Flexible 的关系

查看 Flutter 源码可以看到：

// Expanded 的源码定义（简化版）
class Expanded extends Flexible {
  const Expanded({
    super.key,
    super.flex,
    required super.child,
  }) : super(fit: FlexFit.tight);
}

所以：Expanded = Flexible(fit: FlexFit.tight)

6.3 对比示例

7. Spacer — 弹性空白

Spacer 用于在 Row/Column 中创建弹性的空白间距，它是对 Expanded(child: SizedBox.shrink()) 的简写封装。

7.1 Spacer 的本质

查看 Flutter 源码：

class Spacer extends StatelessWidget {
  const Spacer({super.key, this.flex = 1});

  final int flex;

  @override
  Widget build(BuildContext context) {
    return Expanded(
      flex: flex,
      child: const SizedBox.shrink(),
    );
  }
}

所以 Spacer 本质上就是一个 Expanded + SizedBox 的组合，用于占据弹性空间而不渲染任何可见内容。

7.2 使用示例

import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: const Text('Spacer 使用示例')),
        body: Padding(
          padding: const EdgeInsets.all(16),
          child: Column(
            crossAxisAlignment: CrossAxisAlignment.start,
            children: [
              // 示例1：将元素推到两端（类似 spaceBetween 效果）
              const Text('左右两端对齐：', style: TextStyle(fontWeight: FontWeight.bold)),
              const SizedBox(height: 8),
              Container(
                color: Colors.grey[200],
                child: Row(
                  children: [
                    const Text('左侧内容'),
                    const Spacer(),
                    const Text('右侧内容'),
                  ],
                ),
              ),
              const SizedBox(height: 20),

              // 示例2：使用 flex 控制空白比例
              const Text('Spacer + flex 控制间距比例：', style: TextStyle(fontWeight: FontWeight.bold)),
              const SizedBox(height: 8),
              Container(
                color: Colors.grey[200],
                child: Row(
                  children: [
                    Container(width: 50, height: 50, color: Colors.red),
                    const Spacer(flex: 1),
                    Container(width: 50, height: 50, color: Colors.green),

8.2 溢出问题（overflow）

当子组件的总尺寸超过 Row/Column 在主轴方向上的可用空间时，就会产生**溢出（overflow）**错误。Flutter 会在溢出位置显示黄黑条纹警告。

溢出示例：

// ⚠️ 这段代码会产生溢出错误！
Row(
  children: [
    Container(width: 200, height: 60, color: Colors.red),
    Container(width: 200, height: 60, color: Colors.green),
    Container(width: 200, height: 60, color: Colors.blue),
    // 在 400 宽的设备上，总宽度 600 > 400，溢出！
  ],
)

8.3 解决溢出的三种方式

方法一：使用 Expanded / Flexible

让子组件自适应可用空间，不超出边界：

import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: const Text('Expanded 解决溢出')),
        body: Row(
          children: [
            // 使用 Expanded 让三个色块自适应分配空间
            Expanded(child: Container(height: 60, color: Colors.red)),
            Expanded(child: Container(height: 60, color: Colors.green)),
            Expanded(child: Container(height: 60, color: Colors.blue)),
          ],
        ),
      ),
    );
  }
}

方法二：使用 SingleChildScrollView

让整个 Row/Column 可滚动：

import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: const Text('ScrollView 解决溢出')),
        body: SingleChildScrollView(
          scrollDirection: Axis.horizontal, // 水平滚动
          child: Row(
            children: List.generate(
              10,
              (index) => Container(
                width: 100,
                height: 100,
                margin: const EdgeInsets.all(4),
                color: Colors.primaries[index % Colors.primaries.length],
                child: Center(
                  child: Text(
                    '${index + 1}',
                    style: const TextStyle(color: Colors.white, fontSize: 24),
                  ),
                ),
              ),
            ),
          ),
        ),
      ),
    );
  }
}

方法三：使用 Wrap 替代 Row

当内容需要自动换行时，使用 Wrap 组件（将在第08课详细学习）：

import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

  Widget build(BuildContext context) {

            spacing: 8, // 水平间距

                    style: const TextStyle(color: Colors.white),
                  ),
                ),
              ),
            ),
          ),
        ),
      ),
    );
  }
}

8.4 Column 嵌套在 Column 中的常见问题

当 Column 嵌套在另一个 Column 中且内部 Column 没有高度约束时，内部 Column 会尝试取无限高度导致错误。解决办法：

import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: const Text('嵌套 Column')),
        body: Column(
          children: [
            const Text('顶部内容'),
            const SizedBox(height: 16),
            // ✅ 用 Expanded 包裹内部 Column，给它有限高度约束
            Expanded(
              child: Column(
                children: [
                  Container(height: 60, color: Colors.red),
                  const SizedBox(height: 8),
                  Container(height: 60, color: Colors.green),
                  const Spacer(),
                  Container(height: 60, color: Colors.blue),
                ],
              ),
            ),
          ],
        ),
      ),
    );
  }
}

注意事项：

  • Row 嵌套在 Row 中，内部 Row 同样需要约束（用 Expanded 或 Flexible 包裹，或指定固定宽度）
  • Column 嵌套在 ListView/SingleChildScrollView 中时，使用 mainAxisSize: MainAxisSize.min 让 Column 只占据所需空间

                    const SizedBox(height: 16),

                    const SizedBox(height: 16),

                    Row(

这个卡片综合使用了：

  • Row + Expanded：头像和信息自适应布局
  • Column + CrossAxisAlignment.start：文字信息纵向左对齐排列
  • Expanded + Column：三等分统计数据
  • MainAxisAlignment.spaceEvenly：操作按钮均匀分布
  • 多层 Row/Column 嵌套：构建复杂的层级结构

10. 本课小结

核心概念回顾

概念 说明
Row / Column 线性布局组件，本质是 Flex 的特化，分别沿水平/垂直方向排列子组件
主轴（Main Axis） 子组件排列的方向（Row→水平，Column→垂直）
交叉轴（Cross Axis） 与主轴垂直的方向
MainAxisAlignment 控制子组件在主轴上的分布（6 种方式）
CrossAxisAlignment 控制子组件在交叉轴上的对齐（5 种方式）
MainAxisSize 控制 Row/Column 在主轴上占据的空间大小（max / min）
Expanded 强制子组件填满剩余空间，支持 flex 比例分配
Flexible 灵活占用空间，默认 FlexFit.loose 允许子组件保持自身尺寸
Spacer 弹性空白，本质是 Expanded + SizedBox

关键区别对比

对比项 左侧 右侧
Row vs Column 水平排列（主轴→） 垂直排列（主轴↓）
MainAxisSize.max vs min 占满所有可用空间 仅占子组件所需空间
Expanded vs Flexible tight，强制填满 loose，可以更小
Spacer vs SizedBox 弹性空白（按比例） 固定尺寸空白

最佳实践清单

  1. 先确定方向：水平排列用 Row，垂直排列用 Column
  2. 主轴分配看 MainAxisAlignment：6 种方式选择最合适的
  3. 交叉轴对齐看 CrossAxisAlignment：文本对齐优先考虑 baseline
  4. 溢出时三选一：Expanded/Flexible 自适应、SingleChildScrollView 滚动、Wrap 换行
  5. Column 嵌套 Column：内部 Column 用 Expanded 包裹或设置 MainAxisSize.min
  6. 弹性空间：需要内容用 Expanded，纯空白用 Spacer
  7. 比例布局：多个 Expanded 配合 flex 参数实现比例分配

11. 练习题

练习一：导航栏布局

使用 Row 实现如下导航栏：左侧为返回图标，中间为标题文字（居中），右侧为分享和更多两个图标。要求标题始终水平居中。

提示：可以利用 Expanded 让中间区域自适应，配合 TextAlign.center。

练习二：评分展示组件

使用 Row 实现一个水平的评分组件：左侧显示分数"4.5"（大字号），右侧显示五颗星（用 Icon(Icons.star) 表示），星星和分数文字底部基线对齐。

提示：使用 CrossAxisAlignment.baseline 配合 TextBaseline.alphabetic。

练习三：三栏布局

使用 Row + Expanded 实现一个三栏布局：左侧栏占 1/4，中间栏占 1/2，右侧栏占 1/4。每栏内部使用 Column 纵向排列若干文本。

提示：使用 Expanded(flex: 1) 和 Expanded(flex: 2) 来控制比例。

练习四：聊天消息布局

实现一个聊天页面中的单条消息布局：

  • 自己发送的消息靠右对齐（使用 MainAxisAlignment.end）
  • 对方的消息靠左对齐（使用 MainAxisAlignment.start）
  • 消息气泡使用 Flexible 包裹，确保长消息不会溢出

提示：Flexible 的 FlexFit.loose 非常适合这个场景，可以让消息气泡按内容自适应宽度，但不超过最大可用空间。

练习五：完整页面布局

综合运用本课知识，实现一个类似下方结构的页面：

┌─────────────────────────────┐
│  [Logo]   标题    [菜单图标]  │  ← Row + Expanded
├─────────────────────────────┤
│                             │
│    图片占位区（横向铺满）       │  ← Expanded
│                             │
├─────────────────────────────┤
│  标题文字                    │
│  描述文字                    │  ← Column + CrossAxisAlignment.start
│  [标签1] [标签2] [标签3]     │  ← Row
├─────────────────────────────┤
│  [取消]     Spacer    [确定] │  ← Row + Spacer
└─────────────────────────────┘
