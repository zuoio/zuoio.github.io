1. Card — 卡片容器

1.1 简介

Card 是 Material Design 中最基础的信息容器之一，它通过阴影（elevation）将内容从背景中"浮起"，在视觉上形成一个独立的信息单元。Card 通常用于包裹一组相关信息，如联系人卡片、商品简介、设置项等。

1.2 核心属性

属性 类型 说明
elevation double? 阴影高度，值越大阴影越明显，默认为 1.0
shape ShapeBorder? 卡片形状，默认圆角矩形（RoundedRectangleBorder）
margin EdgeInsetsGeometry? 外边距
clipBehavior Clip 子组件超出卡片边界时的裁剪行为
color Color? 卡片背景色
surfaceTintColor Color? Material 3 表面着色，影响卡片在不同 elevation 下的颜色变化
shadowColor Color? 阴影颜色
borderOnForeground bool 是否在子组件前面绘制边框，默认 true

1.3 基本用法

import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      theme: ThemeData(useMaterial3: true),
      home: Scaffold(
        appBar: AppBar(title: const Text('Card 示例')),
        body: Padding(
          padding: const EdgeInsets.all(16.0),
          child: Column(
            children: [
              // 基础卡片
              Card(
                child: Padding(
                  padding: const EdgeInsets.all(16.0),
                  child: Column(
                    crossAxisAlignment: CrossAxisAlignment.start,
                    children: [
                      Text('基础卡片', style: Theme.of(context).textTheme.titleMedium),
                      const SizedBox(height: 8),
                      const Text('这是一个默认样式的 Card 组件。'),
                    ],
                  ),
                ),
              ),
              const SizedBox(height: 16),

              // 自定义阴影和形状
              Card(
                elevation: 8,
                shadowColor: Colors.blue.withOpacity(0.4),
                shape: RoundedRectangleBorder(
                  borderRadius: BorderRadius.circular(16),
                ),
                child: const Padding(
                  padding: EdgeInsets.all(16.0),
                  child: Text('自定义阴影和圆角'),
                ),
              ),
              const SizedBox(height: 16),

              // 带边框的卡片（outlined 风格）
              Card(
                elevation: 0,
                shape: RoundedRectangleBorder(
                  borderRadius: BorderRadius.circular(12),
                  side: BorderSide(color: Theme.of(context).colorScheme.outline),
                ),
                child: const Padding(
                  padding: EdgeInsets.all(16.0),
                  child: Text('Outlined 风格卡片（无阴影，有边框）'),
                ),
              ),
            ],
          ),
        ),
      ),
    );
  }
}

1.4 clipBehavior 的作用

当子组件（如图片）可能超出卡片圆角边界时，需要设置 clipBehavior：

import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: const Text('Card clipBehavior 示例')),
        body: Padding(
          padding: const EdgeInsets.all(16.0),
          child: Card(
            // 启用裁剪，防止子组件（如图片）超出圆角边界
            clipBehavior: Clip.antiAlias,
            shape: RoundedRectangleBorder(
              borderRadius: BorderRadius.circular(16),
            ),
            child: Column(
              mainAxisSize: MainAxisSize.min,
              children: [
                // 图片会自动被卡片的圆角裁剪
                Image.network(
                  'https://picsum.photos/400/200',
                  height: 200,
                  width: double.infinity,
                  fit: BoxFit.cover,
                ),
                const Padding(
                  padding: EdgeInsets.all(16.0),
                  child: Text('设置 clipBehavior 后图片会自动适配卡片圆角'),
                ),
              ],
            ),
          ),
        ),
      ),
    );
  }
}

1.5 Material 3 中的 surfaceTintColor

在 Material 3 中，Card 的颜色不再仅由 color 决定。surfaceTintColor 会根据 elevation 的不同，在卡片表面叠加一层半透明的着色，营造出层次感：

import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      theme: ThemeData(useMaterial3: true, colorSchemeSeed: Colors.blue),
      home: Scaffold(
        appBar: AppBar(title: const Text('surfaceTintColor 演示')),
        body: Padding(
          padding: const EdgeInsets.all(16.0),
          child: Column(
            children: [
              // 不同 elevation 下 surfaceTintColor 的效果
              for (final elevation in [0.0, 1.0, 3.0, 6.0, 12.0])
                Padding(
                  padding: const EdgeInsets.only(bottom: 12),
                  child: Card(
                    elevation: elevation,
                    child: Padding(
                      padding: const EdgeInsets.all(16.0),
                      child: Text('elevation: $elevation'),
                    ),
                  ),
                ),
              const Divider(),
              // 禁用 surfaceTintColor（退回传统阴影效果）
              Card(
                elevation: 6,
                surfaceTintColor: Colors.transparent,
                child: const Padding(
                  padding: EdgeInsets.all(16.0),
                  child: Text('surfaceTintColor: transparent（传统阴影效果）'),
                ),
              ),
            ],
          ),
        ),
      ),
    );
  }
}

注意事项：

  • Card 本身不提供 padding，需要自行在内部添加 Padding 组件。
  • 在 Material 3 主题下，如果不想要 surfaceTintColor 的着色效果，可将其设为 Colors.transparent。
  • 推荐结合 InkWell 为卡片添加点击水波纹效果，而不是直接使用 GestureDetector。

2. ListTile — 标准列表项

2.1 简介

ListTile 是 Flutter 中最常用的列表项组件，遵循 Material Design 规范，提供了标准的列表项布局：前导图标、标题、副标题、尾部操作区域。它是构建设置页面、联系人列表、菜单选项等场景的首选组件。

2.2 核心属性

属性 类型 说明
leading Widget? 列表项前导组件（通常是图标或头像）
title Widget? 主标题
subtitle Widget? 副标题
trailing Widget? 尾部组件（通常是图标或开关）
isThreeLine bool 是否为三行模式（副标题允许两行）
dense bool? 是否为紧凑模式（减小高度和字号）
contentPadding EdgeInsetsGeometry? 内容区域的内边距
onTap VoidCallback? 点击回调
onLongPress VoidCallback? 长按回调
selected bool 是否为选中状态（影响文本和图标颜色）
selectedTileColor Color? 选中时的背景色
enabled bool 是否可交互，默认 true
tileColor Color? 列表项背景色

2.3 基本用法

import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: const Text('ListTile 示例')),
        body: ListView(
          children: [
            // 最简单的 ListTile
            const ListTile(
              title: Text('单行列表项'),
            ),
            const Divider(height: 1),

            // 带前导图标和副标题
            const ListTile(
              leading: Icon(Icons.person),
              title: Text('张三'),
              subtitle: Text('Flutter 开发工程师'),
            ),
            const Divider(height: 1),

            // 带尾部操作
            const ListTile(
              leading: CircleAvatar(child: Text('李')),
              title: Text('李四'),
              subtitle: Text('上次登录：2小时前'),
              trailing: Icon(Icons.chevron_right),
            ),
            const Divider(height: 1),

            // 三行模式
            const ListTile(
              leading: CircleAvatar(
                backgroundImage: NetworkImage('https://picsum.photos/100'),
              ),
              title: Text('王五'),
              subtitle: Text(
                '这是一段较长的副标题内容，在 isThreeLine 为 true 时，'
                '副标题可以显示两行文字而不会被截断。',
              ),
              isThreeLine: true,
              trailing: Icon(Icons.more_vert),
            ),
          ],
        ),
      ),
    );
  }
}

2.4 选中状态与交互

import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatefulWidget {
  const MyApp({super.key});

  @override
  State<MyApp> createState() => _MyAppState();
}

class _MyAppState extends State<MyApp> {
  int _selectedIndex = 0;

  final List<Map<String, dynamic>> _items = [
    {'icon': Icons.inbox, 'title': '收件箱', 'count': 12},
    {'icon': Icons.send, 'title': '已发送', 'count': 5},
    {'icon': Icons.drafts, 'title': '草稿箱', 'count': 3},
    {'icon': Icons.delete, 'title': '回收站', 'count': 0},
  ];

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      theme: ThemeData(useMaterial3: true),
      home: Scaffold(
        appBar: AppBar(title: const Text('ListTile 选中状态')),
        body: ListView.builder(
          itemCount: _items.length,
          itemBuilder: (context, index) {
            final item = _items[index];
            return ListTile(
              leading: Icon(item['icon'] as IconData),
              title: Text(item['title'] as String),
              trailing: (item['count'] as int) > 0
                  ? Badge.count(
                      count: item['count'] as int,
                      child: const Icon(Icons.chevron_right),
                    )
                  : const Icon(Icons.chevron_right),
              selected: _selectedIndex == index,
              selectedTileColor:
                  Theme.of(context).colorScheme.primaryContainer.withOpacity(0.3),
              onTap: () => setState(() => _selectedIndex = index),
              // 紧凑模式与自定义内边距
              dense: false,
              contentPadding: const EdgeInsets.symmetric(horizontal: 24),
            );
          },
        ),
      ),
    );
  }
}

最佳实践：

  • ListTile 的 leading 推荐使用 Icon（24×24）或 CircleAvatar（40×40），以保持视觉一致性。
  • 当列表项需要更复杂的布局时，可使用 Row + Column 自定义替代 ListTile。
  • dense 模式适合信息密度高的列表，如设置页面。

3. ExpansionTile — 可展开的列表项

3.1 简介

ExpansionTile 是一个可以展开/折叠的列表项组件，点击后会展示额外的子内容。常见于 FAQ 页面、设置分组、目录导航等场景。

3.2 核心属性

属性 类型 说明
title Widget 标题（必须）
subtitle Widget? 副标题
children List<Widget> 展开后显示的子组件列表
leading Widget? 前导组件
trailing Widget? 自定义尾部图标（默认为旋转箭头）
initiallyExpanded bool 初始是否展开，默认 false
onExpansionChanged ValueChanged<bool>? 展开/折叠状态变化回调
controlAffinity ListTileControlAffinity? 控制展开图标的位置
tilePadding EdgeInsetsGeometry? 标题区域内边距
childrenPadding EdgeInsetsGeometry? 子内容区域内边距
expandedCrossAxisAlignment CrossAxisAlignment? 子内容交叉轴对齐方式
backgroundColor Color? 展开时的背景色
collapsedBackgroundColor Color? 折叠时的背景色

3.3 基本用法

import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: const Text('ExpansionTile 示例')),
        body: ListView(
          children: [
            // 基础可展开列表项
            const ExpansionTile(
              leading: Icon(Icons.info_outline),
              title: Text('什么是 Flutter？'),
              children: [
                Padding(
                  padding: EdgeInsets.all(16.0),
                  child: Text(
                    'Flutter 是 Google 开发的开源 UI 框架，'
                    '可以用一套代码构建跨平台应用。',
                  ),
                ),
              ],
            ),

            // 默认展开 + 带副标题
            ExpansionTile(
              leading: const Icon(Icons.settings),
              title: const Text('高级设置'),
              subtitle: const Text('自定义应用行为'),
              initiallyExpanded: true,
              childrenPadding: const EdgeInsets.symmetric(horizontal: 16),
              onExpansionChanged: (expanded) {
                debugPrint('高级设置 ${expanded ? "展开" : "折叠"}');
              },
              children: const [
                ListTile(title: Text('缓存管理'), trailing: Icon(Icons.chevron_right)),
                ListTile(title: Text('网络设置'), trailing: Icon(Icons.chevron_right)),
                ListTile(title: Text('开发者选项'), trailing: Icon(Icons.chevron_right)),
              ],
            ),

            // controlAffinity：箭头在前面
            const ExpansionTile(
              controlAffinity: ListTileControlAffinity.leading,
              title: Text('展开图标在前面'),
              children: [
                Padding(
                  padding: EdgeInsets.all(16.0),
                  child: Text('通过 controlAffinity 控制箭头图标位置'),
                ),
              ],
            ),
          ],
        ),
      ),
    );
  }
}

4. ExpansionPanelList — 手风琴面板

4.1 简介

ExpansionPanelList 用于管理一组可展开的面板，常见的"手风琴"效果就是通过它实现的。与 ExpansionTile 不同，ExpansionPanelList 可以统一管理多个面板的展开/折叠状态，支持单开模式（同一时刻只能展开一个面板）。

4.2 核心属性

属性 类型 说明
expansionCallback ExpansionPanelCallback? 面板展开/折叠时的回调，参数为面板索引和是否展开
children List<ExpansionPanel> 面板列表
expandedHeaderPadding EdgeInsets 展开时头部的额外内边距
dividerColor Color? 面板间分割线颜色
elevation double 展开时的阴影高度

ExpansionPanel 核心属性：

属性 类型 说明
headerBuilder ExpansionPanelHeaderBuilder 头部构建器，接收 (context, isExpanded)
body Widget 面板展开后显示的内容
isExpanded bool 是否展开
canTapOnHeader bool 是否允许点击头部展开/折叠，默认 false

4.3 单开模式（手风琴效果）

import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatefulWidget {
  const MyApp({super.key});

  @override
  State<MyApp> createState() => _MyAppState();
}

class _MyAppState extends State<MyApp> {
  // 每个面板的展开状态
  final List<bool> _isExpanded = [false, false, false];

  final List<Map<String, String>> _sections = [
    {'title': '个人信息', 'content': '姓名、年龄、性别等基本信息设置'},
    {'title': '账号安全', 'content': '密码修改、双重验证、登录设备管理'},
    {'title': '通知设置', 'content': '推送通知、邮件通知、消息免打扰'},
  ];

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: const Text('ExpansionPanelList 单开模式')),
        body: SingleChildScrollView(
          child: Padding(
            padding: const EdgeInsets.all(8.0),
            child: ExpansionPanelList(
              // 单开模式：展开新面板时关闭其他面板
              expansionCallback: (int panelIndex, bool isExpanded) {
                setState(() {
                  // 先关闭所有面板
                  for (int i = 0; i < _isExpanded.length; i++) {
                    _isExpanded[i] = false;
                  }
                  // 如果当前面板之前是关闭的，则打开它
                  if (!isExpanded) {
                    _isExpanded[panelIndex] = true;
                  }
                });
              },
              children: List.generate(_sections.length, (index) {
                return ExpansionPanel(
                  headerBuilder: (context, isExpanded) {
                    return ListTile(
                      leading: Icon(
                        [Icons.person, Icons.security, Icons.notifications][index],
                      ),
                      title: Text(_sections[index]['title']!),
                    );
                  },
                  body: Padding(
                    padding: const EdgeInsets.fromLTRB(16, 0, 16, 16),
                    child: Text(_sections[index]['content']!),
                  ),
                  isExpanded: _isExpanded[index],
                  canTapOnHeader: true,
                );
              }),
            ),
          ),
        ),
      ),
    );
  }
}

4.4 多开模式

import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatefulWidget {
  const MyApp({super.key});

  @override
  State<MyApp> createState() => _MyAppState();
}

class _MyAppState extends State<MyApp> {
  final List<bool> _isExpanded = [false, false, false];

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: const Text('ExpansionPanelList 多开模式')),
        body: SingleChildScrollView(
          child: Padding(
            padding: const EdgeInsets.all(8.0),
            child: ExpansionPanelList(
              // 多开模式：每个面板独立切换，互不影响
              expansionCallback: (int panelIndex, bool isExpanded) {
                setState(() {
                  _isExpanded[panelIndex] = !isExpanded;
                });
              },
              children: [
                _buildPanel(0, '第一章：基础入门', '介绍 Dart 语言和 Flutter 框架的基本概念...'),
                _buildPanel(1, '第二章：组件系统', '深入了解 Widget、Element、RenderObject...'),
                _buildPanel(2, '第三章：状态管理', '学习 setState、Provider、Riverpod 等方案...'),
              ],
            ),
          ),
        ),
      ),
    );
  }

  ExpansionPanel _buildPanel(int index, String title, String content) {
    return ExpansionPanel(
      headerBuilder: (context, isExpanded) {
        return ListTile(title: Text(title));
      },
      body: Container(
        padding: const EdgeInsets.fromLTRB(16, 0, 16, 16),
        alignment: Alignment.centerLeft,
        child: Text(content),
      ),
      isExpanded: _isExpanded[index],
      canTapOnHeader: true,
    );
  }
}

注意事项：

  • ExpansionPanelList 必须放在可滚动的容器中（如 SingleChildScrollView），否则展开后可能溢出。
  • 单开模式与多开模式的区别完全在 expansionCallback 的逻辑中控制。
  • canTapOnHeader: true 允许点击整个头部区域来展开面板，而不只是点击箭头图标。
  • Flutter 3.x 中 ExpansionPanelList.radio 已被废弃，推荐通过 expansionCallback 手动实现单开逻辑。

5. DataTable — 数据表格

5.1 简介

DataTable 是 Material Design 风格的数据表格组件，适合在较小的数据集（几十行以内）中展示结构化数据。它内置了排序和行选中功能。

5.2 核心属性

属性 类型 说明
columns List<DataColumn> 表头列定义
rows List<DataRow> 数据行
sortColumnIndex int? 当前排序列的索引
sortAscending bool 是否升序排列，默认 true
onSelectAll ValueSetter<bool?>? 全选/取消全选回调
showCheckboxColumn bool 是否显示复选框列
headingRowColor WidgetStateProperty<Color?>? 表头行背景色
dataRowMinHeight double? 数据行最小高度
dataRowMaxHeight double? 数据行最大高度

5.4 排序功能

import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatefulWidget {
  const MyApp({super.key});

  @override
  State<MyApp> createState() => _MyAppState();
}

class Student {
  final String name;
  final int score;
  final String grade;
  bool selected;

  Student(this.name, this.score, this.grade, {this.selected = false});
}

class _MyAppState extends State<MyApp> {
  int _sortColumnIndex = 0;
  bool _sortAscending = true;

  final List<Student> _students = [
    Student('张三', 92, 'A'),
    Student('李四', 78, 'B'),
    Student('王五', 95, 'A'),
    Student('赵六', 65, 'C'),
    Student('孙七', 88, 'B'),
  ];

  void _sort<T>(Comparable<T> Function(Student s) getField, int columnIndex, bool ascending) {
    _students.sort((a, b) {
      final aValue = getField(a);
      final bValue = getField(b);
      return ascending
          ? Comparable.compare(aValue, bValue)
          : Comparable.compare(bValue, aValue);
    });
    setState(() {
      _sortColumnIndex = columnIndex;
      _sortAscending = ascending;
    });
  }

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: const Text('DataTable 排序与选中')),
        body: SingleChildScrollView(
          child: DataTable(
            sortColumnIndex: _sortColumnIndex,
            sortAscending: _sortAscending,
            // 显示复选框列（当任意行有 onSelectChanged 时自动出现）
            showCheckboxColumn: true,
            // 全选/取消全选
            onSelectAll: (selected) {
              setState(() {
                for (final student in _students) {
                  student.selected = selected ?? false;
                }
              });
            },
            columns: [
              DataColumn(
                label: const Text('姓名'),
                onSort: (columnIndex, ascending) {
                  _sort<String>((s) => s.name, columnIndex, ascending);
                },
              ),
              DataColumn(
                label: const Text('分数'),
                numeric: true,
                onSort: (columnIndex, ascending) {
                  _sort<num>((s) => s.score, columnIndex, ascending);
                },
              ),
              DataColumn(
                label: const Text('等级'),
                onSort: (columnIndex, ascending) {
                  _sort<String>((s) => s.grade, columnIndex, ascending);
                },
              ),
            ],
            rows: _students.map((student) {
              return DataRow(
                selected: student.selected,
                onSelectChanged: (selected) {
                  setState(() {
                    student.selected = selected ?? false;
                  });
                },
                cells: [
                  DataCell(Text(student.name)),
                  DataCell(Text('${student.score}')),
                  DataCell(
                    Container(
                      padding: const EdgeInsets.symmetric(horizontal: 8, vertical: 4),
                      decoration: BoxDecoration(
                        color: student.grade == 'A'
                            ? Colors.green.shade100
                            : student.grade == 'B'
                                ? Colors.orange.shade100
                                : Colors.red.shade100,
                        borderRadius: BorderRadius.circular(4),
                      ),
                      child: Text(student.grade),
                    ),
                  ),
                ],
              );
            }).toList(),
          ),
        ),
      ),
    );
  }
}

最佳实践：

  • DataTable 适用于小规模数据展示（通常不超过 50-100 行），大规模数据请使用 PaginatedDataTable。
  • 当列数较多、表格宽度超出屏幕时，将 DataTable 放入 SingleChildScrollView(scrollDirection: Axis.horizontal) 中。
  • DataColumn 的 numeric: true 会让该列文本右对齐，符合数字的阅读习惯。
  • DataCell 可以接收任意 Widget，不局限于 Text，可以放入按钮、图标、标签等。

InputChip 代表用户输入的一条信息（如搜索关键词、已选联系人），可以点击和删除。

import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatefulWidget {

        _tags.add(trimmed);
        _controller.clear();

                controller: _controller,
                decoration: const InputDecoration(
                  hintText: '输入标签后按回车',
                  border: OutlineInputBorder(),
                ),
                onSubmitted: _addTag,
              ),
              const SizedBox(height: 16),

              // 标签列表
              Wrap(
                spacing: 8,
                runSpacing: 8,
                children: _tags.map((tag) {
                  return InputChip(
                    avatar: CircleAvatar(child: Text(tag[0])),
                    label: Text(tag),
                    // 点击事件
                    onPressed: () => debugPrint('点击了 $tag'),
                    // 删除事件
                    onDeleted: () {
                      setState(() => _tags.remove(tag));
                    },
                  );
                }).toList(),
              ),
            ],
          ),
        ),
      ),
    );
  }

  @override
  void dispose() {
    _controller.dispose();
    super.dispose();
  }
}

9. LinearProgressIndicator 与 CircularProgressIndicator — 进度指示

9.1 简介

Flutter 提供了两种进度指示器：LinearProgressIndicator（线性/条形）和 CircularProgressIndicator（圆形）。它们都支持两种模式：

  • 确定进度（Determinate）：value 为 0.0~1.0 之间的值，表示已完成的进度比例。
  • 不确定进度（Indeterminate）：value 为 null（默认），显示持续动画，表示"正在加载中"。

9.2 核心属性

属性 类型 说明
value double? 进度值（0.0~1.0）。为 null 时显示不确定进度动画
backgroundColor Color? 背景色（轨道颜色）
color Color? 进度条颜色
valueColor Animation<Color?>? 进度条颜色的动画（优先级高于 color）
minHeight（Linear 独有） double? 进度条的最小高度
strokeWidth（Circular 独有） double 圆形进度条的线条宽度，默认 4.0
strokeCap（Circular 独有） StrokeCap? 线条端点样式（圆角 round 或直角 butt）
semanticsLabel String? 无障碍标签

10. Stepper — 步骤条

10.1 简介

Stepper 是一个步骤条组件，用于引导用户按顺序完成一系列操作步骤，如注册流程、设置向导、结账流程等。支持垂直和水平两种布局。

10.2 核心属性

属性 类型 说明
steps List<Step> 步骤列表
currentStep int 当前活动步骤的索引
onStepContinue VoidCallback? 点击"继续"按钮的回调
onStepCancel VoidCallback? 点击"取消"按钮的回调
onStepTapped ValueChanged<int>? 点击步骤标题的回调
type StepperType 布局类型：vertical（垂直）或 horizontal（水平）
physics ScrollPhysics? 滚动物理效果
controlsBuilder ControlsWidgetBuilder? 自定义"继续/取消"按钮区域
elevation double? 阴影高度

Step 核心属性：

属性 类型 说明
title Widget 步骤标题
subtitle Widget? 步骤副标题
content Widget 步骤内容
state StepState 步骤状态：indexed、editing、complete、disabled、error
isActive bool 是否为当前活动步骤

10.3 垂直步骤条

import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatefulWidget {
  const MyApp({super.key});

  @override
  State<MyApp> createState() => _MyAppState();
}

class _MyAppState extends State<MyApp> {
  int _currentStep = 0;

  // 表单数据
  final _nameController = TextEditingController();
  final _emailController = TextEditingController();
  String _plan = '免费版';

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      theme: ThemeData(useMaterial3: true),
      home: Scaffold(
        appBar: AppBar(title: const Text('Stepper 垂直步骤条')),
        body: Stepper(
          type: StepperType.vertical,
          currentStep: _currentStep,
          // 点击"继续"：进入下一步
          onStepContinue: () {
            if (_currentStep < 2) {
              setState(() => _currentStep++);
            } else {
              // 最后一步，提交
              _showResult(context);
            }
          },
          // 点击"取消"：返回上一步
          onStepCancel: () {
            if (_currentStep > 0) {
              setState(() => _currentStep--);
            }
          },
          // 点击步骤标题：跳转到对应步骤
          onStepTapped: (step) {
            setState(() => _currentStep = step);
          },
          steps: [
            Step(
              title: const Text('基本信息'),
              subtitle: const Text('填写姓名'),
              content: TextField(
                controller: _nameController,
                decoration: const InputDecoration(
                  labelText: '姓名',
                  border: OutlineInputBorder(),
                ),
              ),
              isActive: _currentStep >= 0,
              state: _currentStep > 0 ? StepState.complete : StepState.indexed,
            ),
            Step(
              title: const Text('联系方式'),
              subtitle: const Text('填写邮箱'),
              content: TextField(
                controller: _emailController,
                decoration: const InputDecoration(
                  labelText: '邮箱',
                  border: OutlineInputBorder(),
                ),
                keyboardType: TextInputType.emailAddress,
              ),
              isActive: _currentStep >= 1,
              state: _currentStep > 1 ? StepState.complete : StepState.indexed,
            ),
            Step(
              title: const Text('选择套餐'),
              content: Column(
                children: ['免费版', '专业版', '企业版'].map((plan) {
                  return RadioListTile<String>(
                    title: Text(plan),
                    value: plan,
                    groupValue: _plan,
                    onChanged: (value) {
                      setState(() => _plan = value!);
                    },
                  );
                }).toList(),
              ),
              isActive: _currentStep >= 2,
              state: _currentStep > 2 ? StepState.complete : StepState.indexed,
            ),
          ],
        ),
      ),
    );
  }

  void _showResult(BuildContext context) {
    showDialog(
      context: context,
      builder: (context) => AlertDialog(
        title: const Text('注册信息'),
        content: Text(
          '姓名：${_nameController.text}\n'
          '邮箱：${_emailController.text}\n'
          '套餐：$_plan',
        ),
        actions: [
          TextButton(
            onPressed: () => Navigator.pop(context),
            child: const Text('确定'),
          ),
        ],
      ),
    );
  }

  @override
  void dispose() {
    _nameController.dispose();
    _emailController.dispose();
    super.dispose();
  }
}

10.5 自定义控制按钮

import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatefulWidget {
  const MyApp({super.key});

  @override
  State<MyApp> createState() => _MyAppState();
}

class _MyAppState extends State<MyApp> {
  int _currentStep = 0;
  final int _totalSteps = 3;

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      theme: ThemeData(useMaterial3: true),
      home: Scaffold(
        appBar: AppBar(title: const Text('自定义 Stepper 控制按钮')),
        body: Stepper(
          currentStep: _currentStep,
          onStepContinue: () {
            if (_currentStep < _totalSteps - 1) {
              setState(() => _currentStep++);
            }
          },
          onStepCancel: () {
            if (_currentStep > 0) {
              setState(() => _currentStep--);
            }
          },
          // 自定义底部按钮
          controlsBuilder: (context, details) {
            final isLastStep = _currentStep == _totalSteps - 1;
            return Padding(
              padding: const EdgeInsets.only(top: 16),
              child: Row(
                children: [
                  FilledButton(
                    onPressed: details.onStepContinue,
                    child: Text(isLastStep ? '提交' : '下一步'),
                  ),
                  const SizedBox(width: 12),
                  if (_currentStep > 0)
                    OutlinedButton(
                      onPressed: details.onStepCancel,
                      child: const Text('上一步'),
                    ),
                ],
              ),
            );
          },
          steps: [
            Step(
              title: const Text('步骤一'),
              content: const Padding(
                padding: EdgeInsets.symmetric(vertical: 8),
                child: Text('这是第一步的内容'),
              ),
              isActive: _currentStep >= 0,
              state: _currentStep > 0 ? StepState.complete : StepState.editing,
            ),
            Step(
              title: const Text('步骤二'),
              content: const Padding(
                padding: EdgeInsets.symmetric(vertical: 8),
                child: Text('这是第二步的内容'),
              ),
              isActive: _currentStep >= 1,
              state: _currentStep > 1
                  ? StepState.complete
                  : _currentStep == 1
                      ? StepState.editing
                      : StepState.indexed,
            ),
            Step(
              title: const Text('步骤三'),
              content: const Padding(
                padding: EdgeInsets.symmetric(vertical: 8),
                child: Text('这是最后一步的内容'),
              ),
              isActive: _currentStep >= 2,
              state: _currentStep == 2 ? StepState.editing : StepState.indexed,
            ),
          ],
        ),
      ),
    );
  }
}

最佳实践：

  • StepperType.vertical 适用于步骤较多或每步内容较复杂的场景，StepperType.horizontal 适用于步骤较少（2-4 步）的简洁流程。
  • 合理使用 StepState（complete、editing、error 等）来反映每一步的完成状态。
  • 通过 controlsBuilder 自定义底部按钮，可以实现更贴合产品设计的交互方式。
  • 在最后一步将"继续"按钮文案改为"提交"或"完成"，给用户明确的操作预期。

11. 本课小结

本课系统学习了 Flutter 中 10 大类信息展示组件，涵盖了从简单的卡片容器到复杂的数据表格和步骤条。以下是各组件的核心要点回顾：

组件 核心用途 关键特性
Card 信息容器 elevation、shape、clipBehavior、surfaceTintColor（M3）
ListTile 标准列表项 leading/title/subtitle/trailing 四段式布局
ExpansionTile 可展开列表项 内置展开动画，controlAffinity 控制箭头位置
ExpansionPanelList 手风琴面板 expansionCallback 实现单开/多开模式
DataTable 数据表格 内置排序（sortColumnIndex）和行选中功能
Chip 家族 标签组件 五种类型各有定位：展示、操作、多选、单选、输入
Badge 角标 Material 3 组件，Badge.count 快捷显示数字
Divider 分割线 height 是总高度，thickness 是线的粗细
ProgressIndicator 进度指示 value 为 null 是不确定进度，0.0~1.0 是确定进度
Stepper 步骤条 vertical/horizontal 两种布局，controlsBuilder 自定义按钮

学习建议：

  1. Card + ListTile 是最常用的组合，几乎每个应用都会用到，建议反复练习。
  2. Chip 家族要理解各类型的使用场景区别，避免混用。
  3. DataTable 适合小数据量，大数据量请使用 PaginatedDataTable。
  4. 进度指示器要根据场景选择确定/不确定模式，给用户准确的反馈。
