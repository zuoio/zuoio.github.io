1. 按钮组件概述

Flutter 提供了一套基于 Material Design 规范的按钮组件体系。自 Flutter 2.0 起，旧版按钮（RaisedButton、FlatButton、OutlineButton）已被废弃，取而代之的是全新的按钮家族：

按钮类型 视觉特征 强调程度 典型场景
ElevatedButton 有背景色和阴影 高 主要操作（提交、确认）
FilledButton 有背景色，无阴影（M3） 高 主要操作（Material 3 推荐）
FilledButton.tonal 浅色背景填充（M3） 中高 次要但重要的操作
TextButton 无边框无背景 低 次要操作、对话框按钮
OutlinedButton 有边框无背景 中 替代操作、取消
IconButton 仅图标 低 工具栏操作
FloatingActionButton 浮动圆形/扩展形 最高 页面主操作（如新建）

按钮的强调程度从高到低：FAB > ElevatedButton / FilledButton > FilledButton.tonal > OutlinedButton > TextButton

在实际开发中，应根据操作的重要性选择合适的按钮类型，避免在同一页面使用过多高强调按钮，以免分散用户注意力。

2. ElevatedButton — 凸起按钮

2.1 简介与适用场景

ElevatedButton 是带有背景色和阴影的凸起按钮，视觉上最为突出。适用于需要引导用户执行主要操作的场景，例如"提交表单"、"登录"、"下一步"等。

适用场景：当页面需要一个显眼的行动号召按钮（CTA）时使用。在 Material 3 中，FilledButton 正逐步替代 ElevatedButton 成为首选的高强调按钮。

2.2 基本用法

import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: const Text('ElevatedButton 示例')),
        body: Center(
          child: ElevatedButton(
            // onPressed 为 null 时按钮处于禁用状态
            onPressed: () {
              debugPrint('按钮被点击了');
            },
            child: const Text('点击我'),
          ),
        ),
      ),
    );
  }
}

2.3 样式自定义

使用 ElevatedButton.styleFrom 快速定义按钮样式：

import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: const Text('ElevatedButton 样式自定义')),
        body: Center(
          child: ElevatedButton(
            onPressed: () {},
            style: ElevatedButton.styleFrom(
              // 背景颜色
              backgroundColor: Colors.deepPurple,
              // 前景颜色（文字和图标）
              foregroundColor: Colors.white,
              // 阴影颜色
              shadowColor: Colors.deepPurpleAccent,
              // 阴影高度
              elevation: 8,
              // 内边距
              padding: const EdgeInsets.symmetric(
                horizontal: 32,
                vertical: 16,
              ),
              // 最小尺寸
              minimumSize: const Size(200, 50),
              // 形状：圆角矩形
              shape: RoundedRectangleBorder(
                borderRadius: BorderRadius.circular(12),
              ),
              // 文字样式
              textStyle: const TextStyle(
                fontSize: 18,
                fontWeight: FontWeight.bold,
              ),
            ),
            child: const Text('自定义样式按钮'),
          ),
        ),
      ),
    );
  }
}

2.4 长按回调与工具提示

import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: const Text('ElevatedButton 其他回调')),
        body: Center(
          child: ElevatedButton(
            onPressed: () {
              debugPrint('单击');
            },
            onLongPress: () {
              debugPrint('长按');
            },
            // 鼠标悬停时的提示文字（Web 和桌面端可见）

          ),
        ),
      ),
    );
  }
}

注意事项：onPressed 和 onLongPress 都为 null 时，按钮才会进入禁用状态。只要其中一个不为 null，按钮就是可交互的。

3. TextButton — 文字按钮

3.1 简介与适用场景

TextButton 是没有背景色和边框的纯文字按钮，视觉上最为低调。适用于次要操作或不需要过多吸引注意力的场景。

适用场景：

  • 对话框中的"取消"按钮
  • 列表项中的附加操作（如"查看更多"）
  • 工具栏中的文字操作
  • 与 ElevatedButton 搭配使用时作为次要操作

3.2 基本用法

import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: const Text('TextButton 示例')),
        body: Center(
          child: Column(
            mainAxisAlignment: MainAxisAlignment.center,
            children: [
              // 基础文字按钮
              TextButton(
                onPressed: () {
                  debugPrint('文字按钮被点击');
                },
                child: const Text('文字按钮'),
              ),
              const SizedBox(height: 16),
              // 对话框中的典型用法
              Row(
                mainAxisAlignment: MainAxisAlignment.center,
                children: [
                  TextButton(
                    onPressed: () {
                      debugPrint('取消');
                    },
                    child: const Text('取消'),
                  ),
                  const SizedBox(width: 8),
                  ElevatedButton(
                    onPressed: () {
                      debugPrint('确认');
                    },
                    child: const Text('确认'),
                  ),
                ],
              ),
            ],
          ),
        ),
      ),
    );
  }
}

  }
}

4. OutlinedButton — 边框按钮

4.1 简介与适用场景

OutlinedButton 带有边框但没有背景填充，视觉强调程度介于 ElevatedButton 和 TextButton 之间。

适用场景：

  • 作为主要操作的替代方案（如"取消订单"）
  • 当需要按钮有一定存在感但又不想过于突出时
  • 表单中的"重置"操作
  • 与 ElevatedButton 并排使用，表示次要操作

4.2 基本用法

import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: const Text('OutlinedButton 示例')),
        body: Center(
          child: Column(
            mainAxisAlignment: MainAxisAlignment.center,
            children: [
              // 基础边框按钮
              OutlinedButton(
                onPressed: () {
                  debugPrint('边框按钮被点击');
                },
                child: const Text('边框按钮'),
              ),
              const SizedBox(height: 16),
              // 典型搭配：主操作 + 次操作
              Row(
                mainAxisAlignment: MainAxisAlignment.center,
                children: [
                  OutlinedButton(
                    onPressed: () {},
                    child: const Text('取消'),
                  ),
                  const SizedBox(width: 12),
                  ElevatedButton(
                    onPressed: () {},
                    child: const Text('提交'),
                  ),
                ],
              ),
            ],
          ),
        ),
      ),
    );
  }
}

最佳实践：OutlinedButton 的边框颜色通常应与前景色（文字颜色）保持一致或协调，以确保视觉统一。

5. IconButton — 图标按钮

5.1 简介与适用场景

IconButton 是仅包含图标的按钮，不带文字标签。点击时会显示水波纹效果。

适用场景：

  • AppBar 中的导航和操作图标（返回、搜索、更多菜单）
  • 工具栏中的操作按钮
  • 列表项中的快捷操作（收藏、删除、分享）
  • 任何含义通过图标即可清晰传达的交互

5.2 基本用法

import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(
          title: const Text('IconButton 示例'),
          actions: [
            // AppBar 中常见的图标按钮
            IconButton(
              icon: const Icon(Icons.search),
              tooltip: '搜索',
              onPressed: () {
                debugPrint('搜索');
              },
            ),
            IconButton(
              icon: const Icon(Icons.more_vert),
              tooltip: '更多',
              onPressed: () {
                debugPrint('更多选项');
              },
            ),
          ],
        ),
        body: Center(
          child: Row(
            mainAxisAlignment: MainAxisAlignment.center,
            children: [
              IconButton(
                icon: const Icon(Icons.thumb_up),
                tooltip: '点赞',
                onPressed: () {},
              ),
              IconButton(
                icon: const Icon(Icons.comment),
                tooltip: '评论',
                onPressed: () {},
              ),
              IconButton(
                icon: const Icon(Icons.share),
                tooltip: '分享',
                onPressed: () {},
              ),
            ],
          ),
        ),
      ),
    );
  }
}

注意事项：

  • IconButton 默认有一个 48x48 的最小点击区域（符合无障碍访问要求），即使图标本身更小。
  • 务必为每个 IconButton 添加 tooltip 属性，这对于屏幕阅读器和鼠标悬停提示非常重要。
  • Material 3 新增了 IconButton.filled、IconButton.filledTonal、IconButton.outlined 三种变体构造函数。

6. FloatingActionButton — 浮动操作按钮

6.1 简介与适用场景

FloatingActionButton（通常简称 FAB）是悬浮在页面上方的圆形按钮，拥有最高的视觉强调级别。每个页面最多应该只有一个 FAB。

适用场景：

  • 页面中最重要、最常用的操作（如新建、编写、拍照）
  • 需要始终可见、随时可触达的核心操作

不适用场景：

  • 破坏性操作（如删除）
  • 非主要操作
  • 已有明显操作入口的页面

6.2 基本用法

import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: const Text('FloatingActionButton 示例')),
        body: const Center(
          child: Text('页面内容'),
        ),
        // 标准 FAB：圆形，包含一个图标
        floatingActionButton: FloatingActionButton(
          onPressed: () {
            debugPrint('FAB 被点击');
          },
          tooltip: '新建',
          child: const Icon(Icons.add),
        ),
      ),
    );
  }
}

6.3 FAB 的多种变体

import 'package:flutter/material.dart';

              FloatingActionButton(
                onPressed: () {},
                heroTag: 'regular',
                child: const Icon(Icons.edit),
              ),
              const SizedBox(height: 24),
              // 大型 FAB
              FloatingActionButton.large(
                onPressed: () {},
                heroTag: 'large',
                child: const Icon(Icons.camera_alt),
              ),
              const SizedBox(height: 24),
              // 扩展型 FAB：图标 + 文字
              FloatingActionButton.extended(
                onPressed: () {},
                heroTag: 'extended',
                icon: const Icon(Icons.navigation),
                label: const Text('开始导航'),
              ),
            ],
          ),
        ),
      ),
    );
  }
}

  • 在使用 BottomAppBar 时，可搭配 FloatingActionButtonLocation.centerDocked 实现 FAB 嵌入底部栏的效果。

7. FilledButton 与 FilledButton.tonal — Material 3 新增按钮

7.1 简介

FilledButton 和 FilledButton.tonal 是 Material 3 新增的按钮类型，用于补充按钮体系中"高强调"和"中高强调"的空缺位置。

  • FilledButton：使用主题色填充背景，没有阴影。在 Material 3 中，它是推荐替代 ElevatedButton 的首选高强调按钮。
  • FilledButton.tonal：使用柔和的色调颜色填充背景（通常是 secondaryContainer 色），强调程度低于 FilledButton 但高于 OutlinedButton。

适用场景：

  • FilledButton：主要操作，与 Material 3 设计风格保持一致
  • FilledButton.tonal：重要但不是最主要的操作，如"保存草稿"（而"发布"用 FilledButton）

7.2 基本用法

import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      // 启用 Material 3
      theme: ThemeData(useMaterial3: true),
      home: Scaffold(
        appBar: AppBar(title: const Text('FilledButton 示例')),
        body: Center(
          child: Column(
            mainAxisAlignment: MainAxisAlignment.center,
            children: [
              // FilledButton：全填充
              FilledButton(
                onPressed: () {
                  debugPrint('FilledButton 被点击');
                },
                child: const Text('发布'),
              ),
              const SizedBox(height: 16),
              // FilledButton.tonal：色调填充
              FilledButton.tonal(
                onPressed: () {
                  debugPrint('FilledButton.tonal 被点击');
                },
                child: const Text('保存草稿'),
              ),
              const SizedBox(height: 32),
              // 典型组合：体现操作层级
              Row(
                mainAxisAlignment: MainAxisAlignment.center,
                children: [
                  TextButton(
                    onPressed: () {},
                    child: const Text('取消'),
                  ),
                  const SizedBox(width: 8),
                  FilledButton.tonal(
                    onPressed: () {},
                    child: const Text('保存草稿'),
                  ),
                  const SizedBox(width: 8),
                  FilledButton(
                    onPressed: () {},
                    child: const Text('发布'),
                  ),
                ],
              ),
            ],
          ),
        ),
      ),
    );
  }
}

9. ButtonStyle 深入详解

ButtonStyle 是 Flutter 按钮样式系统的核心。所有 Material 按钮（ElevatedButton、TextButton、OutlinedButton、FilledButton、IconButton）都通过 ButtonStyle 来控制外观。

9.1 两种创建方式

方式一：使用 styleFrom 静态方法（推荐入门使用）

每种按钮都有自己的 styleFrom 方法，它接收简单的值类型参数，内部自动转换为 WidgetStateProperty：

// ElevatedButton.styleFrom 示例
ElevatedButton.styleFrom(
  backgroundColor: Colors.blue,        // 背景色
  foregroundColor: Colors.white,        // 前景色（文字、图标）
  disabledBackgroundColor: Colors.grey, // 禁用时背景色
  disabledForegroundColor: Colors.white54, // 禁用时前景色
  shadowColor: Colors.blueAccent,       // 阴影颜色
  elevation: 4,                         // 阴影高度
  padding: EdgeInsets.all(16),          // 内边距
  minimumSize: Size(120, 48),           // 最小尺寸
  maximumSize: Size.infinite,           // 最大尺寸
  fixedSize: Size(200, 50),             // 固定尺寸
  side: BorderSide(color: Colors.blue), // 边框
  shape: RoundedRectangleBorder(...),   // 形状
  textStyle: TextStyle(fontSize: 16),   // 文字样式
  overlayColor: Colors.white24,         // 水波纹覆盖色
  iconSize: 20,                         // 图标大小
  iconColor: Colors.white,              // 图标颜色
)

方式二：直接构造 ButtonStyle（高级用法）

ButtonStyle 构造函数的每个参数都是 WidgetStateProperty 类型，可以根据按钮的不同交互状态返回不同的值：

ButtonStyle(
  backgroundColor: WidgetStateProperty.resolveWith<Color?>(
    (Set<WidgetState> states) {
      if (states.contains(WidgetState.pressed)) {
        return Colors.blue.shade700; // 按下状态
      }
      if (states.contains(WidgetState.hovered)) {
        return Colors.blue.shade600; // 鼠标悬停
      }
      if (states.contains(WidgetState.disabled)) {
        return Colors.grey.shade300; // 禁用状态
      }
      return Colors.blue; // 默认状态
    },
  ),
  foregroundColor: WidgetStateProperty.all(Colors.white),
  elevation: WidgetStateProperty.resolveWith<double?>(
    (Set<WidgetState> states) {
      if (states.contains(WidgetState.pressed)) {
        return 2; // 按下时降低阴影
      }
      if (states.contains(WidgetState.hovered)) {
        return 8; // 悬停时升高阴影
      }
      return 4; // 默认阴影
    },
  ),
)

  1. 按钮 Widget 上直接设置的 style 属性
  2. 通过 Theme Widget 局部覆盖的组件主题
  3. ThemeData 中设置的全局组件主题（如 elevatedButtonTheme）
  4. 按钮类型的默认样式（defaultStyleOf）

10. 按钮的禁用状态

所有 Material 按钮在 onPressed 和 onLongPress 同时为 null 时进入禁用状态。禁用按钮不可点击，并且视觉上会变灰淡化。

10.1 禁用状态示例

import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatefulWidget {
  const MyApp({super.key});

  @override
  State<MyApp> createState() => _MyAppState();
}

class _MyAppState extends State<MyApp> {
  bool _isLoading = false;
  bool _agreed = false;

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      theme: ThemeData(useMaterial3: true),
      home: Scaffold(
        appBar: AppBar(title: const Text('按钮禁用状态')),
        body: Center(
          child: Padding(
            padding: const EdgeInsets.all(24),
            child: Column(
              mainAxisAlignment: MainAxisAlignment.center,
              children: [
                // 始终禁用的按钮
                const ElevatedButton(
                  onPressed: null, // null = 禁用
                  child: Text('始终禁用'),
                ),
                const SizedBox(height: 24),

                // 根据条件禁用：必须同意条款才能提交
                Row(
                  mainAxisAlignment: MainAxisAlignment.center,
                  children: [
                    Checkbox(
                      value: _agreed,
                      onChanged: (value) {
                        setState(() {
                          _agreed = value ?? false;
                        });
                      },
                    ),
                    const Text('我同意用户条款'),
                  ],
                ),
                const SizedBox(height: 8),
                FilledButton(
                  // 未同意时禁用按钮
                  onPressed: _agreed
                      ? () {
                          debugPrint('提交');
                        }
                      : null,
                  child: const Text('提交'),
                ),
                const SizedBox(height: 24),

                // 加载状态时禁用按钮并显示加载指示器
                FilledButton(
                  onPressed: _isLoading
                      ? null
                      : () async {
                          setState(() => _isLoading = true);
                          // 模拟网络请求
                          await Future.delayed(
                            const Duration(seconds: 2),
                          );
                          setState(() => _isLoading = false);
                        },
                  child: _isLoading
                      ? const SizedBox(
                          width: 20,
                          height: 20,
                          child: CircularProgressIndicator(
                            strokeWidth: 2,
                            color: Colors.white,
                          ),
                        )
                      : const Text('保存'),
                ),

                const SizedBox(height: 24),

                // 各种按钮的禁用状态对比
                const Text(
                  '各按钮禁用状态对比：',
                  style: TextStyle(fontWeight: FontWeight.bold),
                ),
                const SizedBox(height: 12),
                const Wrap(
                  spacing: 8,
                  runSpacing: 8,
                  children: [
                    ElevatedButton(
                      onPressed: null,
                      child: Text('Elevated'),
                    ),
                    FilledButton(
                      onPressed: null,
                      child: Text('Filled'),
                    ),
                    OutlinedButton(
                      onPressed: null,
                      child: Text('Outlined'),
                    ),
                    TextButton(
                      onPressed: null,
                      child: Text('Text'),
                    ),
                  ],
                ),
              ],
            ),
          ),
        ),
      ),
    );
  }
}

10.2 自定义禁用状态样式

import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: const Text('自定义禁用样式')),
        body: Center(
          child: ElevatedButton(
            onPressed: null,
            style: ElevatedButton.styleFrom(
              // styleFrom 提供了专门的禁用状态颜色参数
              disabledBackgroundColor: Colors.red.shade100,
              disabledForegroundColor: Colors.red.shade300,
            ),
            child: const Text('自定义禁用颜色'),
          ),
        ),
      ),
    );
  }
}

最佳实践：

  • 禁用按钮时应让用户理解为什么被禁用（如在旁边显示提示文字）。
  • 加载中的按钮应同时禁用并显示加载指示器，防止重复提交。
  • 不要仅靠颜色区分禁用状态，还应降低透明度让视觉差异更明显。

11. 本课小结

核心知识点回顾

  1. 按钮类型选择：根据操作的重要性选择合适的按钮类型

    • 最高强调：FloatingActionButton
    • 高强调：ElevatedButton / FilledButton
    • 中高强调：FilledButton.tonal
    • 中强调：OutlinedButton
    • 低强调：TextButton
    • 纯图标：IconButton

  2. 样式定制：两种方式

    • 简单场景：使用 XxxButton.styleFrom(...) 快速设置
    • 复杂场景：使用 ButtonStyle(...) 构造函数 + WidgetStateProperty.resolveWith 精细控制每个交互状态

  3. ButtonStyle 核心属性：

    • backgroundColor / foregroundColor：背景色与前景色
    • elevation / shadowColor：阴影控制
    • shape：按钮形状（RoundedRectangleBorder、StadiumBorder、CircleBorder 等）
    • padding：内边距
    • minimumSize / maximumSize / fixedSize：尺寸约束
    • overlayColor：交互时的覆盖层颜色
    • side：边框（OutlinedButton 常用）
    • textStyle：文字样式

  4. 全局主题：通过 ThemeData 中的 elevatedButtonTheme、textButtonTheme 等统一管理按钮样式

  5. 禁用状态：onPressed: null 时按钮禁用，可通过 disabledBackgroundColor 和 disabledForegroundColor 自定义禁用外观

  6. Material 3 新按钮：FilledButton 和 FilledButton.tonal 补充了按钮强调层级体系

常见误区

误区 正确做法
所有按钮都用 ElevatedButton 根据操作重要性选择合适的按钮类型
用 Container + GestureDetector 代替按钮 使用 Material 按钮，它们自带无障碍支持和交互反馈
在按钮上覆盖太多自定义样式 优先使用全局主题统一管理，单个按钮只做局部微调
忘记处理按钮的加载和禁用状态 网络请求期间应禁用按钮并显示加载指示
同一页面使用多个 FAB 但未设置 heroTag 多个 FAB 时必须设置不同的 heroTag

练习三：按钮样式工厂（进阶实践）

编写一个 AppButtonStyles 工具类，包含以下静态方法：

  • secondary()：返回次要操作的 ButtonStyle（白色背景，蓝色文字，蓝色边框）
  • danger()：返回危险操作的 ButtonStyle（红色背景，白色文字）

每个样式都要正确处理 hovered、pressed、disabled 三种交互状态。
