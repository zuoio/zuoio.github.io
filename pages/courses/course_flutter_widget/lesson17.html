课程系列：Flutter 常用组件系统学习
适用人群：有 Dart 语言基础，了解 Flutter 基本概念的开发者
课程目标：系统掌握 Flutter 中各种对话框、底部弹出层、提示消息等"弹出层"组件的使用方式，理解其异步返回值机制

弹出层的核心概念

Flutter 中的"弹出层"组件有一个共同特征：它们通过方法调用（而非声明式组件嵌套）来显示，且大多数方法返回一个 Future，代表用户关闭弹出层后的结果。这是因为弹出层本质上是在当前 Navigator 之上 push 了一个新的 Route（对话框）或插入了一个 Overlay（SnackBar、Tooltip 等）。

理解这一点非常重要：showDialog、showModalBottomSheet 等方法都是异步的，返回 Future<T?>。当用户关闭弹出层时，这个 Future 会 complete，你可以通过 await 或 .then() 获取用户的选择结果。

1. showDialog 与 AlertDialog

1.1 基本用法

showDialog 是显示对话框的核心方法，它接收一个 builder 参数来构建对话框内容。AlertDialog 是 Material Design 中最常用的对话框组件。

import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      theme: ThemeData(colorSchemeSeed: Colors.blue, useMaterial3: true),
      home: const AlertDialogDemo(),
    );
  }
}

class AlertDialogDemo extends StatefulWidget {
  const AlertDialogDemo({super.key});

  @override
  State<AlertDialogDemo> createState() => _AlertDialogDemoState();
}

class _AlertDialogDemoState extends State<AlertDialogDemo> {
  String _result = '尚未选择';

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('AlertDialog 演示')),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: [
            Text('用户选择: $_result', style: const TextStyle(fontSize: 18)),
            const SizedBox(height: 24),
            ElevatedButton(
              onPressed: () => _showBasicDialog(context),
              child: const Text('显示基础对话框'),
            ),
            const SizedBox(height: 12),
            ElevatedButton(
              onPressed: () => _showDialogWithResult(context),
              child: const Text('显示带返回值的对话框'),
            ),
          ],
        ),
      ),
    );
  }

  /// 基础 AlertDialog
  void _showBasicDialog(BuildContext context) {
    showDialog(
      context: context,
      // 点击对话框外部区域是否关闭，默认为 true
      barrierDismissible: true,
      // 遮罩层颜色
      barrierColor: Colors.black54,
      builder: (BuildContext dialogContext) {
        return AlertDialog(
          // 对话框标题
          title: const Text('提示'),
          // 对话框内容
          content: const Text('这是一个基础的 AlertDialog，点击按钮或外部区域关闭。'),
          // 操作按钮列表
          actions: [
            TextButton(
              onPressed: () => Navigator.pop(dialogContext),
              child: const Text('知道了'),
            ),
          ],
        );
      },
    );
  }

  /// 带返回值的 AlertDialog —— 展示 Future 机制
  Future<void> _showDialogWithResult(BuildContext context) async {
    // showDialog 返回 Future<T?>，T 由 Navigator.pop 的第二个参数决定
    final bool? confirmed = await showDialog<bool>(
      context: context,
      barrierDismissible: false, // 禁止点击外部关闭，强制用户做出选择
      builder: (BuildContext dialogContext) {
        return AlertDialog(
          icon: const Icon(Icons.warning_amber_rounded, size: 48),
          title: const Text('确认删除'),
          content: const Text('此操作不可撤销，确定要删除这条记录吗？'),
          actions: [
            TextButton(
              onPressed: () => Navigator.pop(dialogContext, false), // 返回 false
              child: const Text('取消'),
            ),
            FilledButton(
              onPressed: () => Navigator.pop(dialogContext, true), // 返回 true
              child: const Text('确认删除'),
            ),
          ],
        );
      },
    );

    // 检查 mounted 防止异步操作后 Widget 已被销毁
    if (!mounted) return;

    setState(() {
      if (confirmed == true) {
        _result = '用户确认删除';
      } else {
        _result = '用户取消删除';
      }
    });
  }
}

1.3 showDialog 的关键参数

参数 类型 说明
context BuildContext 用于定位 Navigator
builder WidgetBuilder 构建对话框内容
barrierDismissible bool 点击遮罩是否关闭（默认 true）
barrierColor Color? 遮罩层颜色（默认半透明黑色）
barrierLabel String? 无障碍标签

注意事项：showDialog 的 builder 回调参数中的 context（示例中命名为 dialogContext）与外层的 context 不同。在调用 Navigator.pop 关闭对话框时，应使用 builder 提供的 context，这样可以精确地 pop 当前对话框的 Route。

2. SimpleDialog

SimpleDialog 适用于让用户在几个选项中做出选择的场景。每个选项由 SimpleDialogOption 表示。

import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      theme: ThemeData(colorSchemeSeed: Colors.teal, useMaterial3: true),
      home: const SimpleDialogDemo(),
    );
  }
}

class SimpleDialogDemo extends StatefulWidget {
  const SimpleDialogDemo({super.key});

  @override
  State<SimpleDialogDemo> createState() => _SimpleDialogDemoState();
}

class _SimpleDialogDemoState extends State<SimpleDialogDemo> {
  String _selectedLanguage = '未选择';

  Future<void> _showLanguageDialog() async {
    final languages = ['Dart', 'Kotlin', 'Swift', 'TypeScript', 'Rust'];

    // SimpleDialog 同样通过 showDialog 显示
    final String? result = await showDialog<String>(
      context: context,
      builder: (BuildContext dialogContext) {
        return SimpleDialog(
          title: const Text('选择你最喜欢的编程语言'),
          // children 通常是一组 SimpleDialogOption
          children: languages.map((lang) {
            return SimpleDialogOption(
              onPressed: () => Navigator.pop(dialogContext, lang),
              padding: const EdgeInsets.symmetric(horizontal: 24, vertical: 12),
              child: Row(
                children: [
                  const Icon(Icons.code, size: 20),
                  const SizedBox(width: 12),
                  Text(lang, style: const TextStyle(fontSize: 16)),
                ],
              ),
            );
          }).toList(),
        );
      },
    );

    if (!mounted) return;
    if (result != null) {
      setState(() => _selectedLanguage = result);
    }
  }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('SimpleDialog 演示')),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: [
            Text('已选择: $_selectedLanguage', style: const TextStyle(fontSize: 18)),
            const SizedBox(height: 24),
            ElevatedButton(
              onPressed: _showLanguageDialog,
              child: const Text('选择编程语言'),
            ),
          ],
        ),
      ),
    );
  }
}

最佳实践：SimpleDialog 适合 2-6 个选项的简单选择场景。如果选项过多，考虑使用 showModalBottomSheet 配合 ListView 替代，体验更好。

3. Dialog

Dialog 是 AlertDialog 和 SimpleDialog 的基类，当内置对话框无法满足设计需求时，可以直接使用 Dialog 进行完全自定义。

import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      theme: ThemeData(colorSchemeSeed: Colors.deepPurple, useMaterial3: true),
      home: const CustomDialogDemo(),
    );
  }
}

class CustomDialogDemo extends StatelessWidget {
  const CustomDialogDemo({super.key});

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('自定义 Dialog 演示')),
      body: Center(
        child: ElevatedButton(
          onPressed: () => _showCustomDialog(context),
          child: const Text('显示自定义对话框'),
        ),
      ),
    );
  }

  void _showCustomDialog(BuildContext context) {
    showDialog(
      context: context,
      builder: (BuildContext dialogContext) {
        return Dialog(
          // 自定义形状：圆角矩形
          shape: RoundedRectangleBorder(
            borderRadius: BorderRadius.circular(20),
          ),
          // 阴影高度
          elevation: 16,
          // 对话框与屏幕边缘的距离
          insetPadding: const EdgeInsets.symmetric(horizontal: 24, vertical: 40),
          // 完全自定义的子组件
          child: Container(
            padding: const EdgeInsets.all(24),
            child: Column(
              mainAxisSize: MainAxisSize.min,
              children: [
                // 顶部图片或图标区
                Container(
                  width: 80,
                  height: 80,
                  decoration: BoxDecoration(
                    color: Theme.of(dialogContext).colorScheme.primaryContainer,
                    shape: BoxShape.circle,
                  ),
                  child: Icon(
                    Icons.celebration,
                    size: 40,
                    color: Theme.of(dialogContext).colorScheme.primary,
                  ),
                ),
                const SizedBox(height: 20),
                const Text(
                  '恭喜完成！',
                  style: TextStyle(fontSize: 22, fontWeight: FontWeight.bold),
                ),
                const SizedBox(height: 12),
                const Text(
                  '你已成功完成所有课程任务。继续保持学习的热情吧！',
                  textAlign: TextAlign.center,
                  style: TextStyle(fontSize: 15, color: Colors.grey),
                ),
                const SizedBox(height: 24),
                // 自定义按钮
                SizedBox(
                  width: double.infinity,
                  child: FilledButton(
                    onPressed: () => Navigator.pop(dialogContext),
                    child: const Text('太棒了！'),
                  ),
                ),
                const SizedBox(height: 8),
                TextButton(
                  onPressed: () => Navigator.pop(dialogContext),
                  child: const Text('稍后查看'),
                ),
              ],
            ),
          ),
        );
      },
    );
  }
}

3.1 Dialog 与 AlertDialog 的关系

组件 定位 适用场景
Dialog 基类，提供基础样式（形状、阴影、内边距） 完全自定义布局
AlertDialog 继承自 Dialog，预设了 title/content/actions 布局 标准确认/警告对话框
SimpleDialog 继承自 Dialog，预设了 title/children 布局 简单选项列表

最佳实践：优先使用 AlertDialog 或 SimpleDialog，只在它们无法满足需求时才用 Dialog 自定义。直接使用 Dialog 需要自己处理所有布局细节，但灵活性最高。

4. showModalBottomSheet

showModalBottomSheet 用于从屏幕底部弹出一个模态面板。用户必须关闭它才能继续操作下面的页面，这一点与非模态的 showBottomSheet 不同。

4.1 基础用法

import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      theme: ThemeData(colorSchemeSeed: Colors.indigo, useMaterial3: true),
      home: const BottomSheetDemo(),
    );
  }
}

class BottomSheetDemo extends StatefulWidget {
  const BottomSheetDemo({super.key});

  @override
  State<BottomSheetDemo> createState() => _BottomSheetDemoState();
}

class _BottomSheetDemoState extends State<BottomSheetDemo> {
  String _selected = '无';

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('ModalBottomSheet 演示')),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: [
            Text('已选择: $_selected', style: const TextStyle(fontSize: 18)),
            const SizedBox(height: 24),
            ElevatedButton(
              onPressed: _showBasicBottomSheet,
              child: const Text('基础底部弹出'),
            ),
            const SizedBox(height: 12),
            ElevatedButton(
              onPressed: _showFullScreenBottomSheet,
              child: const Text('全屏底部弹出'),
            ),
          ],
        ),
      ),
    );
  }

  Future<void> _showBasicBottomSheet() async {
    final result = await showModalBottomSheet<String>(
      context: context,
      // 自定义顶部圆角
      shape: const RoundedRectangleBorder(
        borderRadius: BorderRadius.vertical(top: Radius.circular(20)),
      ),
      // 是否可以点击外部关闭
      isDismissible: true,
      // 是否可以下拉关闭
      enableDrag: true,
      // 是否使用安全区域（避开刘海等）
      useSafeArea: true,
      builder: (BuildContext sheetContext) {
        return Padding(
          padding: const EdgeInsets.all(16),
          child: Column(
            mainAxisSize: MainAxisSize.min, // 高度自适应内容
            crossAxisAlignment: CrossAxisAlignment.stretch,
            children: [
              // 拖拽指示条
              Center(
                child: Container(
                  width: 40,
                  height: 4,
                  margin: const EdgeInsets.only(bottom: 16),
                  decoration: BoxDecoration(
                    color: Colors.grey[400],
                    borderRadius: BorderRadius.circular(2),
                  ),
                ),
              ),
              const Text(
                '选择分享方式',
                style: TextStyle(fontSize: 18, fontWeight: FontWeight.bold),
              ),
              const SizedBox(height: 16),
              _buildShareOption(sheetContext, Icons.link, '复制链接', 'link'),
              _buildShareOption(sheetContext, Icons.chat, '发送给朋友', 'friend'),
              _buildShareOption(sheetContext, Icons.share, '分享到朋友圈', 'moments'),
              _buildShareOption(sheetContext, Icons.bookmark, '收藏', 'favorite'),
              const SizedBox(height: 8),
            ],
          ),
        );
      },
    );

    if (!mounted || result == null) return;
    setState(() => _selected = result);
  }

  Widget _buildShareOption(
    BuildContext sheetContext,
    IconData icon,
    String label,
    String value,
  ) {
    return ListTile(
      leading: Icon(icon),
      title: Text(label),
      shape: RoundedRectangleBorder(borderRadius: BorderRadius.circular(12)),
      onTap: () => Navigator.pop(sheetContext, value),
    );
  }

  Future<void> _showFullScreenBottomSheet() async {
    await showModalBottomSheet(
      context: context,
      // 关键属性：允许底部弹出占据全屏高度
      isScrollControlled: true,
      shape: const RoundedRectangleBorder(
        borderRadius: BorderRadius.vertical(top: Radius.circular(20)),
      ),
      // 约束底部弹出面板的最大宽度/高度
      constraints: BoxConstraints(
        maxHeight: MediaQuery.of(context).size.height * 0.85,
      ),
      builder: (BuildContext sheetContext) {
        return Padding(
          padding: const EdgeInsets.all(16),
          child: Column(
            children: [
              // 拖拽指示条
              Center(
                child: Container(
                  width: 40,
                  height: 4,
                  margin: const EdgeInsets.only(bottom: 16),
                  decoration: BoxDecoration(
                    color: Colors.grey[400],
                    borderRadius: BorderRadius.circular(2),
                  ),
                ),
              ),
              const Text(
                '评论列表',
                style: TextStyle(fontSize: 18, fontWeight: FontWeight.bold),
              ),
              const Divider(),
              // 用 Expanded 包裹可滚动内容
              Expanded(
                child: ListView.builder(
                  itemCount: 30,
                  itemBuilder: (context, index) {
                    return ListTile(
                      leading: CircleAvatar(child: Text('${index + 1}')),
                      title: Text('用户${index + 1}的评论'),
                      subtitle: Text('这是第${index + 1}条评论内容...'),
                    );
                  },
                ),
              ),
            ],
          ),
        );
      },
    );
  }
}

4.2 showModalBottomSheet 的关键参数

参数 类型 说明
builder WidgetBuilder 构建底部面板内容
shape ShapeBorder? 面板形状（通常设置顶部圆角）
isScrollControlled bool 为 true 时允许面板高度超过屏幕一半（默认 false）
isDismissible bool 点击遮罩是否关闭（默认 true）
enableDrag bool 是否允许下拉手势关闭（默认 true）
useSafeArea bool 是否避开系统安全区域（默认 false）
constraints BoxConstraints? 面板尺寸约束
backgroundColor Color? 背景色
elevation double? 阴影高度
showDragHandle bool? 是否显示默认拖拽手柄（Material 3）

注意事项：当 isScrollControlled 为 true 时，builder 返回的组件需要自己管理高度。如果内容不固定，务必使用 constraints 或 FractionallySizedBox 等方式限制最大高度，否则面板可能覆盖整个屏幕。

5. showBottomSheet

showBottomSheet 创建一个持久的、非模态的底部面板。它与 showModalBottomSheet 的核心区别在于：用户可以继续与页面其他部分交互，不会有遮罩层。

import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      theme: ThemeData(colorSchemeSeed: Colors.green, useMaterial3: true),
      home: const PersistentBottomSheetDemo(),
    );
  }
}

class PersistentBottomSheetDemo extends StatefulWidget {
  const PersistentBottomSheetDemo({super.key});

  @override
  State<PersistentBottomSheetDemo> createState() =>
      _PersistentBottomSheetDemoState();
}

class _PersistentBottomSheetDemoState extends State<PersistentBottomSheetDemo> {
  bool _isSheetOpen = false;

  void _toggleBottomSheet() {
    if (_isSheetOpen) return;

    setState(() => _isSheetOpen = true);

    // showBottomSheet 需要通过 Scaffold 的 context 调用
    // 返回 PersistentBottomSheetController，可用于编程式关闭
    final controller = showBottomSheet(
      context: context,
      shape: const RoundedRectangleBorder(
        borderRadius: BorderRadius.vertical(top: Radius.circular(20)),
      ),
      builder: (BuildContext sheetContext) {
        return Container(
          padding: const EdgeInsets.all(16),
          height: 200,
          child: Column(
            children: [
              Center(
                child: Container(
                  width: 40,
                  height: 4,
                  margin: const EdgeInsets.only(bottom: 16),
                  decoration: BoxDecoration(
                    color: Colors.grey[400],
                    borderRadius: BorderRadius.circular(2),
                  ),
                ),
              ),
              const Text(
                '持久底部面板',
                style: TextStyle(fontSize: 18, fontWeight: FontWeight.bold),
              ),
              const SizedBox(height: 8),
              const Text('这是一个非模态面板，你可以继续操作下方页面内容。'),
              const Spacer(),
              FilledButton(
                onPressed: () => Navigator.pop(sheetContext),
                child: const Text('关闭面板'),
              ),
            ],
          ),
        );
      },
    );

    // 监听面板关闭事件
    controller.closed.then((_) {
      if (mounted) {
        setState(() => _isSheetOpen = false);
      }
    });
  }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('showBottomSheet 演示')),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: [
            const Text('页面内容区域', style: TextStyle(fontSize: 18)),
            const SizedBox(height: 8),
            const Text('面板打开后你仍然可以操作这里'),
            const SizedBox(height: 24),
            ElevatedButton(
              onPressed: _isSheetOpen ? null : _toggleBottomSheet,
              child: Text(_isSheetOpen ? '面板已打开' : '打开持久底部面板'),
            ),
          ],
        ),
      ),
    );
  }
}

5.1 showBottomSheet 与 showModalBottomSheet 的对比

特性 showModalBottomSheet showBottomSheet
遮罩层 有（半透明黑色） 无
用户能否与页面交互 不能，必须先关闭 可以
显示方式 push 一个 ModalRoute 通过 Scaffold 显示
返回类型 Future<T?> PersistentBottomSheetController
适用场景 需要用户做出选择后才继续 辅助信息面板、播放器控制条等

注意事项：showBottomSheet 必须在 Scaffold 的子树中调用（即 context 能找到 Scaffold），否则会抛出异常。

6. DraggableScrollableSheet

DraggableScrollableSheet 是一个可拖动的底部面板，用户可以通过手势拖拽来调整面板高度。它通常与 showModalBottomSheet 组合使用。

6.1 独立使用

import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      theme: ThemeData(colorSchemeSeed: Colors.orange, useMaterial3: true),
      home: const DraggableSheetDemo(),
    );
  }
}

class DraggableSheetDemo extends StatelessWidget {
  const DraggableSheetDemo({super.key});

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('DraggableScrollableSheet 演示')),
      body: Stack(
        children: [
          // 背景内容
          Center(
            child: Column(
              mainAxisAlignment: MainAxisAlignment.center,
              children: [
                Icon(Icons.map, size: 100, color: Colors.grey[300]),
                const SizedBox(height: 16),
                const Text('地图内容区域', style: TextStyle(color: Colors.grey)),
              ],
            ),
          ),
          // 可拖动面板
          DraggableScrollableSheet(
            // 初始高度占屏幕的 30%
            initialChildSize: 0.3,
            // 最小高度占屏幕的 10%
            minChildSize: 0.1,
            // 最大高度占屏幕的 80%
            maxChildSize: 0.8,
            // 开启吸附效果
            snap: true,
            // 吸附位置（介于 min 和 max 之间）
            snapSizes: const [0.3, 0.5, 0.8],
            builder: (BuildContext context, ScrollController scrollController) {
              // 必须将 scrollController 传给内部的可滚动组件
              return Container(
                decoration: BoxDecoration(
                  color: Theme.of(context).colorScheme.surface,
                  borderRadius:

6.3 使用 DraggableScrollableController

DraggableScrollableController 可以编程式地控制面板位置，以及监听面板大小变化：

import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      theme: ThemeData(colorSchemeSeed: Colors.pink, useMaterial3: true),
      home: const ControllerSheetDemo(),
    );
  }
}

class ControllerSheetDemo extends StatefulWidget {
  const ControllerSheetDemo({super.key});

  @override
  State<ControllerSheetDemo> createState() => _ControllerSheetDemoState();
}

class _ControllerSheetDemoState extends State<ControllerSheetDemo> {
  final DraggableScrollableController _sheetController =
      DraggableScrollableController();
  double _currentSize = 0.3;

  @override
  void initState() {
    super.initState();
    // 监听面板大小变化
    _sheetController.addListener(_onSheetSizeChanged);
  }

  void _onSheetSizeChanged() {
    setState(() {
      _currentSize = _sheetController.size;
    });
  }

  @override
  void dispose() {
    _sheetController.removeListener(_onSheetSizeChanged);
    _sheetController.dispose();
    super.dispose();
  }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('DraggableScrollableController 演示')),
      body: Stack(
        children: [
          // 背景按钮区
          Padding(
            padding: const EdgeInsets.all(16),
            child: Column(
              children: [
                Text(
                  '当前面板高度: ${(_currentSize * 100).toStringAsFixed(1)}%',
                  style: const TextStyle(fontSize: 16),
                ),
                const SizedBox(height: 16),
                Row(
                  mainAxisAlignment: MainAxisAlignment.spaceEvenly,
                  children: [
                    ElevatedButton(
                      onPressed: () => _sheetController.animateTo(
                        0.1,
                        duration: const Duration(milliseconds: 300),
                        curve: Curves.easeInOut,
                      ),
                      child: const Text('收起'),
                    ),
                    ElevatedButton(
                      onPressed: () => _sheetController.animateTo(
                        0.5,
                        duration: const Duration(milliseconds: 300),
                        curve: Curves.easeInOut,
                      ),
                      child: const Text('半屏'),
                    ),
                    ElevatedButton(
                      onPressed: () => _sheetController.animateTo(
                        0.9,
                        duration: const Duration(milliseconds: 300),
                        curve: Curves.easeInOut,
                      ),
                      child: const Text('全屏'),
                    ),
                  ],
                ),
              ],
            ),
          ),
          // 可拖动面板
          DraggableScrollableSheet(
            controller: _sheetController,
            initialChildSize: 0.3,
            minChildSize: 0.1,
            maxChildSize: 0.9,
            snap: true,
            snapSizes: const [0.1, 0.3, 0.5, 0.9],
            builder: (context, scrollController) {
              return Container(
                decoration: BoxDecoration(
                  color: Theme.of(context).colorScheme.surfaceContainerHighest,
                  borderRadius:
                      const BorderRadius.vertical(top: Radius.circular(20)),
                  boxShadow: [
                    BoxShadow(
                      color: Colors.black.withOpacity(0.15),
                      blurRadius: 10,
                    ),
                  ],
                ),
                child: ListView.builder(
                  controller: scrollController,
                  itemCount: 30,
                  itemBuilder: (context, index) {
                    if (index == 0) {
                      return const Column(
                        children: [
                          SizedBox(height: 12),
                          Icon(Icons.drag_handle, color: Colors.grey),
                          Padding(
                            padding: EdgeInsets.all(16),
                            child: Text(
                              '可编程控制的面板',
                              style: TextStyle(
                                fontSize: 18,
                                fontWeight: FontWeight.bold,
                              ),
                            ),
                          ),
                        ],
                      );
                    }
                    return ListTile(
                      title: Text('列表项 $index'),
                    );
                  },
                ),
              );
            },
          ),
        ],
      ),
    );
  }
}

注意事项：DraggableScrollableSheet 的 builder 回调提供的 scrollController 必须传递给内部的可滚动组件（如 ListView、GridView），否则拖拽手势和滚动不会正常联动。

7. SnackBar 与 ScaffoldMessenger

SnackBar 用于在屏幕底部显示简短的提示消息。通过 ScaffoldMessenger.of(context).showSnackBar() 来显示。

7.1 基础用法与不同 behavior

import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {

class SnackBarDemo extends StatelessWidget {
  const SnackBarDemo({super.key});

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('SnackBar 演示')),
      floatingActionButton: FloatingActionButton(
        onPressed: () {},
        child: const Icon(Icons.add),
      ),
      body: SingleChildScrollView(
        padding: const EdgeInsets.all(16),

          crossAxisAlignment: CrossAxisAlignment.stretch,

                    behavior: SnackBarBehavior.floating,
                    shape: RoundedRectangleBorder(
                      borderRadius: BorderRadius.circular(10),
                    ),
                    margin: const EdgeInsets.all(12),
                    duration: const Duration(seconds: 2),
                  ),
                );
              },
              child: const Text('Floating SnackBar（悬浮）'),

                    action: SnackBarAction(

                          ),
                        );
                      },
                    ),
                    duration: const Duration(seconds: 4),
                  ),
                );
              },
              child: const Text('带撤销操作的 SnackBar'),
            ),
            const SizedBox(height: 12),

            // 4. 带关闭按钮
            ElevatedButton(
              onPressed: () {
                ScaffoldMessenger.of(context).showSnackBar(
                  SnackBar(
                    content: const Text('这条消息可以手动关闭'),
                    behavior: SnackBarBehavior.floating,
                    // 显示右侧关闭图标
                    showCloseIcon: true,
                    closeIconColor: Colors.white,
                    duration: const Duration(seconds: 10),
                  ),
                );
              },
              child: const Text('带关闭按钮的 SnackBar'),

            // 5. 自定义样式
            ElevatedButton(

                            crossAxisAlignment: CrossAxisAlignment.start,

7.2 SnackBar 常用属性

属性 类型 说明
content Widget 主体内容（通常是 Text）
action SnackBarAction? 右侧操作按钮
duration Duration 显示时长（默认 4 秒）
behavior SnackBarBehavior? fixed（贴底）或 floating（悬浮）
showCloseIcon bool 是否显示关闭按钮
backgroundColor Color? 背景色
shape ShapeBorder? 形状
elevation double? 阴影高度
margin EdgeInsetsGeometry? 外边距（仅 floating 模式有效）
padding EdgeInsetsGeometry? 内边距
width double? 固定宽度（仅 floating 模式有效）

最佳实践：

  • 连续显示多条 SnackBar 时，ScaffoldMessenger 会自动排队。如果只想显示最新的一条，先调用 clearSnackBars() 再 showSnackBar()。
  • 使用 floating 模式时，SnackBar 不会遮挡 FloatingActionButton，视觉效果更佳。
  • ScaffoldMessenger.of(context) 会向上查找最近的 ScaffoldMessenger（MaterialApp 会自动创建一个）。如果你在 Scaffold 的 build 方法内创建了一个按钮，该按钮的 context 可以正确找到对应的 ScaffoldMessenger。

            ),
            const SizedBox(height: 24),

            // 2. 在任意点击位置弹出菜单（右键菜单效果）
            GestureDetector(
              onTapDown: (details) => _showMenuAtPosition(details),
              child: Container(
                height: 200,
                decoration: BoxDecoration(
                  color: Colors.grey[200],
                  borderRadius: BorderRadius.circular(12),
                  border: Border.all(color: Colors.grey[400]!),
                ),
                child: const Center(
                  child: Text('在此区域点击弹出菜单', style: TextStyle(color: Colors.grey)),
                ),
              ),
            ),
          ],
        ),
      ),
    );
  }

  /// 在按钮位置弹出菜单
  Future<void> _showMenuAtButton(BuildContext buttonContext) async {
    // 获取按钮的渲染信息
    final RenderBox button = buttonContext.findRenderObject() as RenderBox;
    final RenderBox overlay =
        Navigator.of(context).overlay!.context.findRenderObject() as RenderBox;

    // 计算菜单弹出位置（按钮左下角）
    final RelativeRect position = RelativeRect.fromRect(
      Rect.fromPoints(
        button.localToGlobal(Offset.zero, ancestor: overlay),
        button.localToGlobal(
          button.size.bottomRight(Offset.zero),
          ancestor: overlay,
        ),
      ),
      Offset.zero & overlay.size,
    );

    // showMenu 返回 Future<T?>，T 由 PopupMenuItem 的 value 决定
    final String? result = await showMenu<String>(
      context: context,
      position: position,
      // 菜单项列表（PopupMenuEntry 的子类）
      items: [
        const PopupMenuItem(
          value: 'edit',
          child: ListTile(
            leading: Icon(Icons.edit),
            title: Text('编辑'),
            contentPadding: EdgeInsets.zero,
          ),
        ),
        const PopupMenuItem(
          value: 'share',
          child: ListTile(
            leading: Icon(Icons.share),
            title: Text('分享'),
            contentPadding: EdgeInsets.zero,
          ),
        ),
        const PopupMenuDivider(), // 分割线
        const PopupMenuItem(
          value: 'delete',
          child: ListTile(
            leading: Icon(Icons.delete, color: Colors.red),
            title: Text('删除', style: TextStyle(color: Colors.red)),
            contentPadding: EdgeInsets.zero,
          ),
        ),
      ],
      elevation: 8,
      shape: RoundedRectangleBorder(borderRadius: BorderRadius.circular(12)),
    );

    if (!mounted || result == null) return;
    setState(() => _lastAction = result);
  }

  /// 在点击位置弹出菜单
  Future<void> _showMenuAtPosition(TapDownDetails details) async {
    final RenderBox overlay =
        Navigator.of(context).overlay!.context.findRenderObject() as RenderBox;

    final RelativeRect position = RelativeRect.fromLTRB(
      details.globalPosition.dx,
      details.globalPosition.dy,
      overlay.size.width - details.globalPosition.dx,
      overlay.size.height - details.globalPosition.dy,
    );

    final String? result = await showMenu<String>(
      context: context,
      position: position,
      items: const [
        PopupMenuItem(value: 'copy', child: Text('复制')),
        PopupMenuItem(value: 'paste', child: Text('粘贴')),
        PopupMenuItem(value: 'select_all', child: Text('全选')),
      ],
    );

    if (!mounted || result == null) return;
    setState(() => _lastAction = result);
  }
}

类型 说明

最佳实践：简单场景可直接使用 PopupMenuButton 组件（第15课已介绍），它封装了按钮点击 + showMenu 的完整逻辑。当需要在非按钮位置弹出菜单（如长按、右键等自定义手势）时，才使用 showMenu。

异步返回值机制总结

所有弹出层方法的返回值对比：

方法 返回类型 如何传递结果
showDialog<T> Future<T?> Navigator.pop(context, result)
showModalBottomSheet<T> Future<T?> Navigator.pop(context, result)
showBottomSheet PersistentBottomSheetController 通过 controller.closed 监听关闭
showMenu<T> Future<T?> 用户点击 PopupMenuItem 时自动返回其 value
ScaffoldMessenger.showSnackBar ScaffoldFeatureController 通过 controller.closed 监听关闭原因
ScaffoldMessenger.showMaterialBanner ScaffoldFeatureController 通过 controller.closed 监听关闭原因

核心模式：

// 通用的弹出层异步使用模式
Future<void> handleDialogResult() async {
  // 1. await 弹出层方法
  final result = await showDialog<String>(
    context: context,
    builder: (ctx) => ...,
  );

  // 2. 检查 mounted（防止异步期间 Widget 被销毁）
  if (!mounted) return;

  // 3. 根据 result 处理逻辑（result 为 null 表示用户通过返回/点击外部关闭）
  if (result != null) {
    // 处理用户选择
  }
}

注意事项：

  • 当用户通过点击遮罩层、按返回键等方式关闭弹出层时，Future 的结果为 null。因此返回类型始终是 T?，需要做空值判断。
  • 在 await 之后、使用 context 之前，务必检查 mounted（StatefulWidget）或 context.mounted（Flutter 3.7+），防止操作已销毁的组件。

本课小结

本课系统讲解了 Flutter 中十种对话框与弹出层组件：

  1. showDialog + AlertDialog：最常用的警告/确认对话框，通过 Navigator.pop 返回用户选择结果。
  2. SimpleDialog：简单选项列表对话框，适合少量选项的选择场景。
  3. Dialog：对话框基类，用于完全自定义的对话框布局。
  4. showModalBottomSheet：模态底部弹出面板，isScrollControlled 控制最大高度。
  5. showBottomSheet：非模态底部面板，用户可继续操作页面。
  6. DraggableScrollableSheet：可拖动的底部面板，支持 snap 吸附和编程式控制。
  7. SnackBar：底部提示消息，fixed 贴底和 floating 悬浮两种样式。
  8. MaterialBanner：顶部横幅提示，不会自动消失，需手动关闭。
  9. Tooltip：长按/悬浮显示的简短提示。
  10. showMenu：在指定位置弹出菜单，需要计算 RelativeRect。

核心要点：

  • 大多数弹出层方法返回 Future<T?>，体现了 Flutter 中弹出层的异步交互模式
  • 使用 Navigator.pop(context, value) 传递返回值
  • 异步操作后必须检查 mounted 才能安全使用 context
  • ScaffoldMessenger 是 SnackBar 和 MaterialBanner 的统一管理入口
