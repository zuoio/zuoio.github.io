1. CustomPaint 与 CustomPainter

1.1 CustomPaint 组件

CustomPaint 是 Flutter 提供的自定义绘制组件，它本身不绘制任何内容，而是将绘制工作委托给 CustomPainter 对象。

核心属性：

属性 类型 说明
painter CustomPainter? 在 child 之前绘制（背景层）
foregroundPainter CustomPainter? 在 child 之后绘制（前景层）
size Size 当没有 child 时使用此尺寸，默认为 Size.zero
child Widget? 子组件，CustomPaint 的尺寸由 child 决定

尺寸决定规则：

  1. 如果有 child，CustomPaint 的尺寸由 child 决定
  2. 如果没有 child，则使用 size 属性
  3. 如果 size 也未指定（或为 Size.zero），则使用父约束的最小值
import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: const Text('CustomPaint 基础')),

            child: const Center(
              child: Text(

        ),

    );
  }
}

// 背景层画家 —— 绘制蓝色填充矩形
class MyBackgroundPainter extends CustomPainter {
  @override
  void paint(Canvas canvas, Size size) {
    final paint = Paint()..color = Colors.blue;
    canvas.drawRect(
      Rect.fromLTWH(0, 0, size.width, size.height),
      paint,
    );
  }

  @override
  bool shouldRepaint(covariant CustomPainter oldDelegate) => false;
}

// 前景层画家 —— 绘制红色对角线（会覆盖在 child 之上）
class MyForegroundPainter extends CustomPainter {
  @override
  void paint(Canvas canvas, Size size) {
    final paint = Paint()
      ..color = Colors.red
      ..strokeWidth = 3;
    canvas.drawLine(Offset.zero, Offset(size.width, size.height), paint);
    canvas.drawLine(Offset(size.width, 0), Offset(0, size.height), paint);
  }

  @override
  bool shouldRepaint(covariant CustomPainter oldDelegate) => false;
}

1.2 CustomPainter 抽象类

CustomPainter 是你自定义绘制逻辑的核心，必须实现两个方法：

class MyPainter extends CustomPainter {
  @override
  void paint(Canvas canvas, Size size) {
    // canvas：画布对象，提供各种 draw 方法
    // size：绘制区域的尺寸
  }

  @override
  bool shouldRepaint(covariant MyPainter oldDelegate) {
    // 返回 true 表示需要重绘，false 表示不需要
    // 框架在 CustomPaint 重建时调用此方法来决定是否需要重新执行 paint()
    return false;
  }
}

注意事项：paint() 方法中的 Size 参数就是 CustomPaint 组件最终确定的尺寸。你应该始终基于这个 size 来计算绘制坐标，而不是使用硬编码的数值，这样才能让绘制内容自适应不同尺寸。

1.3 使用 repaint 通知重绘

当你的绘制内容依赖于外部状态变化（如动画），可以传入一个 Listenable（通常是 AnimationController）来通知 CustomPaint 重绘，而不必依赖 setState：

import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: const RepaintNotifierDemo(),
    );
  }
}

class RepaintNotifierDemo extends StatefulWidget {
  const RepaintNotifierDemo({super.key});

  @override
  State<RepaintNotifierDemo> createState() => _RepaintNotifierDemoState();
}

class _RepaintNotifierDemoState extends State<RepaintNotifierDemo>
    with SingleTickerProviderStateMixin {
  late AnimationController _controller;

  @override
  void initState() {
    super.initState();
    _controller = AnimationController(
      vsync: this,
      duration: const Duration(seconds: 2),
    )..repeat(); // 循环播放
  }

  @override
  void dispose() {
    _controller.dispose();
    super.dispose();
  }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('Repaint 通知重绘')),
      body: Center(
        child: CustomPaint(
          size: const Size(200, 200),
          // 传入 _controller 作为 repaint 通知器
          // 每当 _controller 值变化时自动触发重绘，无需 setState
          painter: PulsePainter(animation: _controller),
        ),
      ),
    );
  }
}

class PulsePainter extends CustomPainter {
  final Animation<double> animation;

  PulsePainter({required this.animation}) : super(repaint: animation);

  @override
  void paint(Canvas canvas, Size size) {
    final center = Offset(size.width / 2, size.height / 2);
    final maxRadius = size.width / 2;
    final paint = Paint()
      ..color = Colors.blue.withOpacity(1 - animation.value)
      ..style = PaintingStyle.fill;
    canvas.drawCircle(center, maxRadius * animation.value, paint);
  }

  @override
  bool shouldRepaint(covariant PulsePainter oldDelegate) => false;
}

最佳实践：对于动画驱动的绘制，优先使用 super(repaint: animation) 方式，而不是在 shouldRepaint 中返回 true。前者更高效——只有当 Listenable 通知时才会重绘，而后者需要整个 Widget 重建才会触发检查。

2. Paint 对象

Paint 对象描述了"用什么样的笔来画"，它是 Canvas 绑制的核心配置。

2.1 Paint 常用属性总览

属性 类型 说明
color Color 画笔颜色
strokeWidth double 线条宽度（仅 stroke 模式有效）
style PaintingStyle fill（填充）或 stroke（描边）
strokeCap StrokeCap 线段端点样式：butt、round、square
strokeJoin StrokeJoin 线段连接处样式：miter、round、bevel
shader Shader? 着色器（渐变、图片着色等）
maskFilter MaskFilter? 模糊滤镜（实现阴影效果）
isAntiAlias bool 是否开启抗锯齿，默认 true

2.3 StrokeCap 与 StrokeJoin

import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: const Text('StrokeCap 与 StrokeJoin')),
        body: Center(
          child: CustomPaint(
            size: const Size(400, 350),
            painter: StrokeCapJoinPainter(),
          ),
        ),
      ),
    );
  }
}

class StrokeCapJoinPainter extends CustomPainter {
  @override
  void paint(Canvas canvas, Size size) {
    // === StrokeCap 演示 ===
    final caps = [StrokeCap.butt, StrokeCap.round, StrokeCap.square];
    final capNames = ['butt', 'round', 'square'];

    for (int i = 0; i < caps.length; i++) {
      final paint = Paint()
        ..color = Colors.blue
        ..strokeWidth = 16
        ..strokeCap = caps[i];
      final y = 40.0 + i * 50;
      canvas.drawLine(Offset(30, y), Offset(180, y), paint);

      // 标注端点样式名称
      final tp = TextPainter(
        text: TextSpan(
          text: capNames[i],
          style: const TextStyle(color: Colors.black, fontSize: 14),
        ),
        textDirection: TextDirection.ltr,
      )..layout();
      tp.paint(canvas, Offset(190, y - 8));
    }

    // === StrokeJoin 演示 ===
    final joins = [StrokeJoin.miter, StrokeJoin.round, StrokeJoin.bevel];
    final joinNames = ['miter', 'round', 'bevel'];

    for (int i = 0; i < joins.length; i++) {
      final paint = Paint()
        ..color = Colors.red
        ..strokeWidth = 10
        ..style = PaintingStyle.stroke
        ..strokeJoin = joins[i];

      final offsetX = 30.0 + i * 130;
      final path = Path()
        ..moveTo(offsetX, 220)
        ..lineTo(offsetX + 30, 280)
        ..lineTo(offsetX + 60, 210)
        ..lineTo(offsetX + 90, 280);
      canvas.drawPath(path, paint);

      final tp = TextPainter(
        text: TextSpan(
          text: joinNames[i],
          style: const TextStyle(color: Colors.black, fontSize: 12),
        ),
        textDirection: TextDirection.ltr,
      )..layout();
      tp.paint(canvas, Offset(offsetX + 20, 290));
    }
  }

  @override
  bool shouldRepaint(covariant CustomPainter oldDelegate) => false;
}

class ShaderPainter extends CustomPainter {
  @override
  void paint(Canvas canvas, Size size) {
    // 线性渐变
    final linearRect = const Rect.fromLTWH(20, 20, 300, 80);
    final linearPaint = Paint()
      ..shader = const LinearGradient(
        colors: [Colors.red, Colors.yellow, Colors.green],
      ).createShader(linearRect);
    canvas.drawRect(linearRect, linearPaint);

    // 径向渐变
    final radialRect = const Rect.fromLTWH(20, 130, 300, 100);
    final radialPaint = Paint()
      ..shader = RadialGradient(
        colors: [Colors.purple, Colors.blue, Colors.cyan.withOpacity(0)],
      ).createShader(radialRect);
    canvas.drawOval(radialRect, radialPaint);

    // 扫描渐变
    final sweepRect = const Rect.fromLTWH(100, 270, 150, 120);
    final sweepPaint = Paint()
      ..shader = const SweepGradient(
        colors: [Colors.red, Colors.orange, Colors.yellow, Colors.green,
                 Colors.blue, Colors.purple, Colors.red],
      ).createShader(sweepRect);
    canvas.drawOval(sweepRect, sweepPaint);
  }

  @override
  bool shouldRepaint(covariant CustomPainter oldDelegate) => false;
}

2.5 MaskFilter：模糊阴影

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {

  @override
  Widget build(BuildContext context) {

class MaskFilterPainter extends CustomPainter {
  @override
  void paint(Canvas canvas, Size size) {
    // 带模糊阴影的圆
    final shadowPaint = Paint()
      ..color = Colors.black38
      ..maskFilter = const MaskFilter.blur(BlurStyle.normal, 8);
    canvas.drawCircle(const Offset(155, 105), 60, shadowPaint);

    // 正常绘制的圆（阴影之上）
    final circlePaint = Paint()
      ..color = Colors.blue
      ..style = PaintingStyle.fill;
    canvas.drawCircle(const Offset(150, 100), 60, circlePaint);
  }

  @override
  bool shouldRepaint(covariant CustomPainter oldDelegate) => false;
}

MaskFilter.blur 的两个参数：

  • BlurStyle：模糊样式 —— normal（内外都模糊）、solid（仅外部模糊）、outer（只画外部模糊，不画形状本身）、inner（只在内部模糊）
  • sigma：模糊程度，数值越大越模糊

import 'dart:math';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {

  @override
  Widget build(BuildContext context) {

    );
  }

class ArcPainter extends CustomPainter {
  @override
  void paint(Canvas canvas, Size size) {
    // === 弧线（useCenter = false, stroke 模式）===
    final arcPaint = Paint()
      ..color = Colors.blue
      ..style = PaintingStyle.stroke
      ..strokeWidth = 4;

    canvas.drawArc(
      const Rect.fromLTWH(30, 30, 150, 150), // 外接矩形
      0,            // 起始角度（弧度制，0 = 3点钟方向）
      pi * 1.5,     // 扫过角度（3/4 圆弧）
      false,        // useCenter: false = 弧线
      arcPaint,
    );

    // === 扇形（useCenter = true, fill 模式）===
    final sectorPaint = Paint()
      ..color = Colors.red.withOpacity(0.7)
      ..style = PaintingStyle.fill;

    canvas.drawArc(
      const Rect.fromLTWH(220, 30, 150, 150),
      -pi / 2,     // 从12点钟方向开始
      pi / 3 * 2,  // 扫过 120 度
      true,         // useCenter: true = 扇形（连接到中心点）
      sectorPaint,
    );

    // === 多个扇形拼成饼图 ===
    final rect = const Rect.fromLTWH(120, 200, 150, 150);
    final pieColors = [Colors.blue, Colors.green, Colors.orange, Colors.purple];
    final sweeps = [pi / 2, pi / 3, pi / 4, pi - pi / 3 - pi / 4]; // 总和接近 2*pi
    double startAngle = -pi / 2;

    for (int i = 0; i < pieColors.length; i++) {
      final paint = Paint()..color = pieColors[i];
      canvas.drawArc(rect, startAngle, sweeps[i], true, paint);
      startAngle += sweeps[i];
    }
  }

  @override
  bool shouldRepaint(covariant CustomPainter oldDelegate) => false;
}

注意事项：Canvas 中所有角度参数都使用弧度制（radian），而不是角度制（degree）。pi 表示 180°，pi / 2 表示 90°。角度转弧度公式：radian = degree * pi / 180。

4. Canvas 变换与裁剪

Canvas 提供了一组变换和裁剪操作，让你可以对画布进行平移、旋转、缩放等操作。这些操作通常配合 save() 和 restore() 使用。

4.1 save 与 restore

save() 保存当前画布状态（变换矩阵、裁剪区域），restore() 恢复到上一次 save() 的状态。它们总是成对使用。

class TransformPainter extends CustomPainter {
  @override
  void paint(Canvas canvas, Size size) {
    final paint = Paint()
      ..color = Colors.blue
      ..style = PaintingStyle.stroke
      ..strokeWidth = 2;

    const rect = Rect.fromLTWH(0, 0, 60, 40);

    // --- 原始矩形 ---
    canvas.save();
    canvas.translate(50, 50); // 将原点平移到 (50, 50)
    canvas.drawRect(rect, paint);
    canvas.restore();

    // --- 旋转 ---
    canvas.save();
    canvas.translate(200, 70); // 先移到旋转中心
    canvas.rotate(pi / 6);     // 旋转 30 度
    paint.color = Colors.red;
    canvas.drawRect(rect, paint);
    canvas.restore();

    // --- 缩放 ---
    canvas.save();
    canvas.translate(50, 180);
    canvas.scale(2.0, 1.5); // X 方向放大 2 倍，Y 方向放大 1.5 倍
    paint.color = Colors.green;
    canvas.drawRect(rect, paint);
    canvas.restore();

    // --- 组合变换：先平移再旋转 ---
    canvas.save();
    canvas.translate(250, 250);
    for (int i = 0; i < 12; i++) {
      canvas.rotate(pi / 6); // 每次旋转 30 度
      paint.color = Colors.primaries[i % Colors.primaries.length];
      canvas.drawLine(const Offset(0, 0), const Offset(80, 0), paint);
    }
    canvas.restore();
  }

  @override
  bool shouldRepaint(covariant CustomPainter oldDelegate) => false;
}

最佳实践：save() 和 restore() 必须严格配对。养成习惯——每次做变换操作前先 save()，操作完成后立即 restore()。未匹配的 save/restore 会导致后续绘制的坐标系错乱。

class ClipPainter extends CustomPainter {
  @override
  void paint(Canvas canvas, Size size) {
    // === clipRect：矩形裁剪 ===
    canvas.save();
    canvas.clipRect(const Rect.fromLTWH(20, 20, 140, 140));
    // 以下绘制只会在裁剪区域内显示
    final gradientPaint = Paint()
      ..shader = const LinearGradient(
        colors: [Colors.red, Colors.blue],
      ).createShader(const Rect.fromLTWH(0, 0, 200, 200));
    canvas.drawCircle(const Offset(90, 90), 100, gradientPaint);
    canvas.restore();

    // === clipPath：自定义路径裁剪 ===
    canvas.save();
    final starPath = _createStarPath(
      center: const Offset(270, 90),
      outerRadius: 70,
      innerRadius: 35,
    );
    canvas.clipPath(starPath);
    // 裁剪区域内绘制渐变
    final starPaint = Paint()
      ..shader = const LinearGradient(
        colors: [Colors.orange, Colors.purple],
        begin: Alignment.topLeft,
        end: Alignment.bottomRight,
      ).createShader(const Rect.fromLTWH(200, 20, 140, 140));
    canvas.drawRect(const Rect.fromLTWH(200, 20, 140, 140), starPaint);
    canvas.restore();

    // === 圆形裁剪 ===
    canvas.save();
    final circlePath = Path()
      ..addOval(const Rect.fromLTWH(100, 200, 140, 140));
    canvas.clipPath(circlePath);
    final checkerPaint = Paint();
    for (int row = 0; row < 10; row++) {
      for (int col = 0; col < 10; col++) {
        checkerPaint.color =
            (row + col) % 2 == 0 ? Colors.black : Colors.white;
        canvas.drawRect(
          Rect.fromLTWH(100.0 + col * 14, 200.0 + row * 14, 14, 14),
          checkerPaint,
        );
      }
    }
    canvas.restore();
  }

  Path _createStarPath({
    required Offset center,
    required double outerRadius,
    required double innerRadius,
    int points = 5,
  }) {
    final path = Path();
    final angle = 3.14159265 / points;
    for (int i = 0; i < 2 * points; i++) {
      final r = i.isEven ? outerRadius : innerRadius;
      final a = i * angle - 3.14159265 / 2;
      final x = center.dx + r * _cos(a);
      final y = center.dy + r * _sin(a);
      if (i == 0) {
        path.moveTo(x, y);
      } else {
        path.lineTo(x, y);
      }
    }
    path.close();
    return path;
  }

  double _cos(double radians) => radians == 0 ? 1 : _cosValue(radians);
  double _cosValue(double r) {
    return (r >= 0 ? 1 : -1) * _rawCos(r.abs());
  }

  double _rawCos(double r) {
    // 使用 dart:math
    return r.isNaN ? 0 : _mathCos(r);
  }

  double _mathCos(double r) => r.isNaN ? 0 : _dartMathCos(r);
  double _dartMathCos(double r) {
    // 简化处理，实际应导入 dart:math
    final x = r % (2 * 3.14159265);
    double sum = 1.0;
    double term = 1.0;
    for (int i = 1; i <= 10; i++) {
      term *= -x * x / ((2 * i - 1) * (2 * i));
      sum += term;
    }
    return sum;
  }

  double _sin(double r) {
    final x = r % (2 * 3.14159265);
    double sum = x;
    double term = x;
    for (int i = 1; i <= 10; i++) {
      term *= -x * x / ((2 * i) * (2 * i + 1));
      sum += term;
    }
    return sum;
  }

  @override
  bool shouldRepaint(covariant CustomPainter oldDelegate) => false;
}

上面的星形裁剪示例使用了手动三角函数实现。实际开发中应当导入 dart:math 使用标准的 cos() 和 sin() 函数，下面是更简洁的写法：

import 'dart:math';

Path createStarPath({
  required Offset center,
  required double outerRadius,
  required double innerRadius,
  int points = 5,
}) {
  final path = Path();
  final angle = pi / points;
  for (int i = 0; i < 2 * points; i++) {
    final r = i.isEven ? outerRadius : innerRadius;
    final a = i * angle - pi / 2; // 从顶部开始
    final x = center.dx + r * cos(a);
    final y = center.dy + r * sin(a);
    if (i == 0) {
      path.moveTo(x, y);
    } else {
      path.lineTo(x, y);
    }
  }
  path.close();
  return path;
}

5.2 quadraticBezierTo —— 二次贝塞尔曲线

二次贝塞尔曲线由三个点定义：起始点（当前路径点）、控制点和终点。曲线从起始点出发，"被控制点吸引"，最终到达终点。

起始点 ----→ 控制点（控制曲线的弯曲方向和程度）
    ↘             ↗
      曲线 ~~~~~~
            ↘
           终点

控制点不在曲线上，但决定了曲线的弯曲方向。可以想象控制点像一块磁铁，把曲线往它的方向"拉"。

import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: const Text('二次贝塞尔曲线')),
        body: Center(
          child: CustomPaint(
            size: const Size(350, 350),
            painter: QuadBezierPainter(),
          ),
        ),
      ),
    );
  }
}

class QuadBezierPainter extends CustomPainter {
  @override
  void paint(Canvas canvas, Size size) {
    // 定义三个关键点
    const start = Offset(30, 200);     // 起始点
    const control = Offset(175, 30);   // 控制点
    const end = Offset(320, 200);      // 终点

    // 绘制贝塞尔曲线
    final curvePath = Path()
      ..moveTo(start.dx, start.dy)
      ..quadraticBezierTo(control.dx, control.dy, end.dx, end.dy);

    final curvePaint = Paint()
      ..color = Colors.blue
      ..style = PaintingStyle.stroke
      ..strokeWidth = 3;
    canvas.drawPath(curvePath, curvePaint);

    // 绘制辅助线（起始点→控制点→终点）
    final guidePaint = Paint()
      ..color = Colors.grey
      ..style = PaintingStyle.stroke
      ..strokeWidth = 1;
    canvas.drawLine(start, control, guidePaint);
    canvas.drawLine(control, end, guidePaint);

    // 绘制关键点
    final pointPaint = Paint()..style = PaintingStyle.fill;

    pointPaint.color = Colors.green;
    canvas.drawCircle(start, 6, pointPaint);   // 起始点（绿色）

    pointPaint.color = Colors.red;
    canvas.drawCircle(control, 6, pointPaint); // 控制点（红色）

    pointPaint.color = Colors.green;
    canvas.drawCircle(end, 6, pointPaint);     // 终点（绿色）

    // 标注文字
    _drawLabel(canvas, '起始点', start.translate(-10, 10));
    _drawLabel(canvas, '控制点', control.translate(-10, -20));
    _drawLabel(canvas, '终点', end.translate(-10, 10));

    // === 演示不同控制点位置的效果 ===
    final controls2 = [
      const Offset(175, 250), // 控制点在下方
      const Offset(30, 30),   // 控制点靠近起始点上方
      const Offset(320, 30),  // 控制点靠近终点上方
    ];
    final colors = [Colors.orange, Colors.purple, Colors.teal];

    for (int i = 0; i < controls2.length; i++) {
      final p = Path()
        ..moveTo(30, 300)
        ..quadraticBezierTo(controls2[i].dx, controls2[i].dy + 100, 320, 300);
      canvas.drawPath(
        p,
        Paint()
          ..color = colors[i]
          ..style = PaintingStyle.stroke
          ..strokeWidth = 2,
      );
    }
  }

  void _drawLabel(Canvas canvas, String text, Offset offset) {
    final tp = TextPainter(
      text: TextSpan(
        text: text,
        style: const TextStyle(color: Colors.black, fontSize: 12),
      ),
      textDirection: TextDirection.ltr,
    )..layout();
    tp.paint(canvas, offset);
  }

  @override
  bool shouldRepaint(covariant CustomPainter oldDelegate) => false;
}

    ↘
      曲线前半段 ~~~
                   ~~~曲线后半段
                     ↗
           控制点2（控制曲线后半段的弯曲）← 终点

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {

  @override

class ArcToPathPainter extends CustomPainter {
  @override
  void paint(Canvas canvas, Size size) {
    final paint = Paint()
      ..color = Colors.blue
      ..style = PaintingStyle.stroke
      ..strokeWidth = 3;

    // arcTo：在路径中添加一段弧线
    // 参数：外接矩形、起始角度、扫过角度、是否 moveTo 弧线起点
    final path1 = Path()
      ..moveTo(30, 100)
      ..lineTo(80, 100)
      ..arcTo(
        const Rect.fromLTWH(80, 50, 100, 100), // 外接矩形
        0,        // 起始角度（3 点钟方向）
        pi,       // 扫过 180 度
        false,    // false = 从当前点连线到弧线起点
      )
      ..lineTo(230, 100);
    canvas.drawPath(path1, paint);

    // arcToPoint：画弧线到指定终点
    // 使用椭圆参数（radius）和旋转角度来定义弧线
    final path2 = Path()
      ..moveTo(30, 250)
      ..arcToPoint(
        const Offset(200, 250),        // 终点
        radius: const Radius.circular(60), // 弧线半径
        clockwise: true,                   // 顺时针
      )
      ..arcToPoint(
        const Offset(320, 250),
        radius: const Radius.circular(60),
        clockwise: false,                  // 逆时针
      );

    paint.color = Colors.red;
    canvas.drawPath(path2, paint);
  }

  @override
  bool shouldRepaint(covariant CustomPainter oldDelegate) => false;
}

    // addRect
    final path = Path();
    paint.color = Colors.blue;
    path.addRect(const Rect.fromLTWH(20, 20, 120, 70));

    // addOval
    final ovalPath = Path()
      ..addOval(const Rect.fromLTWH(180, 20, 140, 70));
    paint.color = Colors.red;
    canvas.drawPath(ovalPath, paint);

    // addRRect
    final rrectPath = Path()
      ..addRRect(RRect.fromRectAndRadius(
        const Rect.fromLTWH(20, 120, 120, 70),
        const Radius.circular(15),
      ));
    paint.color = Colors.green;
    canvas.drawPath(rrectPath, paint);

    // addPolygon：添加多边形
    final hexagonPoints = <Offset>[];
    for (int i = 0; i < 6; i++) {
      final angle = (i * 60 - 90) * 3.14159265 / 180;
      hexagonPoints.add(Offset(
        250 + 40 * _cos(angle),
        155 + 40 * _sin(angle),
      ));
    }
    final polygonPath = Path()
      ..addPolygon(hexagonPoints, true); // true 表示闭合
    paint.color = Colors.purple;
    canvas.drawPath(polygonPath, paint);

    // 组合：一个 Path 中可以添加多个子图形
    final combinedPath = Path()
      ..addRect(const Rect.fromLTWH(20, 230, 80, 80))
      ..addOval(const Rect.fromLTWH(130, 230, 80, 80))
      ..addRRect(RRect.fromRectAndRadius(
        const Rect.fromLTWH(240, 230, 80, 80),
        const Radius.circular(10),
      ));
    paint.color = Colors.orange;
    paint.style = PaintingStyle.fill;
    canvas.drawPath(combinedPath, paint);
  }

  double _cos(double r) {
    double sum = 1.0, term = 1.0;
    for (int i = 1; i <= 10; i++) {
      term *= -r * r / ((2 * i - 1) * (2 * i));
      sum += term;
    }
    return sum;
  }

  double _sin(double r) {
    double sum = r, term = r;
    for (int i = 1; i <= 10; i++) {
      term *= -r * r / ((2 * i) * (2 * i + 1));
      sum += term;
    }
    return sum;
  }

  bool shouldRepaint(covariant CustomPainter oldDelegate) => false;
}

class PathCombinePainter extends CustomPainter {
  @override
  void paint(Canvas canvas, Size size) {
    final operations = <MapEntry<PathOperation, String>>[
      const MapEntry(PathOperation.union, 'union\n(并集)'),
      const MapEntry(PathOperation.intersect, 'intersect\n(交集)'),
      const MapEntry(PathOperation.difference, 'difference\n(差集 A-B)'),
      const MapEntry(PathOperation.reverseDifference, 'reverseDifference\n(差集 B-A)'),
      const MapEntry(PathOperation.xor, 'xor\n(异或)'),
    ];

    for (int i = 0; i < operations.length; i++) {
      final offsetX = (i % 2) * 190.0 + 20;
      final offsetY = (i ~/ 2) * 160.0 + 20;

      // 两个相交的圆形路径
      final pathA = Path()
        ..addOval(Rect.fromCircle(
          center: Offset(offsetX + 50, offsetY + 55),
          radius: 40,
        ));
      final pathB = Path()
        ..addOval(Rect.fromCircle(
          center: Offset(offsetX + 90, offsetY + 55),
          radius: 40,
        ));

      // 路径布尔运算
      final combined = Path.combine(
        operations[i].key,
        pathA,
        pathB,
      );

      final paint = Paint()
        ..color = Colors.blue.withOpacity(0.6)
        ..style = PaintingStyle.fill;
      canvas.drawPath(combined, paint);

      // 绘制原始圆的轮廓
      final outlinePaint = Paint()
        ..color = Colors.grey.withOpacity(0.3)
        ..style = PaintingStyle.stroke
        ..strokeWidth = 1;
      canvas.drawPath(pathA, outlinePaint);
      canvas.drawPath(pathB, outlinePaint);

      // 标签
      final tp = TextPainter(
        text: TextSpan(
          text: operations[i].value,
          style: const TextStyle(color: Colors.black87, fontSize: 11),
        ),
        textDirection: TextDirection.ltr,
        textAlign: TextAlign.center,
      )..layout(maxWidth: 160);
      tp.paint(canvas, Offset(offsetX + 10, offsetY + 100));
    }
  }

  @override
  bool shouldRepaint(covariant CustomPainter oldDelegate) => false;
}

PathOperation 的五种模式：

操作 说明
union 并集：合并两个路径覆盖的区域
intersect 交集：只保留两个路径重叠的区域
difference 差集：路径 A 减去与 B 重叠的部分
reverseDifference 反向差集：路径 B 减去与 A 重叠的部分
xor 异或：保留两个路径不重叠的部分

7. shouldRepaint 优化策略

shouldRepaint 是性能优化的关键。当 CustomPaint 组件重建时，Flutter 会调用此方法来决定是否需要重新执行 paint()。

7.1 何时返回 false

当绘制内容完全不依赖外部参数（静态图形）时，返回 false：

class StaticGridPainter extends CustomPainter {
  @override
  void paint(Canvas canvas, Size size) {
    // 绘制固定的网格背景
    final paint = Paint()
      ..color = Colors.grey.withOpacity(0.3)
      ..strokeWidth = 1;
    for (double x = 0; x <= size.width; x += 20) {
      canvas.drawLine(Offset(x, 0), Offset(x, size.height), paint);
    }
    for (double y = 0; y <= size.height; y += 20) {
      canvas.drawLine(Offset(0, y), Offset(size.width, y), paint);
    }
  }

  @override
  bool shouldRepaint(covariant StaticGridPainter oldDelegate) {
    return false; // 内容永远不变，无需重绘
  }
}

7.2 何时返回 true

当绘制内容依赖于外部参数，且参数发生了变化时，返回 true：

class DataChartPainter extends CustomPainter {
  final List<double> data;
  final Color color;

  DataChartPainter({required this.data, required this.color});

  void paint(Canvas canvas, Size size) {

7.3 优化策略总结

场景 shouldRepaint 策略
静态图形（无参数依赖） 始终返回 false
依赖少量参数 比较新旧参数，变化时返回 true
动画驱动 使用 super(repaint: animation)，shouldRepaint 返回 false
复杂对象（如大列表） 使用 listEquals 等深度比较，避免引用比较的误判 
import 'package:flutter/foundation.dart';

class OptimizedPainter extends CustomPainter {
  final List<double> data;
  final Color color;
  final double progress;

  OptimizedPainter({
    required this.data,
    required this.color,
    required this.progress,
  });

  @override
  void paint(Canvas canvas, Size size) {
    // 绘制逻辑...
  }

  @override
  bool shouldRepaint(covariant OptimizedPainter oldDelegate) {
    // 逐个字段精确比较
    return oldDelegate.color != color ||
        oldDelegate.progress != progress ||
        !listEquals(oldDelegate.data, data); // 列表深度比较
  }
}

最佳实践：

  1. 永远不要无脑返回 true——这会导致每次父组件 build 时都重绘
  2. 动画场景使用 repaint 参数替代 setState + shouldRepaint: true
  3. 对于复杂的不可变数据，考虑使用 identical() 做引用比较来加速判断
  4. 可以用 RepaintBoundary 包裹 CustomPaint，隔离重绘区域，避免影响其他组件

class PieChartPainter extends CustomPainter {
  final List<PieChartData> data;
  final Animation<double> animation;

  PieChartPainter({
    required this.data,
    required this.animation,
  }) : super(repaint: animation);

  @override
  void paint(Canvas canvas, Size size) {
    final center = Offset(size.width / 2, size.height / 2);
    final radius = min(size.width, size.height) / 2 - 20;
    final total = data.fold<double>(0, (sum, item) => sum + item.value);

    double startAngle = -pi / 2; // 从12点方向开始
    final animatedSweep = 2 * pi * animation.value; // 动画控制总扫过角度

    for (int i = 0; i < data.length; i++) {
      final sweepAngle = (data[i].value / total) * animatedSweep;

      // 绘制扇形
      final paint = Paint()
        ..color = data[i].color
        ..style = PaintingStyle.fill;
      canvas.drawArc(
        Rect.fromCircle(center: center, radius: radius),
        startAngle,
        sweepAngle,
        true,
        paint,
      );

      // 绘制扇形边框
      final borderPaint = Paint()
        ..color = Colors.white
        ..style = PaintingStyle.stroke
        ..strokeWidth = 2;
      canvas.drawArc(
        Rect.fromCircle(center: center, radius: radius),
        startAngle,
        sweepAngle,
        true,
        borderPaint,
      );

      // 当动画完成时绘制标签
      if (animation.value > 0.95) {
        _drawLabel(
          canvas,
          center,
          radius,
          startAngle,
          sweepAngle,
          data[i],
          total,
        );
      }

      startAngle += sweepAngle;
    }
  }

  void _drawLabel(
    Canvas canvas,
    Offset center,
    double radius,
    double startAngle,
    double sweepAngle,
    PieChartData item,
    double total,
  ) {
    // 标签放在扇形中间角度的外侧
    final midAngle = startAngle + sweepAngle / 2;
    final labelRadius = radius * 0.65;
    final labelPos = Offset(
      center.dx + labelRadius * cos(midAngle),
      center.dy + labelRadius * sin(midAngle),
    );

    final percent = (item.value / total * 100).toStringAsFixed(0);
    final tp = TextPainter(
      text: TextSpan(
        text: '$percent%',
        style: const TextStyle(
          color: Colors.white,
          fontSize: 14,
          fontWeight: FontWeight.bold,
        ),
      ),
      textDirection: TextDirection.ltr,
    )..layout();

    tp.paint(
      canvas,
      Offset(labelPos.dx - tp.width / 2, labelPos.dy - tp.height / 2),
    );
  }

  @override
  bool shouldRepaint(covariant PieChartPainter oldDelegate) {
    return oldDelegate.data != data;
  }
}

饼图实现要点：

  1. 使用 drawArc 的 useCenter: true 绘制扇形
  2. 起始角度从 -pi/2（12 点方向）开始，更符合直觉
  3. 动画通过 animation.value 控制总扫过角度，实现从 0 展开到完整饼图的效果
  4. 标签位于扇形的中间角度方向，距圆心 65% 半径处
  5. 白色边框让各扇形的边界更清晰

    required this.gridColor,
    required this.axisColor,
  }) : super(repaint: animation);

  @override
  void paint(Canvas canvas, Size size) {
    if (data.isEmpty) return;

    final chartWidth = size.width - paddingLeft - paddingRight;
    final chartHeight = size.height - paddingTop - paddingBottom;
    final chartRect = Rect.fromLTWH(
      paddingLeft, paddingTop, chartWidth, chartHeight,
    );

    final maxValue = data.map((p) => p.value).reduce(max);
    // 向上取整到最近的 100 的倍数
    final yMax = ((maxValue / 100).ceil() * 100).toDouble();

    _drawGrid(canvas, chartRect, yMax);
    _drawAxes(canvas, chartRect);
    _drawYLabels(canvas, chartRect, yMax);
    _drawXLabels(canvas, chartRect);
    _drawLineAndFill(canvas, chartRect, yMax);
    _drawDataPoints(canvas, chartRect, yMax);
  }

  // 绘制网格线
  void _drawGrid(Canvas canvas, Rect chartRect, double yMax) {
    final paint = Paint()
      ..color = gridColor
      ..strokeWidth = 1;

    const yDivisions = 5;
    for (int i = 0; i <= yDivisions; i++) {
      final y = chartRect.bottom - (i / yDivisions) * chartRect.height;
      canvas.drawLine(
        Offset(chartRect.left, y),
        Offset(chartRect.right, y),
        paint,
      );
    }
  }

  // 绘制坐标轴
  void _drawAxes(Canvas canvas, Rect chartRect) {
    final paint = Paint()
      ..color = axisColor
      ..strokeWidth = 1.5;

    // X 轴
    canvas.drawLine(
      Offset(chartRect.left, chartRect.bottom),
      Offset(chartRect.right, chartRect.bottom),
      paint,
    );
    // Y 轴
    canvas.drawLine(
      Offset(chartRect.left, chartRect.top),
      Offset(chartRect.left, chartRect.bottom),
      paint,
    );
  }

  // 绘制 Y 轴刻度标签
  void _drawYLabels(Canvas canvas, Rect chartRect, double yMax) {
    const yDivisions = 5;
    for (int i = 0; i <= yDivisions; i++) {
      final value = (yMax / yDivisions * i).toInt();
      final y = chartRect.bottom - (i / yDivisions) * chartRect.height;

      final tp = TextPainter(
        text: TextSpan(
          text: '$value',
          style: TextStyle(color: axisColor, fontSize: 11),
        ),
        textDirection: TextDirection.ltr,
      )..layout();
      tp.paint(canvas, Offset(chartRect.left - tp.width - 8, y - tp.height / 2));
    }
  }

  // 绘制 X 轴标签
  void _drawXLabels(Canvas canvas, Rect chartRect) {
    final xStep = chartRect.width / (data.length - 1);
    for (int i = 0; i < data.length; i++) {
      final x = chartRect.left + i * xStep;
      final tp = TextPainter(
        text: TextSpan(
          text: data[i].label,
          style: TextStyle(color: axisColor, fontSize: 10),
        ),
        textDirection: TextDirection.ltr,
      )..layout();
      tp.paint(canvas, Offset(x - tp.width / 2, chartRect.bottom + 8));
    }
  }

  // 绘制折线和渐变填充区域
  void _drawLineAndFill(Canvas canvas, Rect chartRect, double yMax) {
    if (data.length < 2) return;

    final xStep = chartRect.width / (data.length - 1);
    final animProgress = animation.value;

    // 计算所有数据点坐标
    final points = <Offset>[];
    for (int i = 0; i < data.length; i++) {
      final x = chartRect.left + i * xStep;
      final normalizedY = (data[i].value / yMax) * chartRect.height;
      final y = chartRect.bottom - normalizedY * animProgress; // 动画插值
      points.add(Offset(x, y));
    }

    // 构建折线路径
    final linePath = Path()..moveTo(points[0].dx, points[0].dy);
    for (int i = 1; i < points.length; i++) {
      linePath.lineTo(points[i].dx, points[i].dy);
    }

    // 构建填充路径（折线 + 底部闭合区域）
    final fillPath = Path.from(linePath)
      ..lineTo(points.last.dx, chartRect.bottom)
      ..lineTo(points.first.dx, chartRect.bottom)
      ..close();

    // 绘制渐变填充
    final fillPaint = Paint()
      ..shader = LinearGradient(
        begin: Alignment.topCenter,
        end: Alignment.bottomCenter,
        colors: [
          lineColor.withOpacity(0.3 * animProgress),
          lineColor.withOpacity(0.02),
        ],
      ).createShader(chartRect);
    canvas.drawPath(fillPath, fillPaint);

    // 绘制折线
    final linePaint = Paint()
      ..color = lineColor
      ..style = PaintingStyle.stroke
      ..strokeWidth = 2.5
      ..strokeCap = StrokeCap.round
      ..strokeJoin = StrokeJoin.round;
    canvas.drawPath(linePath, linePaint);
  }

  // 绘制数据点
  void _drawDataPoints(Canvas canvas, Rect chartRect, double yMax) {
    final xStep = chartRect.width / (data.length - 1);
    final animProgress = animation.value;

    if (animProgress < 0.3) return; // 动画前期不显示数据点
    final pointOpacity = ((animProgress - 0.3) / 0.7).clamp(0.0, 1.0);

    for (int i = 0; i < data.length; i++) {
      final x = chartRect.left + i * xStep;
      final normalizedY = (data[i].value / yMax) * chartRect.height;
      final y = chartRect.bottom - normalizedY * animProgress;
      final pos = Offset(x, y);

      // 白色外圈
      canvas.drawCircle(
        pos,
        5,
        Paint()..color = Colors.white.withOpacity(pointOpacity),
      );
      // 彩色内圈
      canvas.drawCircle(
        pos,
        4,
        Paint()..color = lineColor.withOpacity(pointOpacity),
      );

      // 数据值标签（动画完成后显示）
      if (animProgress > 0.9) {
        final labelOpacity = ((animProgress - 0.9) / 0.1).clamp(0.0, 1.0);
        final tp = TextPainter(
          text: TextSpan(
            text: data[i].value.toInt().toString(),
            style: TextStyle(
              color: lineColor.withOpacity(labelOpacity),
              fontSize: 10,
              fontWeight: FontWeight.w600,
            ),
          ),
          textDirection: TextDirection.ltr,
        )..layout();
        tp.paint(canvas, Offset(x - tp.width / 2, y - tp.height - 8));
      }
    }
  }

  @override
  bool shouldRepaint(covariant LineChartPainter oldDelegate) {
    return oldDelegate.data != data ||
        oldDelegate.lineColor != lineColor ||
        oldDelegate.fillColor != fillColor;
  }
}

折线图实现要点：

  1. 坐标系映射：将数据值映射到画布坐标，Y 轴需要翻转（Canvas 向下为正，数据向上为正）
  2. 边距管理：左侧预留 Y 轴标签空间，底部预留 X 轴标签空间
  3. 动画效果：通过 animation.value 控制 Y 轴位置的插值，实现数据从底部"生长"的效果
  4. 渐变填充：使用 LinearGradient 创建从上到下的渐变，填充折线下方区域
  5. 分层绘制顺序：网格 → 坐标轴 → 填充 → 折线 → 数据点 → 标签，后绘制的在上层

本课小结

本课系统讲解了 Flutter 自定义绘制的完整知识体系：

  1. CustomPaint 与 CustomPainter：CustomPaint 组件通过 painter（背景）和 foregroundPainter（前景）委托绘制，CustomPainter 的 paint(Canvas, Size) 方法是核心绑制入口。尺寸由 child 或 size 属性决定。

  2. Paint 对象：控制绘制样式的"画笔"，包括颜色、描边/填充模式、线条宽度、端点样式、渐变着色器和模糊滤镜等属性。

  3. Canvas 基础 API：drawLine（线）、drawRect（矩形）、drawRRect（圆角矩形）、drawCircle（圆）、drawOval（椭圆）、drawArc（弧/扇形）、drawPoints（点）、drawImage（图片）是绑制基础图形的核心方法。

  4. Canvas 变换与裁剪：save/restore 管理画布状态，translate/rotate/scale 实现坐标变换，clipRect/clipPath 实现裁剪。变换操作必须成对使用 save/restore。

  5. Path 路径：最强大的绘制工具，支持直线（lineTo）、二次贝塞尔曲线（quadraticBezierTo，1 个控制点）、三次贝塞尔曲线（cubicTo，2 个控制点）、弧线（arcTo/arcToPoint）以及预定义图形（addRect/addOval/addRRect/addPolygon）。Path.combine 支持五种布尔运算。

  6. TextPainter 文字绘制：通过 TextSpan 定义文字内容和样式，layout() 计算布局后 paint() 绘制到 Canvas，三步缺一不可。

  7. shouldRepaint 优化：静态内容返回 false，参数变化时精确比较后返回 true，动画驱动绑制使用 repaint 参数而非 shouldRepaint。

  8. 实战图表：饼图使用 drawArc + useCenter: true 绘制扇形，折线图需要坐标系映射、分层绘制、渐变填充等技术。

一句话总结：CustomPaint + Canvas 是 Flutter 中"画出一切"的终极武器——掌握 Paint、Path 和坐标变换，就没有画不出来的 UI。
