1. ClipRRect：圆角裁剪

1.1 基本概念

ClipRRect 用于将子组件裁剪为圆角矩形。它会真正裁剪子组件的绘制区域，超出圆角部分的像素不会被渲染。

1.2 核心属性

属性 类型 说明
borderRadius BorderRadiusGeometry 圆角半径，默认为 BorderRadius.zero
clipBehavior Clip 裁剪行为，默认 Clip.antiAlias
child Widget? 被裁剪的子组件

1.3 ClipRRect 与 Container(BoxDecoration) 圆角的区别

这是一个非常常见的误区：

  • Container + BoxDecoration(borderRadius:)：只是给 Container 自身的装饰（背景色、边框等）加上圆角，不会裁剪子组件。如果子组件（如图片）超出圆角区域，仍然会显示出来。
  • ClipRRect：真正裁剪子组件的绘制区域，超出圆角的部分会被完全剪掉。

1.4 代码示例

import 'package:flutter/material.dart';

class ClipRRectDemo extends StatelessWidget {
  const ClipRRectDemo({super.key});

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('ClipRRect 圆角裁剪')),
      body: Padding(
        padding: const EdgeInsets.all(16),
        child: Column(
          crossAxisAlignment: CrossAxisAlignment.start,
          children: [
            // ❌ Container 的 borderRadius 不会裁剪子组件
            const Text('Container + BoxDecoration（不裁剪子组件）：',
                style: TextStyle(fontWeight: FontWeight.bold)),
            const SizedBox(height: 8),
            Container(
              width: 200,
              height: 150,
              decoration: BoxDecoration(
                borderRadius: BorderRadius.circular(20),
                color: Colors.blue,
              ),
              // 图片超出圆角区域仍然可见
              child: Image.network(
                'https://picsum.photos/200/150',
                fit: BoxFit.cover,
              ),
            ),
            const SizedBox(height: 24),

            // ✅ ClipRRect 真正裁剪子组件
            const Text('ClipRRect（真正裁剪子组件）：',
                style: TextStyle(fontWeight: FontWeight.bold)),
            const SizedBox(height: 8),
            ClipRRect(
              borderRadius: BorderRadius.circular(20),
              child: SizedBox(
                width: 200,
                height: 150,
                child: Image.network(
                  'https://picsum.photos/200/150',
                  fit: BoxFit.cover,
                ),
              ),
            ),
            const SizedBox(height: 24),

            // 不同圆角半径
            const Text('不同圆角半径：',
                style: TextStyle(fontWeight: FontWeight.bold)),
            const SizedBox(height: 8),
            Row(
              children: [
                // 四个角不同的圆角
                ClipRRect(
                  borderRadius: const BorderRadius.only(
                    topLeft: Radius.circular(30),
                    bottomRight: Radius.circular(30),
                  ),
                  child: Container(
                    width: 100,
                    height: 100,
                    color: Colors.orange,
                    child: const Center(child: Text('对角圆角')),
                  ),
                ),
                const SizedBox(width: 16),
                // 完全圆形（半径 >= 宽高的一半）
                ClipRRect(
                  borderRadius: BorderRadius.circular(50),
                  child: Container(
                    width: 100,
                    height: 100,
                    color: Colors.purple,
                    child: const Center(
                      child: Text('圆形', style: TextStyle(color: Colors.white)),
                    ),
                  ),
                ),
              ],
            ),
          ],
        ),
      ),
    );
  }
}

注意事项：ClipRRect 的裁剪操作有一定的性能开销。如果只是需要给 Container 的背景加圆角，且不需要裁剪子组件，使用 BoxDecoration(borderRadius:) 即可，性能更好。只有当子组件确实会溢出圆角区域时（如图片），才需要使用 ClipRRect。

2. ClipOval：椭圆/圆形裁剪

2.1 基本概念

ClipOval 将子组件裁剪为椭圆形。如果子组件是正方形，则裁剪结果为圆形。这是实现圆形头像最常用的方式之一。

2.2 代码示例

import 'package:flutter/material.dart';

class ClipOvalDemo extends StatelessWidget {
  const ClipOvalDemo({super.key});

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('ClipOval 椭圆裁剪')),
      body: Padding(
        padding: const EdgeInsets.all(16),
        child: Column(
          children: [
            // 正方形 → 圆形
            const Text('正方形容器 → 圆形裁剪：'),
            const SizedBox(height: 8),
            ClipOval(
              child: Container(
                width: 120,
                height: 120,
                color: Colors.blue,
                child: Image.network(
                  'https://picsum.photos/120',
                  fit: BoxFit.cover,
                ),
              ),
            ),
            const SizedBox(height: 24),

            // 矩形 → 椭圆形
            const Text('矩形容器 → 椭圆裁剪：'),
            const SizedBox(height: 8),
            ClipOval(
              child: Container(
                width: 200,
                height: 120,
                color: Colors.green,
                child: const Center(
                  child: Text('椭圆', style: TextStyle(color: Colors.white)),
                ),
              ),
            ),
            const SizedBox(height: 24),

            // 圆形头像的常见写法
            const Text('圆形头像（推荐用 CircleAvatar）：'),
            const SizedBox(height: 8),
            Row(
              mainAxisAlignment: MainAxisAlignment.spaceEvenly,
              children: [
                // 方式一：ClipOval
                ClipOval(
                  child: Image.network(
                    'https://picsum.photos/100',
                    width: 80,
                    height: 80,
                    fit: BoxFit.cover,
                  ),
                ),
                // 方式二：CircleAvatar（内置方案，更方便）
                const CircleAvatar(
                  radius: 40,
                  backgroundImage: NetworkImage('https://picsum.photos/100'),
                ),
                // 方式三：Container + BoxDecoration（需配合 clipBehavior）
                Container(
                  width: 80,
                  height: 80,
                  clipBehavior: Clip.antiAlias,
                  decoration: const BoxDecoration(shape: BoxShape.circle),
                  child: Image.network(
                    'https://picsum.photos/100',
                    fit: BoxFit.cover,
                  ),
                ),
              ],
            ),
          ],
        ),
      ),
    );
  }
}

最佳实践：对于圆形头像场景，推荐使用 CircleAvatar，它内部已经处理好了裁剪逻辑且语义更清晰。ClipOval 更适合需要将任意子组件裁剪为椭圆/圆形的通用场景。

3. ClipPath：自定义路径裁剪

3.1 基本概念

ClipPath 允许你使用自定义路径（Path）来裁剪子组件，是三种 Clip 组件中最灵活的。你需要通过实现 CustomClipper<Path> 来定义裁剪路径。

3.2 CustomClipper 详解

方法 说明
Path getClip(Size size) 返回裁剪路径。size 为子组件的尺寸，用于计算路径坐标
bool shouldReclip(covariant CustomClipper<Path> oldClipper) 当 Clipper 被替换时是否需要重新计算裁剪路径。返回 true 则重新裁剪

3.3 代码示例：波浪裁剪

import 'package:flutter/material.dart';

/// 波浪形裁剪器
class WaveClipper extends CustomClipper<Path> {
  @override
  Path getClip(Size size) {
    final path = Path();
    // 从左上角开始
    path.lineTo(0, size.height * 0.75);

    // 第一段波浪（向上凸起）
    path.quadraticBezierTo(
      size.width * 0.25,  // 控制点 x
      size.height,        // 控制点 y
      size.width * 0.5,   // 终点 x
      size.height * 0.75, // 终点 y
    );

    // 第二段波浪（向下凹陷）
    path.quadraticBezierTo(
      size.width * 0.75,  // 控制点 x
      size.height * 0.5,  // 控制点 y
      size.width,         // 终点 x
      size.height * 0.75, // 终点 y
    );

    // 连接到右上角并闭合
    path.lineTo(size.width, 0);
    path.close();
    return path;
  }

  @override
  bool shouldReclip(WaveClipper oldClipper) => false;
}

/// 三角形裁剪器
class TriangleClipper extends CustomClipper<Path> {
  @override
  Path getClip(Size size) {
    final path = Path();
    path.moveTo(size.width / 2, 0);         // 顶部中间
    path.lineTo(size.width, size.height);    // 右下角
    path.lineTo(0, size.height);             // 左下角
    path.close();
    return path;
  }

  @override
  bool shouldReclip(TriangleClipper oldClipper) => false;
}

class ClipPathDemo extends StatelessWidget {
  const ClipPathDemo({super.key});

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('ClipPath 自定义裁剪')),
      body: Column(
        children: [
          // 波浪裁剪头部
          ClipPath(
            clipper: WaveClipper(),
            child: Container(
              height: 200,
              decoration: const BoxDecoration(
                gradient: LinearGradient(
                  colors: [Colors.deepPurple, Colors.purpleAccent],
                  begin: Alignment.topLeft,
                  end: Alignment.bottomRight,
                ),
              ),
              child: const Center(
                child: Text(
                  '波浪裁剪头部',
                  style: TextStyle(
                    color: Colors.white,
                    fontSize: 24,
                    fontWeight: FontWeight.bold,
                  ),
                ),
              ),
            ),
          ),
          const SizedBox(height: 32),

          // 三角形裁剪
          ClipPath(
            clipper: TriangleClipper(),
            child: Container(
              width: 150,
              height: 150,
              color: Colors.teal,
              child: const Center(
                child: Text('三角形',
                    style: TextStyle(color: Colors.white)),
              ),
            ),
          ),
        ],
      ),
    );
  }
}

注意事项：

  1. shouldReclip 返回 false 表示路径不会变化，可以避免不必要的重计算。如果裁剪路径依赖于外部参数（如动画值），则应返回 true 或比较参数是否变化。
  2. ClipPath 的性能开销比 ClipRRect 和 ClipOval 更大，因为自定义路径裁剪无法利用硬件加速的圆角裁剪优化。仅在必要时使用。

4. Transform：变换组件

4.1 基本概念

Transform 在绘制阶段对子组件施加矩阵变换，包括旋转、缩放、平移和倾斜。它不影响布局——子组件在布局阶段仍然占据变换前的空间。

4.2 核心属性

属性 类型 说明
transform Matrix4 4x4 变换矩阵
origin Offset? 变换原点偏移（相对于 alignment 计算后的位置）
alignment AlignmentGeometry? 变换原点对齐方式，默认为子组件左上角
transformHitTests bool 是否对命中测试也施加变换，默认 true

4.3 旋转：Transform.rotate

import 'dart:math' as math;
import 'package:flutter/material.dart';

class TransformRotateDemo extends StatelessWidget {
  const TransformRotateDemo({super.key});

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('Transform.rotate 旋转')),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: [
            // 旋转 45 度（弧度制：π/4）
            Transform.rotate(
              angle: math.pi / 4, // 45度
              child: Container(
                width: 100,
                height: 100,
                color: Colors.blue,
                child: const Center(
                  child: Text('45°', style: TextStyle(color: Colors.white)),
                ),
              ),
            ),
            const SizedBox(height: 60),

            // 指定旋转原点为左上角
            Transform.rotate(
              angle: math.pi / 6, // 30度
              origin: const Offset(-50, -50), // 左上角（相对于中心点的偏移）
              child: Container(
                width: 100,
                height: 100,
                color: Colors.red,
                child: const Center(
                  child: Text('左上角旋转', style: TextStyle(color: Colors.white)),
                ),
              ),
            ),
            const SizedBox(height: 60),

            // 通过 alignment 设置旋转锚点
            Transform.rotate(
              angle: -math.pi / 4, // -45度
              alignment: Alignment.bottomRight, // 以右下角为锚点旋转
              child: Container(
                width: 100,
                height: 100,
                color: Colors.green,
                child: const Center(
                  child: Text('右下角锚点',
                      style: TextStyle(color: Colors.white, fontSize: 12)),
                ),
              ),
            ),
          ],
        ),
      ),
    );
  }
}

4.6 倾斜：Matrix4

Flutter 没有提供 Transform.skew 快捷构造函数，但可以通过 Matrix4.skewX 和 Matrix4.skewY 实现倾斜效果。

import 'dart:math' as math;
import 'package:flutter/material.dart';

class TransformSkewDemo extends StatelessWidget {
  const TransformSkewDemo({super.key});

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('Transform + Matrix4 倾斜')),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: [
            // X 轴倾斜
            Transform(
              alignment: Alignment.center,
              transform: Matrix4.skewX(0.3), // X 轴倾斜 0.3 弧度
              child: Container(
                width: 120,
                height: 80,
                color: Colors.blue,
                child: const Center(
                  child: Text('skewX(0.3)',
                      style: TextStyle(color: Colors.white)),
                ),
              ),
            ),
            const SizedBox(height: 40),

            // Y 轴倾斜
            Transform(
              alignment: Alignment.center,
              transform: Matrix4.skewY(0.2), // Y 轴倾斜 0.2 弧度
              child: Container(
                width: 120,
                height: 80,
                color: Colors.red,
                child: const Center(
                  child: Text('skewY(0.2)',
                      style: TextStyle(color: Colors.white)),
                ),
              ),
            ),
            const SizedBox(height: 40),

            // 组合变换：旋转 + 缩放 + 平移
            Transform(
              alignment: Alignment.center,
              transform: Matrix4.identity()
                ..rotateZ(math.pi / 12)  // 旋转 15度
                ..scale(1.2)              // 缩放 1.2 倍
                ..translate(10.0, 5.0),   // 平移
              child: Container(
                width: 120,
                height: 80,
                color: Colors.green,
                child: const Center(
                  child: Text('组合变换',
                      style: TextStyle(color: Colors.white)),
                ),
              ),
            ),
            const SizedBox(height: 40),

            // 透视效果（3D 感）
            Transform(
              alignment: Alignment.center,
              transform: Matrix4.identity()
                ..setEntry(3, 2, 0.002) // 透视参数
                ..rotateX(0.5),          // 绕 X 轴旋转
              child: Container(
                width: 150,
                height: 100,
                color: Colors.purple,
                child: const Center(
                  child: Text('3D 透视',
                      style: TextStyle(color: Colors.white, fontSize: 18)),
                ),
              ),
            ),
          ],
        ),
      ),
    );
  }
}

注意事项：

  1. Transform 的变换发生在绘制阶段，不影响布局。这意味着变换后的组件仍然占据变换前的空间，可能与其他组件重叠。
  2. Matrix4 的链式调用顺序很重要——矩阵乘法不满足交换律。rotateZ 再 translate 与 translate 再 rotateZ 结果不同。
  3. 使用 ..setEntry(3, 2, value) 设置透视参数时，值通常在 0.001 ~ 0.003 之间，值越大透视效果越明显。

5. RotatedBox：布局级旋转

5.1 基本概念

RotatedBox 以 90 度的倍数旋转子组件。与 Transform.rotate 不同，RotatedBox 的旋转参与布局计算——旋转后的尺寸会被正确计算并传递给父组件。

5.2 RotatedBox 与 Transform.rotate 的区别

特性 RotatedBox Transform.rotate
旋转角度 仅支持 90° 的倍数 支持任意角度
是否影响布局 ✅ 旋转后参与布局计算 ❌ 布局阶段仍用旋转前尺寸
旋转阶段 布局（layout）阶段 绘制（paint）阶段
适用场景 侧边栏文字、旋转 Tab 等 装饰性旋转、动画旋转

5.3 代码示例

            // 实用场景：侧边标签
            const Text('实用场景 —— 侧边标签：',
                style: TextStyle(fontWeight: FontWeight.bold)),
            const SizedBox(height: 8),
            Row(
              children: [

}

6. Opacity 与 AnimatedOpacity：透明度控制

6.1 Opacity

Opacity 组件通过设置 opacity 参数（0.0 ~ 1.0）控制子组件的透明度。

6.2 AnimatedOpacity

AnimatedOpacity 是 Opacity 的隐式动画版本，当 opacity 值改变时会自动执行过渡动画。

6.3 代码示例

import 'package:flutter/material.dart';

class OpacityDemo extends StatefulWidget {
  const OpacityDemo({super.key});

  @override
  State<OpacityDemo> createState() => _OpacityDemoState();
}

class _OpacityDemoState extends State<OpacityDemo> {
  double _opacity = 1.0;

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('Opacity 透明度控制')),
      body: Center(
        child: Column(
          mainAxisAlignment: MainAxisAlignment.center,
          children: [
            // 静态 Opacity
            const Text('Opacity（无动画）：'),
            const SizedBox(height: 8),
            Row(
              mainAxisAlignment: MainAxisAlignment.spaceEvenly,
              children: [
                Opacity(
                  opacity: 1.0,
                  child: Container(width: 60, height: 60, color: Colors.blue),
                ),
                Opacity(
                  opacity: 0.6,
                  child: Container(width: 60, height: 60, color: Colors.blue),
                ),
                Opacity(
                  opacity: 0.3,
                  child: Container(width: 60, height: 60, color: Colors.blue),
                ),
                Opacity(
                  opacity: 0.0, // 完全透明，但仍参与布局和命中测试
                  child: Container(width: 60, height: 60, color: Colors.blue),
                ),
              ],
            ),
            const SizedBox(height: 8),
            const Text('1.0      0.6      0.3      0.0'),
            const SizedBox(height: 40),

            // AnimatedOpacity（带动画效果）
            const Text('AnimatedOpacity（带过渡动画）：'),
            const SizedBox(height: 8),
            AnimatedOpacity(
              opacity: _opacity,
              duration: const Duration(milliseconds: 500),
              curve: Curves.easeInOut,
              child: Container(
                width: 150,
                height: 150,
                decoration: BoxDecoration(
                  color: Colors.purple,
                  borderRadius: BorderRadius.circular(16),
                ),
                child: const Center(
                  child: Text('渐隐渐现',
                      style: TextStyle(color: Colors.white, fontSize: 18)),
                ),
              ),
            ),
            const SizedBox(height: 16),

            // 使用 Slider 控制透明度
            Slider(
              value: _opacity,
              onChanged: (value) => setState(() => _opacity = value),
            ),
            Text('当前透明度: ${_opacity.toStringAsFixed(2)}'),
          ],
        ),
      ),
    );
  }
}

性能注意事项：

  1. Opacity 组件会将子组件绘制到一个离屏缓冲区（offscreen buffer）中，然后再以指定透明度合成到屏幕上。对于简单的颜色透明度，直接使用 Color.withOpacity() 性能更好。
  2. 当 opacity 为 0.0 时，子组件虽然不可见但仍然参与布局和命中测试。如果需要完全移除，应使用 Visibility 或 Offstage。
  3. 当 opacity 为 1.0 时，Flutter 会优化跳过离屏缓冲区步骤。

7. Visibility 与 Offstage：显示/隐藏策略

7.1 对比

特性 Visibility Offstage
隐藏时是否保留空间 可选（maintainSize 参数） 不保留（完全移出布局）
隐藏时是否保留状态 可选（maintainState 参数） ✅ 保留状态
隐藏时是否响应交互 可选（maintainInteractivity） ❌ 不响应
替换组件 支持 replacement 参数 不支持
控制参数 visible（true/false） offstage（true/false）

7.2 Visibility 详解

Visibility 提供了非常细粒度的控制：

  • visible: false 时默认完全移除子组件（类似条件渲染）
  • maintainSize: true → 保留空间但不可见（类似 CSS 的 visibility: hidden）
  • maintainState: true → 保留子组件的 State（不会触发 dispose）
  • maintainAnimation: true → 保留动画状态
  • maintainInteractivity: true → 隐藏时仍然可以响应手势

7.3 代码示例

import 'package:flutter/material.dart';

class VisibilityDemo extends StatefulWidget {
  const VisibilityDemo({super.key});

  @override
  State<VisibilityDemo> createState() => _VisibilityDemoState();
}

class _VisibilityDemoState extends State<VisibilityDemo> {
  bool _isVisible = true;
  bool _isOnstage = true;

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('Visibility 与 Offstage')),
      body: Padding(
        padding: const EdgeInsets.all(16),
        child: Column(
          crossAxisAlignment: CrossAxisAlignment.start,
          children: [
            // --- Visibility 示例 ---
            const Text('Visibility（保留空间）：',
                style: TextStyle(fontWeight: FontWeight.bold, fontSize: 16)),
            const SizedBox(height: 8),
            Row(
              children: [
                Container(
                  width: 80,
                  height: 80,
                  color: Colors.green,
                  child: const Center(child: Text('A')),
                ),
                Visibility(
                  visible: _isVisible,
                  maintainSize: true,     // 隐藏后保留原来的空间
                  maintainState: true,    // 保留子组件的 State
                  maintainAnimation: true,
                  child: Container(
                    width: 80,
                    height: 80,
                    color: Colors.blue,
                    child: const Center(
                      child: Text('B', style: TextStyle(color: Colors.white)),
                    ),
                  ),
                ),
                Container(
                  width: 80,
                  height: 80,
                  color: Colors.red,
                  child: const Center(
                    child: Text('C', style: TextStyle(color: Colors.white)),
                  ),
                ),
              ],
            ),
            const SizedBox(height: 8),
            ElevatedButton(
              onPressed: () => setState(() => _isVisible = !_isVisible),
              child: Text(_isVisible ? '隐藏 B（保留空间）' : '显示 B'),

            const SizedBox(height: 24),

              ),
              child: Container(
                padding: const EdgeInsets.all(16),
                decoration: BoxDecoration(
                  color: Colors.blue.shade100,
                  borderRadius: BorderRadius.circular(8),
                ),
                child: const Text('这是原始内容'),
              ),
            ),

            const SizedBox(height: 24),

            // --- Offstage 示例 ---
            const Text('Offstage（完全移出布局，保留状态）：',
                style: TextStyle(fontWeight: FontWeight.bold, fontSize: 16)),
            const SizedBox(height: 8),
            Row(
              children: [
                Container(
                  width: 80,
                  height: 80,
                  color: Colors.green,
                  child: const Center(child: Text('A')),
                ),
                Offstage(
                  offstage: !_isOnstage, // true = 隐藏
                  child: Container(
                    width: 80,
                    height: 80,
                    color: Colors.orange,
                    child: const Center(
                      child: Text('B', style: TextStyle(color: Colors.white)),
                    ),
                  ),
                ),
                Container(
                  width: 80,
                  height: 80,
                  color: Colors.red,
                  child: const Center(
                    child: Text('C', style: TextStyle(color: Colors.white)),
                  ),
                ),
              ],
            ),
            const SizedBox(height: 8),
            ElevatedButton(
              onPressed: () => setState(() => _isOnstage = !_isOnstage),
              child: Text(_isOnstage ? '隐藏 B（Offstage）' : '显示 B'),
            ),
            const SizedBox(height: 8),
            const Text(
              '注意观察：Offstage 隐藏后 C 会紧挨 A，不保留空间',
              style: TextStyle(color: Colors.grey),
            ),
          ],
        ),
      ),
    );
  }
}

最佳实践：

  • 需要隐藏但保留空间 → Visibility(maintainSize: true)
  • 需要隐藏且不占空间，但保留状态 → Offstage
  • 需要隐藏且不需要保留状态 → 直接用条件表达式 if (show) widget
  • Offstage 虽然不可见，子组件的 State 仍然存活，定时器、动画等仍在运行，注意资源管理。

8. BackdropFilter：高斯模糊与毛玻璃效果

8.1 基本概念

BackdropFilter 对其下方已绘制的内容（而非子组件）施加滤镜效果，最常见的用途是创建毛玻璃（frosted glass）效果。

关键要点：

  • BackdropFilter 模糊的是它下方的图层，不是它的子组件
  • 通常需要配合 Stack 使用，将 BackdropFilter 叠在背景内容之上
  • 子组件（child）会被绘制在滤镜效果之上

8.2 核心属性

属性 类型 说明
filter ImageFilter 滤镜效果，通常使用 ImageFilter.blur(sigmaX:, sigmaY:)
blendMode BlendMode 混合模式，默认 BlendMode.srcOver
child Widget? 绘制在滤镜之上的子组件

8.3 代码示例

import 'dart:ui';
import 'package:flutter/material.dart';

class BackdropFilterDemo extends StatelessWidget {
  const BackdropFilterDemo({super.key});

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('BackdropFilter 高斯模糊')),
      body: Stack(
        fit: StackFit.expand,
        children: [
          // 底层：背景图片
          Image.network(
            'https://picsum.photos/800/1200',
            fit: BoxFit.cover,
          ),

          // 中层：底部毛玻璃区域
          Positioned(
            left: 16,
            right: 16,
            bottom: 40,
            child: ClipRRect(
              borderRadius: BorderRadius.circular(20),
              child: BackdropFilter(
                filter: ImageFilter.blur(sigmaX: 15, sigmaY: 15),
                child: Container(
                  padding: const EdgeInsets.all(24),
                  decoration: BoxDecoration(
                    // 半透明白色叠加在模糊效果上
                    color: Colors.white.withOpacity(0.2),
                    borderRadius: BorderRadius.circular(20),
                    border: Border.all(
                      color: Colors.white.withOpacity(0.3),
                    ),
                  ),
                  child: const Column(
                    crossAxisAlignment: CrossAxisAlignment.start,
                    children: [
                      Text(
                        '毛玻璃效果',
                        style: TextStyle(
                          color: Colors.white,
                          fontSize: 24,
                          fontWeight: FontWeight.bold,
                        ),
                      ),
                      SizedBox(height: 8),
                      Text(
                        'BackdropFilter 会模糊其下方已绘制的内容，'
                        '配合半透明背景实现毛玻璃效果。',
                        style: TextStyle(color: Colors.white70, fontSize: 14),
                      ),
                    ],
                  ),
                ),
              ),
            ),
          ),

          // 顶部也加一个小的毛玻璃标签
          Positioned(
            top: 16,
            right: 16,
            child: ClipRRect(
              borderRadius: BorderRadius.circular(12),
              child: BackdropFilter(
                filter: ImageFilter.blur(sigmaX: 10, sigmaY: 10),
                child: Container(
                  padding: const EdgeInsets.symmetric(
                      horizontal: 16, vertical: 8),
                  color: Colors.black.withOpacity(0.2),

                    style: TextStyle(color: Colors.white),
                  ),
                ),
              ),
            ),
          ),
        ],
      ),
    );
  }
}

注意事项：

  1. BackdropFilter 必须配合 ClipRRect（或其他裁剪组件）使用，否则模糊效果会扩展到整个父组件区域。
  2. 高斯模糊的 sigma 值越大，模糊效果越明显，性能开销也越大。建议 sigma 不超过 20。
  3. BackdropFilter 是相对昂贵的操作，会触发离屏渲染。避免在列表中大量使用。
  4. iOS 风格的毛玻璃效果也可以直接使用 CupertinoNavigationBar 等 Cupertino 组件，它们内置了模糊效果。

9. ShaderMask：着色器遮罩效果

9.1 基本概念

ShaderMask 用一个着色器（Shader）对子组件的绘制结果进行遮罩混合。最常见的用途是实现渐变文字和渐变图标效果。

9.2 核心属性

属性 类型 说明
shaderCallback ShaderCallback 回调函数，接收子组件的绘制区域 Rect，返回一个 Shader
blendMode BlendMode 混合模式，默认 BlendMode.modulate
child Widget? 被遮罩的子组件

9.3 blendMode 常用值

  • BlendMode.srcIn：只保留着色器与子组件重叠的部分（常用于渐变文字）
  • BlendMode.srcATop：在子组件上叠加着色器
  • BlendMode.modulate：将着色器颜色与子组件颜色相乘

9.4 代码示例

import 'package:flutter/material.dart';

class ShaderMaskDemo extends StatelessWidget {
  const ShaderMaskDemo({super.key});

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('ShaderMask 着色器遮罩')),
      body: Padding(
        padding: const EdgeInsets.all(16),
        child: Column(
          crossAxisAlignment: CrossAxisAlignment.start,
          children: [
            // 渐变文字
            const Text('渐变文字效果：',
                style: TextStyle(fontWeight: FontWeight.bold)),
            const SizedBox(height: 8),
            ShaderMask(
              blendMode: BlendMode.srcIn, // 关键：只保留文字区域的渐变
              shaderCallback: (Rect bounds) {
                return const LinearGradient(
                  colors: [Colors.blue, Colors.purple, Colors.red],
                ).createShader(bounds);
              },
              child: const Text(
                'Flutter 渐变文字',
                style: TextStyle(
                  fontSize: 36,
                  fontWeight: FontWeight.bold,
                ),
              ),
            ),
            const SizedBox(height: 24),

            // 渐变图标
            const Text('渐变图标效果：',
                style: TextStyle(fontWeight: FontWeight.bold)),
            const SizedBox(height: 8),
            ShaderMask(
              blendMode: BlendMode.srcIn,
              shaderCallback: (Rect bounds) {
                return const LinearGradient(
                  colors: [Colors.orange, Colors.pink],
                  begin: Alignment.topLeft,
                  end: Alignment.bottomRight,
                ).createShader(bounds);
              },
              child: const Row(
                mainAxisSize: MainAxisSize.min,
                children: [
                  Icon(Icons.favorite, size: 48),
                  SizedBox(width: 8),
                  Icon(Icons.star, size: 48),
                  SizedBox(width: 8),
                  Icon(Icons.thumb_up, size: 48),
                ],
              ),
            ),
            const SizedBox(height: 24),

            // 径向渐变效果
            const Text('径向渐变效果：',
                style: TextStyle(fontWeight: FontWeight.bold)),
            const SizedBox(height: 8),
            ShaderMask(
              blendMode: BlendMode.srcATop,
              shaderCallback: (Rect bounds) {
                return RadialGradient(
                  center: Alignment.center,
                  radius: 0.8,
                  colors: [
                    Colors.yellow.withOpacity(0.8),
                    Colors.transparent,
                  ],
                ).createShader(bounds);
              },
              child: Container(
                width: 200,
                height: 200,
                decoration: BoxDecoration(
                  color: Colors.blue,
                  borderRadius: BorderRadius.circular(16),
                ),
                child: const Center(
                  child: Text('径向高光',
                      style: TextStyle(color: Colors.white, fontSize: 18)),
                ),
              ),
            ),
            const SizedBox(height: 24),

            // 渐变淡出效果（列表底部渐隐）
            const Text('底部渐隐效果：',
                style: TextStyle(fontWeight: FontWeight.bold)),
            const SizedBox(height: 8),
            SizedBox(
              height: 120,
              child: ShaderMask(
                blendMode: BlendMode.dstIn,
                shaderCallback: (Rect bounds) {
                  return const LinearGradient(
                    begin: Alignment.topCenter,
                    end: Alignment.bottomCenter,
                    colors: [Colors.white, Colors.transparent],
                    stops: [0.6, 1.0], // 前 60% 完全可见，后 40% 渐隐
                  ).createShader(bounds);
                },
                child: ListView(
                  children: List.generate(
                    10,
                    (i) => ListTile(title: Text('列表项 $i')),
                  ),
                ),
              ),
            ),
          ],
        ),
      ),
    );
  }
}

最佳实践：

  • 渐变文字务必使用 BlendMode.srcIn，它会将渐变限制在文字的不透明像素区域内。
  • shaderCallback 中的 bounds 是子组件的绘制区域，可以直接传给 Gradient.createShader()。
  • ShaderMask 的性能开销中等，对于简单渐变效果可以放心使用，但避免在高频重建的组件中使用复杂着色器。

/// 波浪裁剪器
class WaveHeaderClipper extends CustomClipper<Path> {
  @override
  Path getClip(Size size) {
    final path = Path();
    path.lineTo(0, size.height - 50);

    // 使用三次贝塞尔曲线绘制平滑波浪
    final firstControlPoint = Offset(size.width * 0.25, size.height);
    final firstEndPoint = Offset(size.width * 0.5, size.height - 30);
    path.cubicTo(
      firstControlPoint.dx, firstControlPoint.dy,
      firstEndPoint.dx - 30, firstEndPoint.dy,
      firstEndPoint.dx, firstEndPoint.dy,
    );

    final secondControlPoint = Offset(size.width * 0.75, size.height - 80);
    final secondEndPoint = Offset(size.width, size.height - 40);
    path.cubicTo(
      secondControlPoint.dx - 30, secondControlPoint.dy,
      secondControlPoint.dx, secondControlPoint.dy,
      secondEndPoint.dx, secondEndPoint.dy,
    );

    path.lineTo(size.width, 0);
    path.close();
    return path;
  }

  @override
  bool shouldReclip(WaveHeaderClipper oldClipper) => false;
}

/// 倒圆弧裁剪器
class CurvedBottomClipper extends CustomClipper<Path> {
  @override
  Path getClip(Size size) {
    final path = Path();
    path.lineTo(0, size.height - 40);

    // 一条平滑的弧线
    path.quadraticBezierTo(
      size.width / 2, size.height + 20,
      size.width, size.height - 40,
    );

    path.lineTo(size.width, 0);
    path.close();
    return path;
  }

  @override
  bool shouldReclip(CurvedBottomClipper oldClipper) => false;
}

11. 本课小结

本课涵盖了 Flutter 中用于视觉效果处理的关键组件：

组件 核心用途 重点
ClipRRect 圆角裁剪 与 Container 装饰圆角的区别：真正裁剪 vs 只装饰
ClipOval 椭圆/圆形裁剪 圆形头像可优先用 CircleAvatar
ClipPath 自定义路径裁剪 CustomClipper、shouldReclip 优化
Transform 旋转/缩放/平移/倾斜 不影响布局，Matrix4 链式调用顺序敏感
RotatedBox 90° 倍数旋转 参与布局计算，适合侧边文字
Opacity / AnimatedOpacity 透明度控制 opacity=0 仍占空间和参与命中测试
Visibility / Offstage 显示/隐藏 各有不同的空间、状态保留策略
BackdropFilter 高斯模糊/毛玻璃 需配合 Stack 和 ClipRRect 使用
ShaderMask 着色器遮罩 渐变文字用 BlendMode.srcIn

性能建议速查

  1. 只在子组件溢出时才用 Clip 系列组件，否则用 BoxDecoration 足够
  2. Opacity 适合动态变化场景，静态透明度优先用 Color.withOpacity()
  3. BackdropFilter 开销大，避免在列表或频繁重建的组件中使用
  4. Transform 不影响布局，如需布局跟随旋转用 RotatedBox
  5. 隐藏组件时优先考虑条件渲染 if，仅在需要保留状态时使用 Offstage
