第02课 文本组件

文本是任何应用中最基础、最常见的 UI 元素。Flutter 提供了一套功能丰富的文本组件体系，从简单的单行文字到复杂的富文本排版，再到可选中、可交互的文本，均有对应的解决方案。本课将系统讲解 Flutter 中所有常用的文本组件及其核心用法。

1. Text 组件

Text 是 Flutter 中最基础、使用频率最高的文本组件。它用于在界面上显示一段样式统一的文本。

1.1 基础用法

最简单的用法只需要传入一个字符串：

import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: const Text('Text 基础用法')),
        body: const Center(
          child: Text('Hello, Flutter!'),
        ),
      ),
    );
  }
}

1.2 TextStyle 详解

TextStyle 是控制文本外观的核心类，可以设置字体大小、粗细、颜色、间距、装饰线等多种属性。

1.2.1 fontSize —— 字体大小

fontSize 的单位是逻辑像素（logical pixels），Flutter 默认字体大小为 14.0。

Text(
  '这是 24 像素的文字',
  style: TextStyle(fontSize: 24.0),
)

1.2.2 fontWeight —— 字体粗细

fontWeight 控制字体的粗细程度，取值范围从 FontWeight.w100（最细）到 FontWeight.w900（最粗）。常用的 FontWeight.bold 等同于 FontWeight.w700。

Text(
  '粗体文字',
  style: TextStyle(fontWeight: FontWeight.bold),
)

1.2.3 color —— 文字颜色

Text(
  '红色文字',
  style: TextStyle(color: Colors.red),
)

1.2.4 letterSpacing 与 wordSpacing —— 字符间距与单词间距

  • letterSpacing：每个字符之间的额外间距（对中文来说就是每个汉字之间的间距）
  • wordSpacing：每个单词之间的额外间距（以空格为分隔符，对纯中文无明显效果）
Text(
  'Hello Flutter World 你好世界',
  style: TextStyle(
    letterSpacing: 4.0,  // 字符间距
    wordSpacing: 10.0,   // 单词间距
  ),
)

1.2.5 height —— 行高

height 是一个倍数因子，实际行高 = fontSize * height。例如 fontSize 为 16，height 为 1.5，则实际行高为 24 逻辑像素。不设置时 Flutter 会使用字体自带的默认行高。

Text(
  '这是一段多行文本，用于演示行高的效果。'
  '当文本内容较长时，设置合适的行高可以提升可读性。'
  '一般正文建议行高倍数为 1.4 ~ 1.8。',
  style: TextStyle(
    fontSize: 16.0,
    height: 1.6, // 行高倍数
  ),
)

1.2.6 decoration —— 装饰线

decoration 用于添加文本装饰线（下划线、删除线、上划线），配合 decorationColor、decorationStyle 和 decorationThickness 使用。

Text(
  '带下划线的文字',
  style: TextStyle(
    decoration: TextDecoration.underline,
    decorationColor: Colors.blue,
    decorationStyle: TextDecorationStyle.dashed, // 虚线下划线
    decorationThickness: 2.0, // 装饰线粗细
  ),
)

TextDecoration 提供以下常量：

  • TextDecoration.none —— 无装饰
  • TextDecoration.underline —— 下划线
  • TextDecoration.overline —— 上划线
  • TextDecoration.lineThrough —— 删除线

也可以组合多种装饰：

TextDecoration.combine([
  TextDecoration.underline,
  TextDecoration.overline,
])

1.2.7 完整 TextStyle 示例

下面的示例演示了 TextStyle 的主要属性组合使用：

import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: const Text('TextStyle 详解')),
        body: Padding(
          padding: const EdgeInsets.all(16.0),

            crossAxisAlignment: CrossAxisAlignment.start,
            children: [
              // 基础样式
              const Text(
                '默认样式文本',
              ),
              const SizedBox(height: 12),

              // 字体大小与粗细
              const Text(
                '大号粗体文本',
                style: TextStyle(
                  fontSize: 28.0,
                  fontWeight: FontWeight.w800,
                ),
              ),
              const SizedBox(height: 12),

              // 颜色与斜体
              const Text(
                '蓝色斜体文本',
                style: TextStyle(
                  color: Colors.blue,
                  fontStyle: FontStyle.italic,
                  fontSize: 18.0,
                ),
              ),
              const SizedBox(height: 12),

              // 字符间距
              const Text(
                '宽 字 符 间 距',
                style: TextStyle(
                  letterSpacing: 8.0,
                  fontSize: 18.0,
                ),
              ),
              const SizedBox(height: 12),

              // 行高
              const Text(
                '这是一段较长的文本，用来演示行高属性的效果。'
                '合适的行高能让多行文本更易阅读，'
                '推荐正文使用 1.5 到 1.8 的行高倍数。',
                style: TextStyle(
                  fontSize: 16.0,
                  height: 1.8,
                ),
              ),
              const SizedBox(height: 12),

              // 装饰线
              const Text(
                '删除线效果（原价 ¥299）',
                style: TextStyle(
                  fontSize: 16.0,
                  decoration: TextDecoration.lineThrough,
                  decorationColor: Colors.red,
                  decorationThickness: 2.0,
                  color: Colors.grey,
                ),
              ),
              const SizedBox(height: 12),

              // 阴影效果
              Text(
                '带阴影的文字',
                style: TextStyle(
                  fontSize: 24.0,
                  fontWeight: FontWeight.bold,
                  color: Colors.black,
                  shadows: [
                    Shadow(
                      offset: const Offset(2.0, 2.0),
                      blurRadius: 4.0,
                      color: Colors.grey.shade400,
                    ),
                  ],
                ),
              ),
            ],
          ),
        ),
      ),
    );
  }
}

1.3 textAlign —— 文本对齐

textAlign 控制文本在其布局区域内的水平对齐方式。注意：只有当 Text 组件的宽度大于文本实际宽度时，对齐才会有可见的效果。

可用的对齐方式：

  • TextAlign.left —— 左对齐（默认）
  • TextAlign.right —— 右对齐
  • TextAlign.center —— 居中对齐
  • TextAlign.justify —— 两端对齐（自动调整单词间距使文本填满整行）
  • TextAlign.start —— 根据文本方向对齐（LTR 时等于 left，RTL 时等于 right）
  • TextAlign.end —— 与 start 相反
import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: const Text('文本对齐')),
        body: const Padding(
          padding: EdgeInsets.all(16.0),
          child: Column(
            crossAxisAlignment: CrossAxisAlignment.stretch,
            children: [
              Text(
                '左对齐（默认）',
                textAlign: TextAlign.left,
                style: TextStyle(fontSize: 16),
              ),
              SizedBox(height: 16),
              Text(
                '居中对齐',
                textAlign: TextAlign.center,
                style: TextStyle(fontSize: 16),
              ),
              SizedBox(height: 16),
              Text(
                '右对齐',
                textAlign: TextAlign.right,
                style: TextStyle(fontSize: 16),
              ),
              SizedBox(height: 16),
              Text(
                '两端对齐：这是一段较长的文本内容，用来演示 justify 的效果，'
                '两端对齐会自动调整单词和字符之间的间距来填满整行。',
                textAlign: TextAlign.justify,
                style: TextStyle(fontSize: 16),
              ),
            ],
          ),
        ),
      ),
    );
  }
}

1.4 overflow —— 溢出处理

当文本内容超过可用空间时，overflow 决定如何处理溢出的部分：

  • TextOverflow.clip —— 直接裁剪（默认）
  • TextOverflow.fade —— 渐隐效果
  • TextOverflow.ellipsis —— 显示省略号 ...
  • TextOverflow.visible —— 允许溢出显示

1.5 maxLines —— 最大行数

maxLines 限制文本显示的最大行数，超出部分按 overflow 设置处理。maxLines 和 overflow 通常搭配使用。

import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    const longText = '这是一段很长的文本内容，用于演示文本溢出和最大行数的处理效果。'
        'Flutter 提供了多种溢出处理方式，包括裁剪、渐隐和省略号。'
        '在实际开发中，我们经常需要限制文本的显示行数，'
        '比如列表项中的简介、新闻摘要等场景。';

    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: const Text('溢出与最大行数')),
        body: Padding(
          padding: const EdgeInsets.all(16.0),
          child: Column(
            crossAxisAlignment: CrossAxisAlignment.start,
            children: [
              // 省略号处理
              const Text('1. maxLines + ellipsis:',
                  style: TextStyle(fontWeight: FontWeight.bold)),
              const SizedBox(height: 4),
              const Text(
                longText,
                maxLines: 2,
                overflow: TextOverflow.ellipsis,
                style: TextStyle(fontSize: 15),
              ),
              const SizedBox(height: 20),

              // 渐隐处理
              const Text('2. maxLines + fade:',
                  style: TextStyle(fontWeight: FontWeight.bold)),
              const SizedBox(height: 4),
              const Text(
                longText,
                maxLines: 2,
                overflow: TextOverflow.fade,
                style: TextStyle(fontSize: 15),
              ),
              const SizedBox(height: 20),

              // 直接裁剪
              const Text('3. maxLines + clip (默认):',
                  style: TextStyle(fontWeight: FontWeight.bold)),
              const SizedBox(height: 4),
              const Text(
                longText,
                maxLines: 2,
                overflow: TextOverflow.clip,
                style: TextStyle(fontSize: 15),
              ),
              const SizedBox(height: 20),

              // 不限制行数
              const Text('4. 不限制行数（完整显示）:',
                  style: TextStyle(fontWeight: FontWeight.bold)),
              const SizedBox(height: 4),
              const Text(
                longText,
                style: TextStyle(fontSize: 15),
              ),
            ],
          ),
        ),
      ),
    );
  }
}

最佳实践： 在列表项、卡片标题等有限空间中，务必设置 maxLines 和 overflow: TextOverflow.ellipsis，防止文本溢出导致布局异常。

1.6 其他常用属性

属性 类型 说明
textScaler TextScaler 文本缩放因子，替代已弃用的 textScaleFactor
softWrap bool 是否自动换行，默认为 true

2. Text.rich 与 TextSpan

当需要在同一行或同一段文本中混合使用不同的样式时，单一的 Text 组件无法胜任——这时就需要 Text.rich 和 TextSpan。

2.1 基本概念

  • Text.rich 是 Text 的一个命名构造函数，接收一个 InlineSpan（通常是 TextSpan）而非纯字符串
  • TextSpan 表示一段带样式的文本片段，支持通过 children 属性嵌套多个子 TextSpan
  • 每个 TextSpan 可以有独立的 style、text 和 recognizer（手势识别）

2.2 富文本拼接

import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: const Text('Text.rich 富文本')),
        body: const Padding(
          padding: EdgeInsets.all(16.0),
          child: Column(
            crossAxisAlignment: CrossAxisAlignment.start,
            children: [
              // 基础富文本拼接
              Text.rich(
                TextSpan(
                  text: '欢迎使用 ', // 默认样式部分
                  style: TextStyle(fontSize: 18, color: Colors.black),
                  children: [
                    TextSpan(
                      text: 'Flutter',
                      style: TextStyle(
                        color: Colors.blue,
                        fontWeight: FontWeight.bold,
                        fontSize: 22,
                      ),
                    ),
                    TextSpan(
                      text: ' 进行开发！',
                    ),
                  ],
                ),
              ),
              SizedBox(height: 24),

              // 模拟"价格标签"场景
              Text.rich(
                TextSpan(
                  children: [
                    TextSpan(
                      text: '¥',
                      style: TextStyle(
                        fontSize: 14,
                        color: Colors.red,
                        fontWeight: FontWeight.bold,
                      ),
                    ),
                    TextSpan(
                      text: '199',
                      style: TextStyle(
                        fontSize: 28,
                        color: Colors.red,
                        fontWeight: FontWeight.bold,
                      ),
                    ),
                    TextSpan(
                      text: '.00',
                      style: TextStyle(
                        fontSize: 14,
                        color: Colors.red,
                      ),
                    ),
                    TextSpan(
                      text: '  ¥399',
                      style: TextStyle(
                        fontSize: 14,
                        color: Colors.grey,
                        decoration: TextDecoration.lineThrough,
                      ),
                    ),
                  ],
                ),
              ),
            ],
          ),
        ),
      ),
    );
  }
}

2.3 嵌套 TextSpan

TextSpan 支持多层嵌套，子 TextSpan 会继承父级的 style（未显式设置的属性会从父级继承）：

import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: const Text('嵌套 TextSpan')),
        body: const Padding(
          padding: EdgeInsets.all(16.0),
          child: Text.rich(
            TextSpan(
              // 顶层样式：所有子节点默认继承此样式
              style: TextStyle(fontSize: 16, color: Colors.black87),
              children: [
                TextSpan(text: '这是普通文字，'),
                TextSpan(
                  text: '这段是粗体',
                  style: TextStyle(fontWeight: FontWeight.bold),
                  children: [
                    // 嵌套的子节点会继承父级的 bold 样式
                    TextSpan(
                      text: '（继承粗体并添加红色）',
                      style: TextStyle(color: Colors.red),
                    ),
                  ],
                ),
                TextSpan(text: '，回到普通样式。'),
              ],
            ),
          ),
        ),
      ),
    );
  }
}

注意事项： TextSpan 的样式继承是"合并"式的——子 TextSpan 只覆盖显式设置的属性，其余属性从父级继承。这与 CSS 的样式继承机制类似。

2.4 GestureRecognizer —— 部分文字可点击

TextSpan 的 recognizer 属性接收一个 GestureRecognizer，可以让某一段文字响应手势（如点击）。这在实现"同意协议""阅读更多"等场景时非常有用。

注意：使用 GestureRecognizer 时必须在 dispose 中释放它，否则会导致内存泄漏。

import 'package:flutter/gestures.dart';
import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: const AgreementPage(),
    );
  }
}

class AgreementPage extends StatefulWidget {
  const AgreementPage({super.key});

  @override
  State<AgreementPage> createState() => _AgreementPageState();
}

class _AgreementPageState extends State<AgreementPage> {
  // 必须持有 recognizer 的引用，以便在 dispose 中释放
  late final TapGestureRecognizer _termsRecognizer;
  late final TapGestureRecognizer _privacyRecognizer;

  @override
  void initState() {
    super.initState();
    _termsRecognizer = TapGestureRecognizer()
      ..onTap = () {
        // 跳转到用户协议页面
        ScaffoldMessenger.of(context).showSnackBar(
          const SnackBar(content: Text('点击了用户协议')),
        );
      };
    _privacyRecognizer = TapGestureRecognizer()
      ..onTap = () {
        // 跳转到隐私政策页面
        ScaffoldMessenger.of(context).showSnackBar(
          const SnackBar(content: Text('点击了隐私政策')),
        );
      };
  }

  @override
  void dispose() {
    // 释放手势识别器，防止内存泄漏
    _termsRecognizer.dispose();
    _privacyRecognizer.dispose();
    super.dispose();
  }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('协议同意场景')),
      body: Padding(
        padding: const EdgeInsets.all(16.0),
        child: Text.rich(
          TextSpan(
            style: const TextStyle(fontSize: 14, color: Colors.black87),
            children: [
              const TextSpan(text: '注册即表示您同意'),
              TextSpan(
                text: '《用户协议》',
                style: const TextStyle(color: Colors.blue),
                recognizer: _termsRecognizer,
              ),
              const TextSpan(text: '和'),
              TextSpan(
                text: '《隐私政策》',
                style: const TextStyle(color: Colors.blue),
                recognizer: _privacyRecognizer,
              ),
              const TextSpan(text: '。'),
            ],
          ),
        ),
      ),
    );
  }
}

最佳实践： 对于可点击的文字链接，建议使用不同的颜色（如蓝色）和下划线来提示用户该文本可以交互，提升可发现性。

3. RichText 组件

RichText 是一个更底层的富文本组件，它直接接收 InlineSpan 并渲染。

3.1 基本用法

import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: const Text('RichText 示例')),
        body: Padding(
          padding: const EdgeInsets.all(16.0),
          child: RichText(
            text: const TextSpan(
              style: TextStyle(fontSize: 16, color: Colors.black),
              children: [
                TextSpan(text: '使用 '),
                TextSpan(
                  text: 'RichText',
                  style: TextStyle(
                    fontWeight: FontWeight.bold,
                    color: Colors.deepPurple,
                    fontSize: 20,
                  ),
                ),
                TextSpan(text: ' 组件可以实现'),
                TextSpan(
                  text: '灵活的富文本排版',
                  style: TextStyle(
                    color: Colors.orange,
                    fontStyle: FontStyle.italic,
                  ),
                ),
                TextSpan(text: '。'),
              ],
            ),
          ),
        ),
      ),
    );
  }
}

3.2 RichText 中嵌入 Widget —— WidgetSpan

RichText 的一大优势是支持 WidgetSpan，可以在文本中内嵌任意 Widget（如图标、图片、标签等）：

import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: const Text('WidgetSpan 示例')),
        body: Padding(
          padding: const EdgeInsets.all(16.0),
          child: RichText(
            text: TextSpan(
              style: const TextStyle(fontSize: 16, color: Colors.black87),
              children: [
                const TextSpan(text: '点击 '),
                WidgetSpan(
                  alignment: PlaceholderAlignment.middle,
                  child: Icon(
                    Icons.favorite,
                    color: Colors.red,
                    size: 20,
                  ),
                ),
                const TextSpan(text: ' 收藏本文，点击 '),
                WidgetSpan(
                  alignment: PlaceholderAlignment.middle,
                  child: Container(
                    padding: const EdgeInsets.symmetric(
                      horizontal: 6,
                      vertical: 2,
                    ),
                    decoration: BoxDecoration(
                      color: Colors.blue,
                      borderRadius: BorderRadius.circular(4),
                    ),
                    child: const Text(
                      'NEW',
                      style: TextStyle(
                        color: Colors.white,
                        fontSize: 10,
                        fontWeight: FontWeight.bold,
                      ),
                    ),
                  ),
                ),
                const TextSpan(text: ' 查看最新内容。'),
              ],
            ),
          ),
        ),
      ),
    );
  }
}

3.3 Text.rich 与 RichText 的区别

Text.rich 和 RichText 都可以实现富文本，但存在以下关键区别：

对比项 Text.rich RichText
本质 Text 的命名构造函数，内部使用 RichText 独立的 Widget，直接渲染 InlineSpan
默认样式继承 自动继承 DefaultTextStyle 不会自动继承 DefaultTextStyle，需手动指定
文本方向 从 Directionality 自动获取 必须通过 textDirection 参数或 Directionality 提供
selectionColor 支持 支持
使用场景 大多数富文本场景 需要完全控制文本样式、不希望被外层默认样式影响时 
import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: const Text('Text.rich vs RichText')),
        body: const Padding(
          padding: EdgeInsets.all(16.0),
          // 设置默认文本样式
          child: DefaultTextStyle(
            style: TextStyle(fontSize: 20, color: Colors.green),
            child: Column(
              crossAxisAlignment: CrossAxisAlignment.start,
              children: [
                // Text.rich 会继承 DefaultTextStyle（绿色 20px）
                Text('普通 Text：继承了绿色大字'),
                SizedBox(height: 16),

                Text.rich(
                  TextSpan(
                    text: 'Text.rich：',
                    children: [
                      TextSpan(
                        text: '也继承了绿色大字',
                        style: TextStyle(fontWeight: FontWeight.bold),
                      ),
                    ],
                  ),
                ),
                SizedBox(height: 16),

                // 注意：这里由于 RichText 不继承 DefaultTextStyle，
                // 所以如果不手动指定，文字可能显示为默认的白色（不可见）
                // 必须在 TextSpan 中显式设置样式
              ],
            ),
          ),
        ),
      ),
    );
  }
}

最佳实践： 在大多数场景下，优先使用 Text.rich，因为它能自动继承 DefaultTextStyle，代码更简洁。只有在需要完全控制文本样式（不受外层默认样式影响）时，才使用 RichText。

4. SelectableText 组件

默认情况下，Flutter 中的 Text 组件是不可选中的。当用户需要复制文本内容时（如文章正文、错误信息、地址等），应使用 SelectableText。

4.1 基本用法

import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: const Text('SelectableText 示例')),
        body: const Padding(
          padding: EdgeInsets.all(16.0),
          child: Column(
            crossAxisAlignment: CrossAxisAlignment.start,
            children: [
              Text(
                '这是普通 Text，不可选中复制',
                style: TextStyle(fontSize: 16),
              ),
              SizedBox(height: 20),
              SelectableText(
                '这是 SelectableText，长按或拖动可以选中文字并复制。'
                '在需要用户复制内容的场景下非常实用，'
                '比如订单号、联系方式、文章内容等。',
                style: TextStyle(fontSize: 16),
              ),
            ],
          ),
        ),
      ),
    );
  }
}

4.2 onTap 回调与交互控制

SelectableText 支持 onTap 回调和 onSelectionChanged 回调，还可以通过 showCursor、cursorColor、cursorWidth 等属性自定义光标样式：

import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: const SelectableTextDemo(),
    );
  }
}

class SelectableTextDemo extends StatefulWidget {
  const SelectableTextDemo({super.key});

  @override
  State<SelectableTextDemo> createState() => _SelectableTextDemoState();
}

class _SelectableTextDemoState extends State<SelectableTextDemo> {
  String _selectedText = '';

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('SelectableText 交互')),
      body: Padding(
        padding: const EdgeInsets.all(16.0),
        child: Column(
          crossAxisAlignment: CrossAxisAlignment.start,
          children: [
            SelectableText(
              '订单编号：FL20260324001\n'
              '收货地址：北京市朝阳区某某路100号\n'
              '联系电话：138-0000-0000',
              style: const TextStyle(fontSize: 16, height: 1.8),
              showCursor: true,
              cursorColor: Colors.blue,
              cursorWidth: 2.0,
              onTap: () {
                ScaffoldMessenger.of(context).showSnackBar(
                  const SnackBar(
                    content: Text('提示：长按可选择文字'),
                    duration: Duration(seconds: 1),
                  ),
                );
              },
              onSelectionChanged: (selection, cause) {
                // 获取选中的文本
                const fullText = '订单编号：FL20260324001\n'
                    '收货地址：北京市朝阳区某某路100号\n'
                    '联系电话：138-0000-0000';
                if (selection.start != selection.end) {
                  setState(() {
                    _selectedText = fullText.substring(
                      selection.start,
                      selection.end,
                    );
                  });
                }
              },
            ),
            const SizedBox(height: 20),
            Text(
              '当前选中: $_selectedText',
              style: const TextStyle(
                fontSize: 14,
                color: Colors.grey,
              ),
            ),
          ],
        ),
      ),
    );
  }
}

4.3 SelectableText.rich —— 可选中的富文本

SelectableText 也有 rich 构造函数，用于创建可选中的富文本：

import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: const Text('SelectableText.rich')),
        body: Padding(
          padding: const EdgeInsets.all(16.0),
          child: SelectableText.rich(
            TextSpan(
              style: const TextStyle(fontSize: 16, color: Colors.black87),
              children: const [
                TextSpan(text: '这是一段'),
                TextSpan(
                  text: '可选中的富文本',
                  style: TextStyle(
                    color: Colors.blue,
                    fontWeight: FontWeight.bold,
                  ),
                ),
                TextSpan(text: '，你可以选中其中任意部分进行复制。'),
                TextSpan(
                  text: '\n\n红色加粗部分',
                  style: TextStyle(
                    color: Colors.red,
                    fontWeight: FontWeight.bold,
                    fontSize: 18,
                  ),
                ),
                TextSpan(text: '也可以被选中。'),
              ],
            ),
          ),
        ),
      ),
    );
  }
}

注意事项：

  • SelectableText 不支持 maxLines 和 overflow 属性（因为可选中的文本需要完整展示）
  • SelectableText 不支持 TextSpan 的 recognizer 属性，如需同时支持可选和可点击，需要考虑使用 SelectionArea 包裹 Text.rich 的方案
  • 在 Flutter 3.3+ 中，可以使用 SelectionArea 组件包裹任意子树来实现区域内文本可选

5. DefaultTextStyle 组件

DefaultTextStyle 是一个 InheritedWidget，它可以为子树中所有未显式设置样式的 Text 组件提供默认样式。

5.1 使用场景

在实际开发中，一个页面或模块内的多个文本常常共享相同的基础样式（字体大小、颜色等）。如果逐一设置会产生大量重复代码，此时可以使用 DefaultTextStyle 统一管理。

5.2 基本用法

import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: const Text('DefaultTextStyle 示例')),
        body: Padding(
          padding: const EdgeInsets.all(16.0),
          child: Column(
            crossAxisAlignment: CrossAxisAlignment.start,
            children: [
              // 使用 DefaultTextStyle 统一设置子树中的文本样式
              DefaultTextStyle(
                style: const TextStyle(
                  fontSize: 18,
                  color: Colors.indigo,
                  height: 1.6,
                ),
                child: const Column(
                  crossAxisAlignment: CrossAxisAlignment.start,
                  children: [
                    Text('这段文字继承了默认样式：靛蓝色、18px、1.6倍行高'),
                    SizedBox(height: 8),
                    Text('这段文字也继承了相同的默认样式'),
                    SizedBox(height: 8),
                    // 局部覆盖部分样式
                    Text(
                      '这段文字覆盖了颜色为红色',
                      style: TextStyle(color: Colors.red),
                    ),
                  ],
                ),
              ),
              const SizedBox(height: 24),

              // DefaultTextStyle 作用域之外的文字不受影响
              const Text(
                '这段文字在 DefaultTextStyle 作用域之外，使用应用级默认样式',
                style: TextStyle(fontSize: 14),
              ),
            ],
          ),
        ),
      ),
    );
  }
}

5.3 merge 行为

DefaultTextStyle.merge 可以在已有默认样式基础上合并新的样式，而不是完全替换。这在需要局部微调样式时非常有用：

import 'package:flutter/material.dart';

void main() => runApp(const MyApp());

class MyApp extends StatelessWidget {
  const MyApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      home: Scaffold(
        appBar: AppBar(title: const Text('DefaultTextStyle.merge')),
        body: const Padding(
          padding: EdgeInsets.all(16.0),
          child: DefaultTextStyle(
            // 外层默认样式：16px、黑色、普通粗细
            style: TextStyle(
              fontSize: 16,
              color: Colors.black87,
              fontWeight: FontWeight.normal,
            ),
            child: Column(
              crossAxisAlignment: CrossAxisAlignment.start,
              children: [
                Text('继承外层样式：16px、黑色、普通粗细'),
                SizedBox(height: 16),

                // merge 方式：在已有基础上叠加 bold，保留 fontSize 和 color
                DefaultTextStyle.merge(
                  style: TextStyle(fontWeight: FontWeight.bold),
                  child: Column(
                    crossAxisAlignment: CrossAxisAlignment.start,
                    children: [
                      Text('merge 后：16px、黑色、粗体（只覆盖了 fontWeight）'),
                      SizedBox(height: 8),

                      // 再次 merge：叠加颜色，保留 fontSize 和 bold
                      DefaultTextStyle.merge(
                        style: TextStyle(color: Colors.deepOrange),
                        child: Text(
                          '二次 merge：16px、深橙色、粗体',
                        ),
                      ),
                    ],
                  ),
                ),
              ],
            ),
          ),
        ),
      ),
    );
  }
}

5.4 DefaultTextStyle 的工作原理

当 Text 组件构建时，如果没有设置 style，或者 style 中某些属性为 null，它会通过 DefaultTextStyle.of(context) 获取最近的祖先 DefaultTextStyle 的样式，然后将二者合并。合并规则是：

  1. Text 自身的 style 中显式设置的属性优先
  2. 未设置的属性从 DefaultTextStyle 继承
  3. 如果连 DefaultTextStyle 也没有设置，则使用 Flutter 框架的默认值
// 等效的手动合并写法
final effectiveStyle = DefaultTextStyle.of(context).style.merge(widget.style);

注意事项：

  • DefaultTextStyle 的 softWrap、overflow、maxLines、textWidthBasis 也会被子 Text 继承
  • RichText 不会继承 DefaultTextStyle，这是 Text / Text.rich 与 RichText 的重要区别
  • MaterialApp 和 Scaffold 会默认在组件树中注入 DefaultTextStyle，这就是为什么即使不设置 style，Text 也有默认外观

6. 本课小结

本课系统学习了 Flutter 中的五大文本组件：

组件 核心用途 关键特点
Text 显示单一样式文本 最基础、最常用；支持 TextStyle、对齐、溢出处理
Text.rich 显示混合样式的富文本 基于 TextSpan 拼接；自动继承 DefaultTextStyle
RichText 底层富文本渲染 不继承 DefaultTextStyle；支持 WidgetSpan
SelectableText 可选中复制的文本 支持 onTap、onSelectionChanged；有 rich 构造
DefaultTextStyle 统一子树文本样式 InheritedWidget；支持 merge 合并行为

核心要点回顾：

  1. TextStyle 是控制文本外观的核心，熟练掌握 fontSize、fontWeight、color、height、letterSpacing、decoration 等属性
  2. 富文本使用 Text.rich + TextSpan，通过嵌套实现不同片段不同样式
  3. TextSpan 的 recognizer 可以实现部分文字可点击，但注意在 dispose 中释放
  4. RichText 与 Text.rich 的最大区别：是否自动继承 DefaultTextStyle
  5. SelectableText 用于需要复制文本的场景，但不支持 maxLines 和 overflow
  6. DefaultTextStyle 通过 InheritedWidget 机制向下传递样式，merge 方法可以叠加而非替换样式

练习一：商品价格标签

实现一个商品价格显示组件，要求：

  • 接着显示 "¥" 符号（红色 14px）

  • 接着显示价格小数部分（红色 14px）
  • 最后显示原价，带删除线（灰色小字）

提示：使用 Text.rich 和多个 TextSpan。

练习二：用户协议富文本

实现一段注册协议文字：
"点击注册即表示您已阅读并同意《服务条款》和《隐私政策》"

要求：

  • "《服务条款》"和"《隐私政策》"为蓝色，可点击（点击后弹出 SnackBar 提示）
  • 其余文字为默认样式
  • 注意正确管理 GestureRecognizer 的生命周期

练习三：文章预览卡片

使用 Card + Column 实现一个文章预览卡片：

  • 标题：最多 1 行，超出显示省略号，字体 18px 粗体
  • 摘要：最多 3 行，超出显示省略号，字体 14px，行高 1.6
  • 全文：使用 SelectableText，允许用户选中复制
  • 使用 DefaultTextStyle 为卡片内所有文本设置统一的字体和颜色

练习四：样式继承实验

创建一个嵌套的 DefaultTextStyle 结构，验证以下行为：

  1. 子 Text 不设置 style 时完全继承父级样式
  2. 子 Text 设置部分 style 属性时，只覆盖设置的属性，其余继承
  3. DefaultTextStyle.merge 和直接使用 DefaultTextStyle 的区别
  4. RichText 是否会继承 DefaultTextStyle（预期答案：不会）
