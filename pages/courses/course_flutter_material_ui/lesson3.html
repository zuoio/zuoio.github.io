第03课 排版系统 - TextTheme 与字体

课程系列：Flutter Material Design 系统学习
前置条件：已掌握 Text / TextStyle 的基本用法（组件课程第02课）
课程目标：全面掌握 Material 3 排版体系，学会为应用配置专业的品牌字体方案

1. Material 3 Type Scale 完整体系

排版（Typography）是 Material Design 3 的三大核心支柱之一（Color、Typography、Shape）。一套统一的排版系统能让应用在视觉上保持一致性和专业感。

Material 3 定义了 5 大类别、每类 3 个级别，共 15 种文本样式：

类别 定位 使用场景
Display 最大的展示文本 欢迎页、启动页的醒目大标题，营销页面的核心口号
Headline 页面级标题 页面顶部标题，重要内容区域标题
Title 组件级标题 卡片标题、对话框标题、AppBar 标题、列表分组标题
Body 正文内容 段落文本、列表项描述、表单说明
Label 小型辅助文本 按钮文字、标签页标签、Chip 文字、输入框标签、底部导航文字

每个类别有 Large、Medium、Small 三个级别，提供层次分明的视觉层级。

1.1 默认参数一览表

以下是 Material 3 中 15 种 TextStyle 的默认参数（基于 Roboto 字体）：

样式名称 字体大小 (fontSize) 字重 (fontWeight) 行高 (height) 字间距 (letterSpacing)
displayLarge 57 W400 (Regular) 1.12 (64px) -0.25
displayMedium 45 W400 (Regular) 1.16 (52px) 0
displaySmall 36 W400 (Regular) 1.22 (44px) 0
headlineLarge 32 W400 (Regular) 1.25 (40px) 0
headlineMedium 28 W400 (Regular) 1.29 (36px) 0
headlineSmall 24 W400 (Regular) 1.33 (32px) 0
titleLarge 22 W400 (Regular) 1.27 (28px) 0
titleMedium 16 W500 (Medium) 1.50 (24px) 0.15
titleSmall 14 W500 (Medium) 1.43 (20px) 0.1
bodyLarge 16 W400 (Regular) 1.50 (24px) 0.5
bodyMedium 14 W400 (Regular) 1.43 (20px) 0.25
bodySmall 12 W400 (Regular) 1.33 (16px) 0.4
labelLarge 14 W500 (Medium) 1.43 (20px) 0.1
labelMedium 12 W500 (Medium) 1.33 (16px) 0.5
labelSmall 11 W500 (Medium) 1.45 (16px) 0.5

注意事项：height 在 Flutter 中是一个倍数值，实际行高 = fontSize × height。例如 displayLarge 的行高为 57 × 1.12 ≈ 64px。

1.2 在代码中使用 TextTheme

Flutter 通过 Theme.of(context).textTheme 访问当前主题的所有文本样式：

1.3 各级别详细说明与使用建议

Display —— 展示文本

Display 是整套排版系统中最大的文本，用于需要强烈视觉冲击力的场景。

class DisplayExample extends StatelessWidget {
  const DisplayExample({super.key});

  @override
  Widget build(BuildContext context) {
    final textTheme = Theme.of(context).textTheme;

    return Column(
      crossAxisAlignment: CrossAxisAlignment.start,
      children: [
        // displayLarge (57px)：欢迎页、启动页的超大标题
        Text('欢迎回来', style: textTheme.displayLarge),

        // displayMedium (45px)：营销页面的核心口号
        Text('发现你的灵感', style: textTheme.displayMedium),

        // displaySmall (36px)：功能介绍页的大标题
        Text('全新体验', style: textTheme.displaySmall),
      ],
    );
  }
}

最佳实践：Display 文本应该简短有力，通常不超过一行。在小屏设备上可考虑降级使用（例如 displayLarge 降为 displayMedium）。

Headline —— 页面标题

Headline 适合作为页面或重要区域的标题，比 Display 更通用。

class HeadlineExample extends StatelessWidget {
  const HeadlineExample({super.key});

  @override

          SliverToBoxAdapter(

              padding: const EdgeInsets.all(16),

                crossAxisAlignment: CrossAxisAlignment.start,

                  const SizedBox(height: 24),

                  // headlineMedium (28px)：页面区域标题
                  Text('最近活动', style: textTheme.headlineMedium),
                  const SizedBox(height: 24),

                  // headlineSmall (24px)：子区域标题
                  Text('推荐内容', style: textTheme.headlineSmall),
                ],
              ),
            ),
          ),
        ],
      ),
    );
  }
}

Title —— 组件标题

Title 用于更小范围的标题，如卡片、对话框、列表分组。

class TitleExample extends StatelessWidget {

        ),
      ],
    );
  }
}

Body 是应用中使用最广泛的样式，承载主要的阅读内容。

Label —— 标签

Label 是最小的文本样式，用于按钮、标签、表单元素等交互组件。

class LabelExample extends StatelessWidget {
  const LabelExample({super.key});

  @override
  Widget build(BuildContext context) {
    final textTheme = Theme.of(context).textTheme;

    return Padding(
      padding: const EdgeInsets.all(16),
      child: Column(
        crossAxisAlignment: CrossAxisAlignment.start,
        children: [
          // labelLarge (14px, Medium)：按钮文字
          FilledButton(
            onPressed: () {},
            child: Text('提交订单', style: textTheme.labelLarge),
          ),
          const SizedBox(height: 12),

          // labelMedium (12px, Medium)：标签、底部导航文字
          Chip(label: Text('Flutter', style: textTheme.labelMedium)),
          const SizedBox(height: 12),

          // labelSmall (11px, Medium)：最小的辅助标签
          Text('必填', style: textTheme.labelSmall?.copyWith(
            color: Theme.of(context).colorScheme.error,
          )),
        ],
      ),
    );
  }
}

最佳实践：Material 组件（如 ElevatedButton、Chip）已经内置了正确的 Label 样式，通常不需要手动指定。只有自定义组件才需要显式使用 labelLarge 等样式。

2. TextTheme 自定义

2.1 通过 ThemeData 覆盖默认样式

在 ThemeData 中通过 textTheme 参数可以自定义所有 15 种文本样式：

import 'package:flutter/material.dart';

void main() {
  runApp(const CustomTextThemeApp());
}

class CustomTextThemeApp extends StatelessWidget {
  const CustomTextThemeApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: '自定义 TextTheme',
      theme: ThemeData(
        useMaterial3: true,
        colorSchemeSeed: Colors.indigo,
        // 自定义 TextTheme
        textTheme: const TextTheme(
          // 只覆盖需要修改的样式，其余保持默认
          displayLarge: TextStyle(
            fontSize: 60,
            fontWeight: FontWeight.w700,
            letterSpacing: -1.5,
            height: 1.1,
          ),
          headlineMedium: TextStyle(
            fontSize: 30,
            fontWeight: FontWeight.w600,
            letterSpacing: -0.5,
          ),
          bodyLarge: TextStyle(
            fontSize: 17,
            fontWeight: FontWeight.w400,
            height: 1.6, // 增加行高，提升可读性
            letterSpacing: 0.3,
          ),
          bodyMedium: TextStyle(
            fontSize: 15,
            fontWeight: FontWeight.w400,
            height: 1.55,
            letterSpacing: 0.2,
          ),
          labelLarge: TextStyle(
            fontSize: 15,
            fontWeight: FontWeight.w600,
            letterSpacing: 0.5,
          ),
        ),
      ),
      home: const CustomTextThemePage(),
    );
  }
}

class CustomTextThemePage extends StatelessWidget {
  const CustomTextThemePage({super.key});

  @override
  Widget build(BuildContext context) {
    final textTheme = Theme.of(context).textTheme;

    return Scaffold(
      appBar: AppBar(title: const Text('自定义排版')),
      body: SingleChildScrollView(
        padding: const EdgeInsets.all(16),
        child: Column(
          crossAxisAlignment: CrossAxisAlignment.start,
          children: [
            Text('品牌标题', style: textTheme.displayLarge),
            const SizedBox(height: 16),
            Text('区域标题', style: textTheme.headlineMedium),
            const SizedBox(height: 16),
            Text(
              '这是一段正文内容，使用了自定义的 bodyLarge 样式，'
              '行高设置为 1.6 倍以提升段落文本的可读性。',
              style: textTheme.bodyLarge,
            ),
            const SizedBox(height: 16),
            FilledButton(
              onPressed: () {},
              child: const Text('立即体验'),
            ),
          ],
        ),
      ),
    );
  }
}

2.2 使用 copyWith 局部覆盖

当只需要修改少数几个样式时，使用 copyWith 更简洁：

ThemeData buildTheme() {
  // 获取默认的 Material 3 主题
  final baseTheme = ThemeData(
    useMaterial3: true,
    colorSchemeSeed: Colors.teal,
  );

  return baseTheme.copyWith(
    textTheme: baseTheme.textTheme.copyWith(
      // 只修改标题字重，其余所有属性保持不变
      displayLarge: baseTheme.textTheme.displayLarge?.copyWith(
        fontWeight: FontWeight.w700,
      ),
      headlineLarge: baseTheme.textTheme.headlineLarge?.copyWith(
        fontWeight: FontWeight.w700,
      ),
      // 修改正文行高
      bodyLarge: baseTheme.textTheme.bodyLarge?.copyWith(
        height: 1.65,
      ),
      bodyMedium: baseTheme.textTheme.bodyMedium?.copyWith(
        height: 1.6,
      ),
    ),
  );
}

2.3 apply 方法批量调整

TextTheme.apply() 可以对整个 TextTheme 的所有样式进行统一的缩放或调整：

ThemeData buildScaledTheme(BuildContext context) {
  final baseTheme = ThemeData(
    useMaterial3: true,
    colorSchemeSeed: Colors.purple,
  );

  return baseTheme.copyWith(
    textTheme: baseTheme.textTheme.apply(
      // 所有字号乘以 1.1（整体放大 10%）
      fontSizeFactor: 1.1,
      // 所有字号再加 0（不额外增减）
      fontSizeDelta: 0,
      // 统一设置字体族
      fontFamily: 'NotoSansSC',
      // 统一设置文本颜色
      bodyColor: Colors.grey[800],
      displayColor: Colors.grey[900],
    ),
  );
}

注意事项：apply 中的 bodyColor 会应用到 Body 和 Label 级别的样式，displayColor 会应用到 Display、Headline 和 Title 级别的样式。

3. Google Fonts 集成

3.1 安装 google_fonts 包

在 pubspec.yaml 中添加依赖：

dependencies:
  flutter:
    sdk: flutter
  google_fonts: ^6.2.1

然后执行安装：

flutter pub get

3.2 基本使用方式

google_fonts 包提供了两种核心用法：

import 'package:flutter/material.dart';
import 'package:google_fonts/google_fonts.dart';

void main() {
  runApp(const GoogleFontsApp());
}

class GoogleFontsApp extends StatelessWidget {
  const GoogleFontsApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Google Fonts 演示',
      theme: ThemeData(
        useMaterial3: true,
        colorSchemeSeed: Colors.blue,
      ),
      home: const GoogleFontsPage(),
    );
  }
}

class GoogleFontsPage extends StatelessWidget {
  const GoogleFontsPage({super.key});

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('Google Fonts')),
      body: SingleChildScrollView(
        padding: const EdgeInsets.all(16),
        child: Column(
          crossAxisAlignment: CrossAxisAlignment.start,
          children: [
            // === 方式一：用于单个 Text 组件 ===
            // 使用命名构造函数（推荐，有代码提示）
            Text(
              'Poppins Bold Title',
              style: GoogleFonts.poppins(
                fontSize: 28,
                fontWeight: FontWeight.w700,
              ),
            ),

            Text(
              'Roboto Mono Code Style',
              style: GoogleFonts.robotoMono(
                fontSize: 16,
                fontWeight: FontWeight.w400,
              ),
            ),
            const SizedBox(height: 12),

              'Lato Regular Text',
              style: GoogleFonts.getFont(
                'Lato',
                fontSize: 16,
                fontWeight: FontWeight.w400,
              ),
            ),
            const SizedBox(height: 24),

            // === 方式二：生成完整的 TextTheme ===
            // 详见 3.3 节

              '下一节展示如何将 Google Fonts 应用到整个 TextTheme',

  }
}

3.3 将 Google Fonts 应用到完整 TextTheme

这是最推荐的做法——将 Google Fonts 集成到 ThemeData 中，让所有 Material 组件自动使用：

import 'package:flutter/material.dart';
import 'package:google_fonts/google_fonts.dart';

void main() {
  runApp(const BrandFontApp());
}

class BrandFontApp extends StatelessWidget {
  const BrandFontApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: '品牌字体方案',
      theme: _buildTheme(),
      home: const BrandFontPage(),
    );
  }

  ThemeData _buildTheme() {
    final baseTheme = ThemeData(
      useMaterial3: true,
      colorSchemeSeed: Colors.deepPurple,
    );

    return baseTheme.copyWith(
      // 方法一：将整个 TextTheme 替换为 Google Fonts
      textTheme: GoogleFonts.notoSansScTextTheme(baseTheme.textTheme),

      // 方法二：分别设置（更灵活，可混合字体）
      // textTheme: GoogleFonts.poppinsTextTheme(baseTheme.textTheme).copyWith(
      //   bodyLarge: GoogleFonts.notoSansSc(
      //     textStyle: baseTheme.textTheme.bodyLarge,
      //   ),
      //   bodyMedium: GoogleFonts.notoSansSc(
      //     textStyle: baseTheme.textTheme.bodyMedium,
      //   ),
      // ),
    );
  }
}

class BrandFontPage extends StatelessWidget {
  const BrandFontPage({super.key});

    final textTheme = Theme.of(context).textTheme;

3.4 离线字体缓存

默认情况下，google_fonts 会在运行时从网络下载字体文件。为了确保离线可用和更快的加载速度，建议将字体文件预置到项目中：

步骤一：将 .ttf 字体文件放入 assets/fonts/ 目录：

your_app/
  assets/
    fonts/
      NotoSansSC-Regular.ttf
      NotoSansSC-Medium.ttf
      NotoSansSC-Bold.ttf
      Poppins-Regular.ttf
      Poppins-Medium.ttf
      Poppins-Bold.ttf

步骤二：在 pubspec.yaml 中注册字体资产：

flutter:
  assets:
    - assets/fonts/

步骤三：禁用网络下载（可选，推荐在生产环境中禁用）：

import 'package:google_fonts/google_fonts.dart';

void main() {
  // 禁止从网络下载字体，只使用本地预置的字体文件
  GoogleFonts.config.allowRuntimeFetching = false;

  runApp(const MyApp());
}

注意事项：

  1. google_fonts 包会优先查找 assets/fonts/ 目录中的字体文件，找不到才会从网络下载。
  2. 字体文件名必须与 Google Fonts 上的原始文件名一致，否则无法正确匹配。
  3. 中文字体文件通常较大（Noto Sans SC 约 8~10MB），要注意对应用体积的影响。

3.5 中文字体注意事项

中文字体在 Flutter 中的使用有以下特殊之处：

import 'package:flutter/material.dart';
import 'package:google_fonts/google_fonts.dart';

class ChineseFontDemo extends StatelessWidget {
  const ChineseFontDemo({super.key});

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      body: SingleChildScrollView(
        padding: const EdgeInsets.all(16),
        child: Column(
          crossAxisAlignment: CrossAxisAlignment.start,
          children: [
            // 推荐的中文字体
            Text(
              '思源黑体（Noto Sans SC）',
              style: GoogleFonts.notoSansSc(fontSize: 20),
            ),
            const SizedBox(height: 8),
            Text(
              '思源宋体（Noto Serif SC）',
              style: GoogleFonts.notoSerifSc(fontSize: 20),
            ),
            const SizedBox(height: 8),

            // 中文字体行高需要特别关注
            Text(
              '中文排版建议使用 1.6~1.8 的行高倍数，'
              '因为中文字符比西文更紧凑，需要更大的行间距来提升可读性。'
              '尤其是在长段落中，合适的行高对用户阅读体验至关重要。',
              style: GoogleFonts.notoSansSc(
                fontSize: 16,
                height: 1.7, // 中文推荐行高
              ),
            ),
          ],
        ),
      ),
    );
  }
}

最佳实践：

  1. 推荐的中文字体：Noto Sans SC（思源黑体）适合正文和 UI 文本；Noto Serif SC（思源宋体）适合需要衬线效果的内容型应用。
  2. 行高设置：中文正文建议 height 设为 1.61.8，比英文 1.41.5 的默认行高更大。
  3. 字间距：中文通常不需要额外的 letterSpacing，设为 0 即可。
  4. 字体大小：中文最小可辨认字号约为 12px，正文建议至少 14px。
  5. 字体文件体积：中文字体包含数万个字形，文件远大于西文字体。生产环境中应预置到项目而非运行时下载。

4. 可变字体（Variable Fonts）

4.1 什么是可变字体

传统字体的每个字重（Regular、Bold 等）是一个独立的文件。可变字体将所有变体打包在一个文件中，通过轴（Axis） 实现平滑的属性过渡。

常见的可变字体轴：

轴标签 名称 描述 取值范围（典型）
wght Weight 字重 100 ~ 900
wdth Width 字宽 75 ~ 125
ital Italic 斜体 0 或 1
slnt Slant 倾斜角度 -90 ~ 90
opsz Optical Size 光学尺寸 因字体而异

4.2 在 Flutter 中使用可变字体

Flutter 通过 FontVariation 类支持可变字体：

import 'package:flutter/material.dart';

}

class VariableFontApp extends StatelessWidget {

          ],

4.3 注册可变字体

在 pubspec.yaml 中注册可变字体文件：

flutter:
  fonts:
    - family: RobotoFlex
      fonts:
        - asset: assets/fonts/RobotoFlex-VariableFont.ttf

注意事项：可变字体只需注册一个文件，不需要像传统字体那样为每个字重注册单独的文件。

4.4 可变字体动画

可变字体的一个强大优势是可以对字体属性做平滑动画：

import 'package:flutter/material.dart';

class AnimatedVariableFontDemo extends StatefulWidget {
  const AnimatedVariableFontDemo({super.key});

  @override
  State<AnimatedVariableFontDemo> createState() =>
      _AnimatedVariableFontDemoState();
}

class _AnimatedVariableFontDemoState extends State<AnimatedVariableFontDemo>
    with SingleTickerProviderStateMixin {
  late AnimationController _controller;
  late Animation<double> _weightAnimation;

  @override
  void initState() {
    super.initState();
    _controller = AnimationController(
      duration: const Duration(seconds: 2),
      vsync: this,
    )..repeat(reverse: true);

    _weightAnimation = Tween<double>(begin: 100, end: 900).animate(
      CurvedAnimation(parent: _controller, curve: Curves.easeInOut),
    );
  }

  @override
  void dispose() {
    _controller.dispose();
    super.dispose();
  }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('可变字体动画')),
      body: Center(
        child: AnimatedBuilder(
          animation: _controller,
          builder: (context, child) {
            return Text(
              'Weight Animation',
              style: TextStyle(
                fontFamily: 'RobotoFlex',
                fontSize: 36,
                fontVariations: [
                  FontVariation('wght', _weightAnimation.value),
                ],
              ),
            );
          },
        ),
      ),
    );
  }
}

最佳实践：可变字体比多个静态字体文件更节省空间，且支持平滑过渡。推荐在需要精细控制字重或做字体动画的场景中使用。

5. 文本无障碍（Accessibility）

确保文本在不同用户偏好下都能良好显示，是排版系统设计的重要一环。

5.1 TextScaler —— 文本缩放

从 Flutter 3.16 开始，textScaleFactor（double）被 textScaler（TextScaler）取代，提供更灵活的非线性缩放能力。

import 'package:flutter/material.dart';

void main() {
  runApp(const AccessibilityApp());
}

class AccessibilityApp extends StatelessWidget {
  const AccessibilityApp({super.key});

  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: '文本无障碍',
      theme: ThemeData(useMaterial3: true),
      home: const TextScalerPage(),
    );
  }
}

class TextScalerPage extends StatefulWidget {
  const TextScalerPage({super.key});

  @override
  State<TextScalerPage> createState() => _TextScalerPageState();
}

class _TextScalerPageState extends State<TextScalerPage> {
  double _scaleFactor = 1.0;

  @override
  Widget build(BuildContext context) {
    // 获取系统文本缩放设置
    final systemTextScaler = MediaQuery.textScalerOf(context);

    return Scaffold(
      appBar: AppBar(title: const Text('文本缩放')),
      body: MediaQuery(
        // 用自定义的 TextScaler 覆盖子树
        data: MediaQuery.of(context).copyWith(
          textScaler: TextScaler.linear(_scaleFactor),
        ),
        child: SingleChildScrollView(
          padding: const EdgeInsets.all(16),
          child: Column(
            crossAxisAlignment: CrossAxisAlignment.start,
            children: [
              // 显示当前缩放信息
              Text(
                '系统缩放: ${systemTextScaler.scale(1.0).toStringAsFixed(2)}x\n'
                '当前预览: ${_scaleFactor.toStringAsFixed(2)}x',
                style: Theme.of(context).textTheme.bodySmall,
              ),
              const SizedBox(height: 16),

              // 缩放滑块（不受子树 MediaQuery 影响）
              MediaQuery(
                data: MediaQuery.of(context),
                child: Slider(
                  value: _scaleFactor,
                  min: 0.5,
                  max: 3.0,
                  divisions: 25,
                  label: '${_scaleFactor.toStringAsFixed(2)}x',
                  onChanged: (v) => setState(() => _scaleFactor = v),
                ),
              ),
              const Divider(),

              // 以下文本受自定义 TextScaler 影响
              Text(
                '标题文本',
                style: Theme.of(context).textTheme.headlineMedium,
              ),
              const SizedBox(height: 8),
              Text(
                '这段正文文本会随着缩放滑块的变化而改变大小。'
                '在无障碍测试中，需要确保文本在 1.0x 到 2.0x 范围内都不会溢出或重叠。',
                style: Theme.of(context).textTheme.bodyLarge,
              ),
              const SizedBox(height: 16),
              FilledButton(
                onPressed: () {},
                child: const Text('按钮文字'),
              ),
            ],
          ),
        ),
      ),
    );
  }
}

5.2 使用 MediaQuery.textScalerOf

在需要根据文本缩放比例调整布局的场景中，使用 MediaQuery.textScalerOf(context) 获取当前的 TextScaler：

class AdaptiveTextLayout extends StatelessWidget {
  const AdaptiveTextLayout({super.key});

  @override

        child: Column(
          crossAxisAlignment: CrossAxisAlignment.start,

                    style: Theme.of(context).textTheme.titleMedium,
                  ),
                ],
              )
            else
              Row(
                children: [
                  const Icon(Icons.info, size: 24),
                  const SizedBox(width: 8),
                  Text(
                    '信息提示',
                    style: Theme.of(context).textTheme.titleMedium,
                  ),
                ],
              ),
          ],
        ),
      ),
    );
  }
}

5.3 防止文本溢出

在大字号设置下文本溢出是最常见的无障碍问题。以下是预防策略：

class OverflowPreventionDemo extends StatelessWidget {
  const OverflowPreventionDemo({super.key});

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('防止溢出')),
      body: Padding(
        padding: const EdgeInsets.all(16),
        child: Column(
          crossAxisAlignment: CrossAxisAlignment.start,
          children: [
            // ❌ 错误：固定高度 + 单行，大字号会截断
            SizedBox(
              height: 30,
              child: Text(
                '这段文字在大字号下会被截断',
                style: Theme.of(context).textTheme.bodyLarge,
                overflow: TextOverflow.ellipsis,
              ),
            ),
            const SizedBox(height: 16),

            // ✅ 正确：不设固定高度，让文本自然换行
            Text(
              '这段文字在大字号下会自动换行，不会被截断。'
              '使用弹性布局让内容自适应。',
              style: Theme.of(context).textTheme.bodyLarge,
            ),
            const SizedBox(height: 24),

            // ✅ 正确：用 Flexible/Expanded 处理 Row 内文本
            Row(
              children: [
                const Icon(Icons.star),
                const SizedBox(width: 8),
                Expanded(
                  child: Text(
                    '使用 Expanded 包裹文本，确保在空间不足时换行而非溢出',
                    style: Theme.of(context).textTheme.bodyMedium,
                  ),
                ),
              ],
            ),
            const SizedBox(height: 24),

            // ✅ 正确：使用 maxLines + overflow 优雅截断
            Text(
              '使用 maxLines 限制最大行数，配合 overflow 属性在内容过多时显示省略号，'
              '提供视觉线索告知用户内容被截断，同时保持布局稳定。',
              style: Theme.of(context).textTheme.bodyMedium,
              maxLines: 2,
              overflow: TextOverflow.ellipsis,
            ),
          ],
        ),
      ),
    );
  }
}

最佳实践：

  1. 避免给文本容器设置固定高度，使用弹性布局让高度自适应。
  2. 在 Row 中显示文本时，用 Expanded 或 Flexible 包裹 Text 组件。
  3. 使用 textScaler 缩放到 2.0x 进行测试，确保所有页面都不会溢出。
  4. 在 Semantics 组件中为装饰性文本提供 excludeSemantics: true，避免屏幕阅读器读取无意义内容。
  5. 确保文本与背景色之间的对比度符合 WCAG 标准（普通文本 ≥ 4.5:1，大号文本 ≥ 3:1）。

import 'package:flutter/material.dart';
import 'package:google_fonts/google_fonts.dart';

/// 应用排版系统
/// 英文标题使用 Poppins，中文正文使用 Noto Sans SC
class AppTypography {
  AppTypography._();

  /// 构建完整的 TextTheme
  static TextTheme buildTextTheme({required Brightness brightness}) {
    final baseColor = brightness == Brightness.light
        ? const Color(0xFF1A1A1A)
        : const Color(0xFFF5F5F5);

    final secondaryColor = brightness == Brightness.light
        ? const Color(0xFF666666)
        : const Color(0xFFBBBBBB);

    // Display 和 Headline 使用 Poppins（英文标题字体）
    final displayLarge = GoogleFonts.poppins(
      fontSize: 57,
      fontWeight: FontWeight.w400,
      height: 1.12,
      letterSpacing: -0.25,
      color: baseColor,
    );

    final displayMedium = GoogleFonts.poppins(
      fontSize: 45,
      fontWeight: FontWeight.w400,
      height: 1.16,
      color: baseColor,
    );

    final displaySmall = GoogleFonts.poppins(
      fontSize: 36,
      fontWeight: FontWeight.w400,
      height: 1.22,
      color: baseColor,
    );

    final headlineLarge = GoogleFonts.poppins(
      fontSize: 32,
      fontWeight: FontWeight.w600,
      height: 1.25,
      color: baseColor,
    );

    final headlineMedium = GoogleFonts.poppins(
      fontSize: 28,
      fontWeight: FontWeight.w600,
      height: 1.29,
      color: baseColor,
    );

    final headlineSmall = GoogleFonts.poppins(
      fontSize: 24,
      fontWeight: FontWeight.w600,
      height: 1.33,
      color: baseColor,
    );

    // Title 使用 Poppins（组件标题）
    final titleLarge = GoogleFonts.poppins(
      fontSize: 22,
      fontWeight: FontWeight.w500,
      height: 1.27,
      color: baseColor,
    );

    final titleMedium = GoogleFonts.poppins(
      fontSize: 16,
      fontWeight: FontWeight.w500,
      height: 1.50,
      letterSpacing: 0.15,
      color: baseColor,
    );

    final titleSmall = GoogleFonts.poppins(
      fontSize: 14,
      fontWeight: FontWeight.w500,
      height: 1.43,
      letterSpacing: 0.1,
      color: baseColor,
    );

    // Body 使用 Noto Sans SC（中文正文字体）
    final bodyLarge = GoogleFonts.notoSansSc(
      fontSize: 16,
      fontWeight: FontWeight.w400,
      height: 1.65, // 中文行高适当加大
      letterSpacing: 0.3,
      color: baseColor,
    );

    final bodyMedium = GoogleFonts.notoSansSc(
      fontSize: 14,
      fontWeight: FontWeight.w400,
      height: 1.6,
      letterSpacing: 0.2,
      color: baseColor,
    );

    final bodySmall = GoogleFonts.notoSansSc(
      fontSize: 12,
      fontWeight: FontWeight.w400,
      height: 1.5,
      letterSpacing: 0.3,
      color: secondaryColor,
    );

    // Label 使用 Noto Sans SC（中文标签字体）
    final labelLarge = GoogleFonts.notoSansSc(
      fontSize: 14,
      fontWeight: FontWeight.w500,
      height: 1.43,
      letterSpacing: 0.1,
      color: baseColor,
    );

    final labelMedium = GoogleFonts.notoSansSc(
      fontSize: 12,
      fontWeight: FontWeight.w500,
      height: 1.33,
      letterSpacing: 0.4,
      color: baseColor,
    );

    final labelSmall = GoogleFonts.notoSansSc(
      fontSize: 11,
      fontWeight: FontWeight.w500,
      height: 1.45,
      letterSpacing: 0.4,
      color: secondaryColor,
    );

    return TextTheme(
      displayLarge: displayLarge,
      displayMedium: displayMedium,
      displaySmall: displaySmall,
      headlineLarge: headlineLarge,
      headlineMedium: headlineMedium,
      headlineSmall: headlineSmall,
      titleLarge: titleLarge,
      titleMedium: titleMedium,
      titleSmall: titleSmall,
      bodyLarge: bodyLarge,
      bodyMedium: bodyMedium,
      bodySmall: bodySmall,
      labelLarge: labelLarge,
      labelMedium: labelMedium,
      labelSmall: labelSmall,
    );
  }
}

import 'package:flutter/material.dart';
import 'app_typography.dart';

class AppTheme {
  AppTheme._();

    final textTheme = AppTypography.buildTextTheme(
      brightness: Brightness.light,

    return ThemeData(
      useMaterial3: true,
      brightness: Brightness.light,
      colorSchemeSeed: const Color(0xFF6750A4),
      textTheme: textTheme,
    );

    final textTheme = AppTypography.buildTextTheme(
      brightness: Brightness.dark,
    );

    return ThemeData(
      useMaterial3: true,
      brightness: Brightness.dark,
      colorSchemeSeed: const Color(0xFF6750A4),
      textTheme: textTheme,
    );
  }
}

7.2 创建可复用的文本样式工具方法

extension TextThemeExtensions on TextTheme {
  /// 正文加粗样式
  TextStyle? get bodyLargeBold => bodyLarge?.copyWith(
        fontWeight: FontWeight.w700,
      );

  /// 带删除线的正文样式（用于折扣价等场景）
  TextStyle? get bodyMediumStrikethrough => bodyMedium?.copyWith(
        decoration: TextDecoration.lineThrough,
        color: bodyMedium?.color?.withValues(alpha: 0.5),
      );

  /// 带下划线的链接样式
  TextStyle? get bodyMediumLink => bodyMedium?.copyWith(
        decoration: TextDecoration.underline,
      );
}

8. 本课小结

核心概念回顾

概念 要点
Type Scale Material 3 定义了 5 类 × 3 级 = 15 种文本样式，覆盖从展示标题到小型标签的所有场景
Display 最大的展示文本（57/45/36px），用于欢迎页、营销页面
Headline 页面级标题（32/28/24px），用于页面标题和重要区域标题
Title 组件级标题（22/16/14px），用于卡片、对话框、AppBar 标题
Body 正文内容（16/14/12px），用于段落文本和描述信息
Label 小型辅助文本（14/12/11px），用于按钮、标签、导航文字
TextTheme 自定义 通过 ThemeData.textTheme 覆盖默认样式，支持 copyWith 和 apply
Google Fonts 通过 google_fonts 包集成 1000+ 字体，支持单独使用和全局 TextTheme 替换
可变字体 使用 FontVariation 实现精细的字重、字宽控制和平滑动画
文本无障碍 使用 TextScaler 替代 textScaleFactor，确保文本在放大后不溢出

关键 API 速查

// 获取 TextTheme
Theme.of(context).textTheme

// Google Fonts 单个样式
GoogleFonts.poppins(fontSize: 24, fontWeight: FontWeight.w600)

// Google Fonts 完整 TextTheme
GoogleFonts.notoSansScTextTheme(baseTextTheme)

// 可变字体
TextStyle(fontVariations: [FontVariation('wght', 600)])

// 文本缩放
MediaQuery.textScalerOf(context)
TextScaler.linear(1.5)

9. 练习题

练习一：Type Scale 展示器（基础）

创建一个页面，以表格或列表形式展示全部 15 种 TextStyle，每行包括：

  • 样式名称
  • 字号 / 字重 / 行高参数
  • 预览文本

要求：使用 ListView.builder 实现，数据从 TextTheme 中动态读取。

练习二：Google Fonts 字体预览器（进阶）

创建一个字体预览工具：

  1. 提供一个 TextField 让用户输入预览文本
  2. 使用 DropdownMenu 让用户选择不同的 Google Fonts 字体（至少 5 种）
  3. 提供字号和字重的 Slider 控制
  4. 实时展示选中字体的渲染效果

练习三：品牌排版方案实战（综合）

为一个虚拟的"在线读书"应用配置完整的排版方案：

  1. 选择一套英文 + 中文字体组合
  2. 自定义所有 15 种 TextStyle
  3. 创建一个包含以下元素的展示页面：
    • 页面大标题（Display 或 Headline）
    • 书籍卡片（Title + Body + Label）
    • 文章阅读区域（Body，优化行高和间距以提升阅读体验）
    • 操作按钮组（Label）
  4. 支持亮色/暗色主题切换
  5. 用 TextScaler 测试在 2.0x 缩放下页面无溢出

练习四：可变字体交互器（挑战）

  1. 下载一款支持多轴的可变字体（如 Roboto Flex）
  2. 创建一个页面，包含多个 Slider 分别控制字重（wght）、字宽（wdth）、光学尺寸（opsz）等轴
  3. 实时预览字体变化效果
  4. 添加一个动画按钮，点击后字重在 100~900 之间平滑动画循环
