附录A：MongoDB 常用 Shell 命令速查表

1. 数据库操作

命令 语法 说明
show dbs show dbs 列出所有数据库及其占用空间
use use <数据库名> 切换到指定数据库，若不存在则在首次写入时自动创建
db.dropDatabase() db.dropDatabase() 删除当前数据库（需先 use 切换到目标库）
db.stats() db.stats() 查看当前数据库的统计信息（文档数、存储大小等）
db.getName() db.getName() 返回当前数据库名称

2. 集合操作

命令 语法 说明
show collections show collections 列出当前数据库中的所有集合
db.createCollection() db.createCollection("集合名", {选项}) 显式创建集合，可指定 capped、大小等选项
drop() db.集合名.drop() 删除指定集合及其所有文档和索引
renameCollection() db.集合名.renameCollection("新名称") 重命名集合
stats() db.集合名.stats() 查看集合的详细统计信息

3. 插入操作

命令 语法 说明
insertOne() db.集合名.insertOne({文档}) 插入单个文档，返回插入的 _id
insertMany() db.集合名.insertMany([{文档1}, {文档2}]) 批量插入多个文档，返回所有插入的 _id 数组

4. 查询操作

命令 语法 说明
find() db.集合名.find({条件}, {投影}) 查询匹配条件的所有文档，返回游标
findOne() db.集合名.findOne({条件}, {投影}) 查询匹配条件的第一个文档
countDocuments() db.集合名.countDocuments({条件}) 精确统计匹配条件的文档数量
estimatedDocumentCount() db.集合名.estimatedDocumentCount() 基于元数据快速估算集合文档总数
distinct() db.集合名.distinct("字段名", {条件}) 返回指定字段的去重值数组

5. 更新操作

命令 语法 说明
updateOne() db.集合名.updateOne({条件}, {更新操作}) 更新匹配条件的第一个文档
updateMany() db.集合名.updateMany({条件}, {更新操作}) 更新所有匹配条件的文档
replaceOne() db.集合名.replaceOne({条件}, {新文档}) 整体替换匹配的第一个文档（不能包含更新操作符）
findOneAndUpdate() db.集合名.findOneAndUpdate({条件}, {更新}, {选项}) 查找并更新文档，可通过 returnDocument 选项返回更新前或更新后的文档
findOneAndReplace() db.集合名.findOneAndReplace({条件}, {新文档}, {选项}) 查找并整体替换文档

6. 删除操作

命令 语法 说明
deleteOne() db.集合名.deleteOne({条件}) 删除匹配条件的第一个文档
deleteMany() db.集合名.deleteMany({条件}) 删除所有匹配条件的文档
findOneAndDelete() db.集合名.findOneAndDelete({条件}, {选项}) 查找并删除文档，返回被删除的文档

7. 查询操作符

比较操作符：

操作符 语法 说明
$eq {字段: {$eq: 值}} 等于，等价于 {字段: 值}
$ne {字段: {$ne: 值}} 不等于
$gt {字段: {$gt: 值}} 大于
$gte {字段: {$gte: 值}} 大于等于
$lt {字段: {$lt: 值}} 小于
$lte {字段: {$lte: 值}} 小于等于
$in {字段: {$in: [值1, 值2]}} 字段值在给定数组中
$nin {字段: {$nin: [值1, 值2]}} 字段值不在给定数组中

逻辑操作符：

操作符 语法 说明
$and {$and: [{条件1}, {条件2}]} 逻辑与，所有条件都满足
$or {$or: [{条件1}, {条件2}]} 逻辑或，至少一个条件满足
$not {字段: {$not: {操作符表达式}}} 逻辑非，对单个条件取反
$nor {$nor: [{条件1}, {条件2}]} 所有条件都不满足

元素操作符：

操作符 语法 说明
$exists {字段: {$exists: true}} 判断字段是否存在
$type {字段: {$type: "string"}} 匹配字段的 BSON 类型

求值操作符：

操作符 语法 说明
$regex {字段: {$regex: /模式/, $options: "i"}} 正则表达式匹配

数组操作符：

操作符 语法 说明
$all {字段: {$all: [值1, 值2]}} 数组字段包含所有指定元素
$elemMatch {字段: {$elemMatch: {条件}}} 数组中至少有一个元素同时满足所有条件
$size {字段: {$size: 数量}} 匹配数组长度等于指定值的文档

8. 更新操作符

字段操作符：

操作符 语法 说明
$set {$set: {字段: 新值}} 设置字段值，字段不存在则创建
$unset {$unset: {字段: ""}} 删除指定字段
$inc {$inc: {字段: 增量}} 将字段值增加指定数值（可为负数）
$mul {$mul: {字段: 乘数}} 将字段值乘以指定数值
$rename {$rename: {旧名: "新名"}} 重命名字段
$min {$min: {字段: 值}} 仅当指定值小于当前值时更新
$max {$max: {字段: 值}} 仅当指定值大于当前值时更新

数组操作符：

操作符 语法 说明
$push {$push: {数组字段: 值}} 向数组末尾添加元素
$pull {$pull: {数组字段: 值或条件}} 从数组中移除匹配的元素
$addToSet {$addToSet: {数组字段: 值}} 向数组添加元素（自动去重）
$pop {$pop: {数组字段: 1或-1}} 移除数组首元素（-1）或尾元素（1）

数组修饰符：

操作符 语法 说明
$each {$push: {数组字段: {$each: [值1, 值2]}}} 与 $push 或 $addToSet 配合批量添加
$position {$push: {数组字段: {$each: [值], $position: 0}}} 指定 $push 插入的位置索引
$slice {$push: {数组字段: {$each: [值], $slice: 5}}} 限制 $push 后数组的最大长度

9. 索引操作

命令 语法 说明
createIndex() db.集合名.createIndex({字段: 1}, {选项}) 创建索引，1 为升序，-1 为降序
dropIndex() db.集合名.dropIndex("索引名"或{键}) 删除指定索引
dropIndexes() db.集合名.dropIndexes() 删除集合上除 _id 外的所有索引
getIndexes() db.集合名.getIndexes() 列出集合上的所有索引
explain() db.集合名.find({条件}).explain("executionStats") 分析查询执行计划和性能指标

10. 聚合操作

命令 语法 说明
aggregate() db.集合名.aggregate([{阶段1}, {阶段2}, ...]) 执行聚合管道，按顺序处理各阶段

常用聚合管道阶段示例：

db.orders.aggregate([
  { $match: { status: "completed" } },
  { $group: { _id: "$customerId", total: { $sum: "$amount" } } },
  { $sort: { total: -1 } },
  { $limit: 10 }
])

11. 用户管理

命令 语法 说明
createUser() db.createUser({user:"用户名", pwd:"密码", roles:[{role:"角色", db:"库名"}]}) 创建数据库用户
dropUser() db.dropUser("用户名") 删除指定用户
updateUser() db.updateUser("用户名", {更新内容}) 更新用户信息（密码、角色等）
grantRolesToUser() db.grantRolesToUser("用户名", [{role:"角色", db:"库名"}]) 为用户授予额外角色

12. 备份恢复

命令 语法 说明

mongorestore mongorestore --uri="连接串" 备份目录 从 BSON 备份文件恢复数据库
mongoexport mongoexport --uri="连接串" -c 集合名 -o 文件.json 将集合导出为 JSON 或 CSV 格式
mongoimport mongoimport --uri="连接串" -c 集合名 --file 文件.json 从 JSON 或 CSV 文件导入数据到集合

附录B：Spring Data MongoDB 常用注解速查表

注解名 作用位置 说明 示例
@Document 类 将 Java 类映射为 MongoDB 集合中的文档，可指定集合名 @Document(collection = "users")
@Id 字段 标记字段为文档的 _id 主键，通常用于 String 或 ObjectId 类型 @Id private String id;
@Field 字段 自定义字段在 MongoDB 中存储的名称，解决 Java 命名与 MongoDB 字段名不一致的问题 @Field("user_name") private String userName;
@Indexed 字段 在该字段上创建单字段索引，可指定唯一性、方向、过期时间等属性 @Indexed(unique = true) private String email;
@CompoundIndex 类 在类级别定义复合索引，指定多个字段的组合索引 @CompoundIndex(def = "{'lastName':1, 'age':-1}")
@CompoundIndexes 类 包含多个 @CompoundIndex 注解的容器注解 @CompoundIndexes({@CompoundIndex(def = "{'a':1,'b':1}"), @CompoundIndex(def = "{'c':1}")})
@TextIndexed 字段 将字段加入文本索引，支持全文搜索 @TextIndexed(weight = 2) private String title;
@GeoSpatialIndexed 字段 在字段上创建地理空间索引（2d 或 2dsphere） @GeoSpatialIndexed(type = GeoSpatialIndexType.GEO_2DSPHERE) private GeoJsonPoint location;
@DBRef 字段 标记字段为文档引用关系，存储为 DBRef 对象，查询时自动解引用加载关联文档 @DBRef private User author;
@Transient 字段 标记字段不持久化到 MongoDB，仅在 Java 对象中使用 @Transient private String tempData;
@CreatedDate 字段 自动填充文档创建时间（需启用审计功能 @EnableMongoAuditing） @CreatedDate private LocalDateTime createdAt;
@LastModifiedDate 字段 自动填充文档最后修改时间（需启用审计功能） @LastModifiedDate private LocalDateTime updatedAt;
@CreatedBy 字段 自动填充文档创建者信息（需配置 AuditorAware 实现） @CreatedBy private String createdBy;
@LastModifiedBy 字段 自动填充文档最后修改者信息（需配置 AuditorAware 实现） @LastModifiedBy private String updatedBy;
@Version 字段 乐观锁版本号，每次更新自动递增，并发写入时检测冲突抛出 OptimisticLockingFailureException @Version private Long version;
@Language 字段 指定文本索引使用的语言，影响分词和停用词处理 @Language private String language;
@TextScore 字段 接收全文搜索的文本匹配得分值 @TextScore private Float score;
@Unwrapped 字段 将嵌套对象的字段展开到父文档中存储，而非作为子文档 @Unwrapped(onEmpty = OnEmpty.USE_NULL) private Address address;
@Sharded 类 标记实体为分片集合，指定分片键 @Sharded(shardKey = {"region", "userId"})

提示： 使用审计注解（@CreatedDate、@LastModifiedDate 等）需要在配置类上添加 @EnableMongoAuditing，并且实体类需要实现或配合 AuditorAware<T> 接口来提供当前操作用户信息。

附录C：聚合管道操作符速查表

1. 管道阶段

阶段 说明
$match 过滤文档，仅让满足条件的文档进入下一阶段，语法同 find() 查询条件
$group 按指定 _id 表达式分组，并对每组应用累加器运算
$project 选择、排除或重新计算字段，重塑文档结构
$sort 对文档按指定字段排序，1 升序，-1 降序
$limit 限制输出文档数量
$skip 跳过指定数量的文档
$unwind 将数组字段展开，每个数组元素生成一个独立文档
$lookup 左外连接另一个集合，实现类似 SQL JOIN 的关联查询
$addFields 向文档添加新字段或覆盖已有字段，不影响其他字段
$set $addFields 的别名，功能完全相同
$replaceRoot 用指定文档或子文档替换当前文档的根
$replaceWith $replaceRoot 的简写形式
$count 统计文档数量，输出包含指定字段名的单文档
$out 将聚合结果写入指定集合（覆盖已有内容）
$merge 将聚合结果合并到指定集合，支持插入/更新/替换等策略
$bucket 按指定边界将文档分到各个桶中
$bucketAuto 自动确定桶边界，将文档均匀分配到指定数量的桶中
$facet 在同一输入文档集上并行执行多个聚合管道，输出合并结果
$sortByCount 按字段值分组并按计数降序排列，等价于 $group + $sort
$sample 从输入文档中随机选择指定数量的文档
$redact 根据条件对文档进行字段级访问控制，递归地保留或剪除子文档
$graphLookup 在集合上执行递归搜索，实现图/树形结构的遍历查询

2. 累加器 / 分组运算符

这些运算符用于 $group 阶段对分组数据进行聚合计算。

运算符 说明 示例
$sum 求和，数值累加 {$sum: "$amount"}
$avg 求平均值 {$avg: "$score"}
$min 取最小值 {$min: "$price"}
$max 取最大值 {$max: "$price"}
$first 取分组中的第一个值（依赖排序） {$first: "$name"}
$last 取分组中的最后一个值（依赖排序） {$last: "$name"}
$push 将值收集到数组中 {$push: "$tag"}
$addToSet 将值收集到数组中（自动去重） {$addToSet: "$category"}
$stdDevPop 计算总体标准差 {$stdDevPop: "$score"}
$stdDevSamp 计算样本标准差 {$stdDevSamp: "$score"}
$count 统计分组中的文档数量（MongoDB 5.0+） {$count: {}}

3. 表达式运算符

3.1 算术运算符

运算符 说明 示例
$add 加法（也可用于日期加毫秒数） {$add: ["$price", "$tax"]}
$subtract 减法（也可用于日期差值计算） {$subtract: ["$total", "$discount"]}
$multiply 乘法 {$multiply: ["$price", "$quantity"]}
$divide 除法 {$divide: ["$total", "$count"]}
$mod 取模（求余数） {$mod: ["$hours", 24]}
$abs 取绝对值 {$abs: "$diff"}
$ceil 向上取整 {$ceil: "$avg"}
$floor 向下取整 {$floor: "$avg"}
$round 四舍五入到指定小数位 {$round: ["$value", 2]}

3.2 字符串运算符

运算符 说明 示例
$concat 拼接多个字符串 {$concat: ["$firstName", " ", "$lastName"]}
$substr 截取子字符串（字节级，已废弃，建议用 $substrCP） {$substr: ["$name", 0, 5]}
$toLower 转为小写 {$toLower: "$email"}
$toUpper 转为大写 {$toUpper: "$code"}
$trim 去除首尾空白字符或指定字符 {$trim: {input: "$name"}}
$split 按分隔符拆分字符串为数组 {$split: ["$tags", ","]}
$strLenCP 返回字符串的 Unicode 码位长度 {$strLenCP: "$name"}
$regexMatch 正则表达式匹配，返回布尔值 {$regexMatch: {input: "$email", regex: /@gmail/}}

3.3 日期运算符

运算符 说明 示例
$year 提取年份 {$year: "$createdAt"}
$month 提取月份（1-12） {$month: "$createdAt"}
$dayOfMonth 提取日期（1-31） {$dayOfMonth: "$createdAt"}
$hour 提取小时（0-23） {$hour: "$createdAt"}
$minute 提取分钟（0-59） {$minute: "$createdAt"}
$second 提取秒数（0-59） {$second: "$createdAt"}
$dateToString 将日期格式化为字符串 {$dateToString: {format: "%Y-%m-%d", date: "$createdAt"}}
$dateFromString 将字符串解析为日期 {$dateFromString: {dateString: "2024-01-15"}}
$dateDiff 计算两个日期之间的差值 {$dateDiff: {startDate: "$start", endDate: "$end", unit: "day"}}

3.4 比较运算符

运算符 说明 示例
$cmp 比较两个值，返回 -1、0 或 1 {$cmp: ["$a", "$b"]}
$eq 判断两个值是否相等，返回布尔值 {$eq: ["$status", "active"]}
$ne 判断两个值是否不等，返回布尔值 {$ne: ["$type", "deleted"]}
$gt 大于比较，返回布尔值 {$gt: ["$score", 60]}
$gte 大于等于比较，返回布尔值 {$gte: ["$age", 18]}
$lt 小于比较，返回布尔值 {$lt: ["$stock", 10]}
$lte 小于等于比较，返回布尔值 {$lte: ["$price", 100]}

3.5 条件运算符

运算符 说明 示例
$cond 三元条件判断，类似 if-then-else {$cond: {if: {$gte: ["$score", 60]}, then: "及格", else: "不及格"}}
$ifNull 如果字段为 null 或不存在则返回替代值 {$ifNull: ["$nickname", "$username"]}
$switch 多分支条件判断，类似 switch-case {$switch: {branches: [{case: {$eq: ["$status", 1]}, then: "启用"}], default: "未知"}}

3.6 数组运算符

运算符 说明 示例
$size 返回数组长度 {$size: "$tags"}
$filter 按条件过滤数组元素 {$filter: {input: "$items", as: "item", cond: {$gte: ["$$item.price", 100]}}}
$map 对数组每个元素应用表达式变换 {$map: {input: "$scores", as: "s", in: {$multiply: ["$$s", 1.1]}}}
$reduce 将数组归约为单个值 {$reduce: {input: "$items", initialValue: 0, in: {$add: ["$$value", "$$this.price"]}}}
$arrayElemAt 获取数组中指定索引位置的元素 {$arrayElemAt: ["$tags", 0]}
$concatArrays 拼接多个数组 {$concatArrays: ["$arr1", "$arr2"]}
$in 判断值是否在数组中，返回布尔值 {$in: ["admin", "$roles"]}
$isArray 判断是否为数组类型，返回布尔值 {$isArray: "$tags"}

3.7 类型转换运算符

运算符 说明 示例
$type 返回值的 BSON 类型字符串 {$type: "$field"}
$toInt 转换为 32 位整数 {$toInt: "$strNum"}
$toLong 转换为 64 位长整数 {$toLong: "$strNum"}
$toDouble 转换为双精度浮点数 {$toDouble: "$strPrice"}
$toString 转换为字符串 {$toString: "$numericId"}
$toObjectId 转换为 ObjectId 类型 {$toObjectId: "$strId"}
$toBool 转换为布尔值 {$toBool: "$flag"}
$toDate 转换为日期类型 {$toDate: "$timestamp"}

6. 工具推荐

工具 链接 说明
MongoDB Compass https://www.mongodb.com/products/compass MongoDB 官方 GUI 工具，免费，支持可视化查询构建、性能分析、索引管理、聚合管道构建
mongosh https://www.mongodb.com/docs/mongodb-shell/ MongoDB 官方新一代 Shell 工具，支持语法高亮、自动补全、丰富的脚本能力
Navicat for MongoDB https://www.navicat.com/en/products/navicat-for-mongodb 商业 GUI 工具，界面友好，适合习惯 Navicat 的用户
Studio 3T https://studio3t.com/ 功能丰富的 MongoDB GUI 工具，支持 SQL 查询、数据导入导出、IntelliShell 等
MongoDB Database Tools https://www.mongodb.com/docs/database-tools/ 官方命令行工具集，包含 mongodump、mongorestore、mongoexport、mongoimport 等
Spring Initializr https://start.spring.io/ Spring 官方项目脚手架，快速生成包含 Spring Data MongoDB 依赖的项目模板
Testcontainers https://testcontainers.com/modules/mongodb/ 基于 Docker 的轻量级测试库，为集成测试提供一次性 MongoDB 实例
Embedded MongoDB（de.flapdoodle） https://github.com/flapdoodle-oss/de.flapdoodle.embed.mongo 嵌入式 MongoDB 库，无需 Docker 即可在单元/集成测试中启动独立 MongoDB 进程

学习建议： 本课程配合 MongoDB 官方文档和 Spring Data MongoDB 参考文档一起使用效果最佳。遇到问题时，优先查阅官方文档，其次在 Stack Overflow 和社区论坛中搜索，通常都能找到解决方案。建议在 MongoDB University 完成 M001（基础）和 M121（聚合框架）两门免费课程来巩固所学内容。
